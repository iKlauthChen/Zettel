# Ingold的线与德勒兹的线——概念映射、根本分岔与通往安全理论的汇合点

## 一、核心命题

**Ingold和德勒兹共享同一个批判目标——西方现代性的"直线霸权"——但两人从完全不同的入口进入，也走向了不同的出路。** Ingold从人类学/物质文化入手，追问书写、行走、编织中线的物质-感知史；德勒兹从哲学/政治经济学入手，追问社会机器如何通过线来组织主体和空间。两者的交汇点在于：**都认为线不是中性的连接工具，而是本体论的基本操作单元。** 两者的分岔点在于：Ingold缺乏德勒兹对权力关系的分析（线如何被国家捕获和利用），德勒兹缺乏Ingold对线的物质-感知维度的展开（手如何握笔、脚如何感知地面）。桥接的任务不是证明"他们说的是一回事"，而是让两种进路在对读中暴露各自的盲点，从而为第三位置——例如安全理论中的"线的政治"——提供更完整的工具集。

## 二、四维平行对照

### 维度1：线的分类学

| | Ingold | Deleuze (via Saldanha) |
|---|---|---|
| 核心区分 | **线缕 (thread)** vs **痕迹 (trace)** | **刚性线** (rigid/molar) vs **弹性线** (supple/molecular) vs **逃逸线** (line of flight) |
| 动态/静态 | 线缕=动态（生长、延伸），痕迹=静态（铭写、保存） | 刚性线=静态（分段、编码），弹性线=动态重组，逃逸线=绝对解域 |
| 本体论定位 | 线缕和痕迹可互相转换——表面随转换生成或消解 | 三种线在任何社会装置中**同时存在**——比例决定装置的形态 |
| 对应关系 | 线缕 ≈ 逃逸线（都是开放的、生长的）；痕迹 ≈ 刚性线（都是封闭的、铭写的） | |
| **盲点暴露** | Ingold缺少弹性线的维度——弹性线意味着线可以在不逃逸的情况下重组配置，而不是只在生长和固定之间二选一 | 德勒兹缺少线缕/痕迹转换的物质机制——线如何通过手、纤维、表面、工具的相互作用改变存在形态 |

### 维度2：空间拓扑

| | Ingold | Deleuze |
|---|---|---|
| 关键对偶 | **远足 (wayfaring)** vs **运输 (transport)** | **平滑空间 (smooth)** vs **条纹空间 (striated)** |
| 运动模式 | 沿着线穿行（along） vs 跨过表面连接（across） | 占据/分布空间 vs 划分/管理空间 |
| 认识论后果 | 栖居知识（dwelling knowledge） vs 装配知识（assembly knowledge） | 触觉/近距空间（haptic） vs 视觉/远距空间（optic） |
| 政治后果 | 远足者创造地方——地方是线的交汇之结 | 国家需要条纹空间——条纹是税收、兵役、统计的前提 |
| **汇合点** | **远足=平滑空间，运输=条纹空间——几乎是拓扑同构。** 两者都认为现代性的本质是第二种模式对第一种的系统性殖民。 |
| **差异** | Ingold的论证是**现象学-民族志的**（因纽特人如何移动、巴特克妇女如何采集），德勒兹的是**政治经济学的**（国家机器如何产生条纹空间、资本如何解域又再辖域）。Ingold告诉你"远足是如何被遗忘的"，德勒兹告诉你"为什么会这样"。 |

### 维度3：反树状模式

| | Ingold | Deleuze & Guattari |
|---|---|---|
| 替代模型 | **菌丝体 (mycelium)** | **块茎 (rhizome)** |
| 特性 | 渗透性、无内外之分、弥漫生长、没有区分介质和个体的表面 | 连接性、任意两点可连、多元入口、异质性连接 |
| 操作逻辑 | 线即是介质——菌丝体不"在"某物之上，它本身就是"物" | 线连接异质元素——块茎"在"一个平面之上建立连接 |
| 知识论姿态 | "研究一个物就是研究构成它的线"——线是物的本体论构成要素 | "书就是块茎"——块茎是知识和存在的组织原则 |
| **根本分岔** | **菌丝体取消平面/线的二元区分**（线本身就是medium）；**块茎保留连接者和被连接者的区分**（平面上有异质元素被连起来）。Ingold更激进但更难操作化——无法区分什么是"在菌丝体之内"和"在菌丝体之外"。德勒兹保留了分析所需的区分，但因此也保留了平面的认识论预设。 |

### 维度4：书写与声音

| | Ingold | Deleuze (差异与重复/意义的逻辑) |
|---|---|---|
| 核心关注 | 书写从手势痕迹变为印刷装配——纸页失去声音 | 书写是虚拟的实现——思想在语言中的"不合时宜的"表达 |
| 批判对象 | 印刷技术割裂了手/眼/耳的连续性 | 教条式思想意象使书写沦为再现——不再是创造概念 |
| 历史叙事 | 书写的物质史：从抄写员的手势到排字工人的装配 | 书写的本体论：书写作为差异的展现 vs 书写作为同一的重复 |
| **互补** | Ingold提供德勒兹缺失的书写物质史（墨水的流动、笔尖的痕迹、纸张的表面） | 德勒兹提供Ingold缺失的书写本体论（书写与差异、虚拟、新事件的关系） |

## 三、四条桥接路径

### 路径A：通过"捕获"（appropriation）桥接

**德勒兹的"国家捕获战争机器"**和**Ingold的"现代性使线变直"**是同一操作在两个层面的表达——都是某种外部性（游牧战争机器 / 远足的缠绕线）被一套新装置（国家 / 印刷-测量-规划装置）征用和改造。差异在于：德勒兹的捕获是政治性的（国家垄断暴力），Ingold的"拉直"是感知-技术性的（书写工具和测量工具改变感受世界的方式）。两者结合可以回答一个安全理论的核心问题：**安全操作在拉直什么？**

→ 接入剩余安全框架：安全化操作本质上也是"使线变直"——把模糊的情势翻译为可操作的边界和红线。但Ingold的论证暗示这种翻译总是不完全的：线缕无法被彻底冻结为痕迹（总有余地）。这个"总是剩余的余地"就是剩余安全的空间。

### 路径B：通过"居住/平滑"（dwelling/smooth）桥接

Ingold的"栖居者"（inhabitant）和德勒兹的游牧者（nomad）共享同一个核心特征：**不是在一个已定义的空间中移动，而是在移动中定义空间。** Ingold说："远足者本人正是运动本身"；德勒兹说："游牧者不是从一个点到另一个点的移动者，而是点的分布在追踪游牧者的路线中被决定。" 把这两个命题放在一起，可以得到一个关于政治主体的新命题：**主体的政治性不在于他占据的位置，而在于他移动的路线。** 这对于安全理论的意义在于：安全的主体不应该是"需要被保护的人口"（占据位置），而应该是"需要被允许移动的运动线"（穿行路径）。情报和控制逻辑试图把主体固定在点上以便管理；剩余安全逻辑承认主体在线上。

### 路径C：通过"直线/思想意象"（straight-line/image-of-thought）桥接

Ingold全书最后的追问是"线是如何变直的"；德勒兹《差异与重复》早期章节追问的是"思想是如何被捕获于教条式意象的"。两者之间存在深层结构同构：德勒兹批判的"再现"（representation）——思想以为自己是在描摹外部现实，实际上只是在一套预设的范畴中调配——与Ingold批判的"点对点连接线"共享同一个操作结构：**把动态过程冻结为静态图示，然后误把图示当成过程的真相。** Ingold对印刷书写的批判和德勒兹对"哲学作为再现"的批判，在"冻结的动态性"这个点上汇合。

### 路径D：通过"线的议会"（parliament of lines）→ 装配理论

Ingold断言"每个事物都是一个线的议会"，这是一个极其激进的命题——实体的本质不是它的属性或功能，而是构成它的各种线的交汇。德勒兹的装配（assemblage）理论——"一个装配是由两个轴线构成的：内容-表达轴线和辖域化-解域化轴线"——提供了让Ingold的直觉可操作化的理论工具。桥接命题：**一个线议会就是一个装配。** Ingold告诉你"线议会由什么线构成"（书写线、行走线、编织线……），德勒兹告诉你"这些线如何组织"（哪些线被刚性化、哪些在逃逸、哪些在重组）。安全分析的任务就是分析一个安全情势中各种线的组成和配置。

## 四、安全理论的应用方向

两条思路的汇合在安全理论中最有生产力的应用不在宏观层面（大国战略），而在微观/感知层面：

1. **红线的本体论**：Ingold的直线批判 + 德勒兹的条纹化批判 可以为"红线"（red line）——安全话语中最常见的直线隐喻——提供一个完整的谱系学批判：红线既是国际政治的修辞操作（德勒兹的捕获维度），又是认知的简化操作（Ingold的拉直维度）。一条红线的设置同时是政治行为和感知行为。

2. **灰色地带的线型分析**：灰色地带冲突之所以难以识别和应对，是因为它混合了远足和运输两种运动模式——行动者像远足者一样"穿行"于法律和认知的缝隙中，但像运输者一样有明确的战略目的地。Ingold的线型分类加上德勒兹的装配分析，可以为灰色地带操作提供一种"线型枚举"分析方法：给定一个行动者的运动轨迹，它在哪些段落更像线缕、哪些更像痕迹、哪些更接近逃逸线？

3. **本体论安全与线的连续性**：本体论安全的文献强调主体需要"叙事的连续性"（Giddens, Steele）。Ingold的"向前生活"命题把这个连续性从叙事层面拉到感知-运动层面——人的本体论安全感首先来自"沿着一条可识别的线移动"的体验，而不是来自"讲一个连贯的故事"的能力。后者是对前者的叙事化再现。如果这种体验被中断（如难民、流离失所者），本体论安全就被破坏了。这为安全研究的"安全感"维度（[[20260621-张清敏安全感被忽视的维度]]）增加了一个身体-运动的底层维度。

## 五、两张传统的各自风险

- **Ingold的风险**：反现代性浪漫主义——把"前现代的远足者"当作本真生活模板，回避了现代基础设施（电网、水系统、通信网络）也是装配，而且不是坏的装配。安全理论若吸收Ingold而不加过滤，会陷入"传统栖居=安全=好 / 现代连接=不安全=坏"的幼稚二元。

- **德勒兹的风险**：概念过于抽象——逃逸线被浪漫化为天然进步的力量（"所有逃逸都是解放"），忽视了逃逸也可以进入更坏的状态（吸毒者的逃逸线是自毁）。Saldanha已经指出这一点。Ingold对线的物质性的坚持（每种线都与具体的手工活动相关）恰好是对德勒兹抽象化的约束。

- **桥接后的风险**：两者的结合可能过于强调"线的流动性"而低估了不变的结构性暴力。不是所有线都值得被尊重——殖民者的线、种族隔离的线、专属经济区划定的线——拉直它们确实是不道德的，但重新缠绕它们也不自动等于正义。
