# 战略转型的动力机制——新古典现实主义因果模型

**出处：** 秦立志《陆海复合型国家战略转型的动力机制——兼论对中国的启示》，《太平洋学报》2019年第27卷第2期，第1—12页。

## 核心贡献

本文是秦立志2016年论文的**理论升级**——从描述性六因素框架升级为**新古典现实主义因果模型**（IV→MV→DV）。2016年列了六个因素但没有说明它们之间的因果逻辑，2019年提供了一个正式的理论因果链。

## 因果模型

```
自变量（体系结构）                     因变量（战略转型）
  地缘格局                                  战略重心转型
  地理互动        →  中介变量（战略决策层）  →  攻防政策转型
  攻防平衡            认知过滤               制衡倾向转型
                      ↓                     联盟策略转型
              包容性/约束性战略环境
```

**核心公式**：
> 外部战略环境作为自变量，需要通过国家战略决策层面这一中介变量的过滤，才能对战略转型发挥作用，且**自变量与中介变量的作用大小通常成反比**。（摘要）

**反比关系的含义**：
- 当外部战略环境**高度约束性**（紧迫威胁/明确机遇）→决策层的过滤空间小→体系结构直接决定转型
- 当外部战略环境**高度包容性**（信息模糊/选择菜单多）→决策层的认知和偏好发挥更大作用→单元/次单元因素占据主导

## "动力机制"的定义

> 动力机制，意指对国家战略转型的生成和演化产生推动或规制作用的变量，涉及体系结构、单元和次单元等多层面要素。（p.4）

**强调**：地理和权力要素都不能单独决定陆海复合型国家在崛起后必然走向地缘扩张——"地理决定论和权力决定论不符合中国'和平崛起'、'建设海洋强国'的防御性诉求"（p.4）。

## 与2016年六因素框架的关系

| 2016年框架 | 2019年新古典现实主义框架 | 升级 |
|-----------|------------------------|------|
| 战略地理结构 | → 体系结构：地缘格局（极的多寡与分布） | 从描述到操作化 |
| 地缘威胁 | → 体系结构：地理互动（海陆相对位置/邻国效应/通达度） | 增加机制解释 |
| 攻防平衡 | → 体系结构：攻防平衡（作为结构性调节因素） | 保持但明确定位 |
| 战略学说 | → 单元/次单元：战略学说（影响认知） | 定位为认知过滤器 |
| 战略决策 | → 单元/次单元：崛起路径+资源汲取+经济转型（影响决策与执行） | 分解为三要素 |
| 资源汲取 | → 单元/次单元：纳入决策执行层面 | 与新古典现实主义对接 |

**Why:** 本文最重要的理论贡献是"A与B成反比"的命题——它调和了结构决定论和国内政治论：不是"结构重要还是国内重要"的二选一，而是"结构压力越大，国内变量的作用空间越小"的条件性关系。这使得分析者可以依据战略环境的紧迫程度来调整分析的重点。

**How to apply:** 分析某一国家的战略转型时，先判断其外部战略环境的"包容性/约束性"程度——如果高度约束（明确威胁），则重点分析体系结构因素；如果高度包容（信息模糊），则重点分析国内政治和决策认知因素。与[[qin-strategic-uncertainty-card1-core-thesis]]（秦立志2023不确定性论）交叉——2023年的"战略不确定性"正是2019年"包容性/约束性战略环境"概念的深化：包容性环境≈高不确定性，约束性环境≈低不确定性。
