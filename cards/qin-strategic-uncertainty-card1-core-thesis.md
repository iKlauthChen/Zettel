# 战略不确定性与安全困境生成机制——核心论题

**出处：** 秦立志《战略不确定性与安全困境的生成机制》，《国际政治科学》2023年第8卷第2期，第61—101页。

## 核心因果模型

**自变量（IV）：战略不确定性** → **中间变量（MV）：威胁预期** → **因变量（DV）：安全困境**

```
战略不确定性 ──→ 威胁预期 ──→ 安全困境
  (互动环境)      (进攻/防御)    (四种烈度)
```

## 关键概念界定

**战略不确定性** ≠ 错误知觉。不确定性反映的是国家实际获得的信息（客观匮乏），错误知觉则是对可获得信息的错误评估（主观歪曲）（p.65）。

**不确定性** ≠ **风险**：
- 风险：可度量的不确定性，与概率有关（如战争爆发概率）→ 可以管控
- 不确定性：本质上无法消除（如无政府状态带来的恐惧感）→ 可以转化为收益或演变为风险
- 常见错误：将不确定性误判为风险提升（p.65）

本文聚焦于**相对的战略不确定性**——完全不确定性与决定论确定性之间的灰色地带（含一定的确定性信息）。不确定性提出也可以演变为收益，风险则主要导向负面结果。

## 战略不确定性的三个测量指标

| 指标 | 核心内容 | 高不确定性 | 低不确定性 |
|------|---------|-----------|-----------|
| **权力中心数量及地理分布** | 极的数量、陆海位置、侧翼/中心 | 多极体系、侧翼位置、海洋争端 | 两极体系、中心位置、陆地争端 |
| **权力转移** | 速率、领域（军事/经济） | 速率较慢时 | 速率较快、军事领域 |
| **大国战略行动** | 内部/外部制衡、渐进/激进路线 | 内部制衡为主、渐进路线 | 外部制衡为主、激进路线 |

当三个指标中有两个以上具有不确定性特征时，视为战略互动环境具有较高不确定性（p.71）。

## 安全困境的矩阵分析（表1）

| | 防御性现实主义威胁预期 | 进攻性现实主义威胁预期 |
|--|----------------------|----------------------|
| **战略不确定性较高** | 烈度较低的安全困境 | 烈度适中的安全困境 |
| **战略不确定性较低** | 烈度较高的安全困境 | 烈度最高的安全困境 |

**关键发现**：较低的战略不确定性（如明确的敌对关系）并不一定降低冲突风险——当双方对彼此意图的确定性认知是消极的时，反而导致更高烈度的安全困境。高不确定性有时可以缓冲冲突。

## 理论创新

秦立志的框架不同于既有安全困境理论之处：
1. 明确将"不确定性"与"风险"/"错误知觉"区分，避免概念混淆
2. 引入威胁预期作为中间变量，打开安全困境生成机制的"黑箱"
3. 以过程追踪法（理论建构型）替代统计验证，关注因果机制而非相关关系
4. 反事实推理贯穿案例——"如果……就不会……"的推演揭示战略选择的替代可能

**Why:** 此卡确立了整篇文章的分析框架——三变量+矩阵+三大放大机制，是理解后续案例与桥接对话的钥匙。

**How to apply:** 分析任何大国竞争关系时，先评估互动环境的不确定性程度，再识别双方的威胁预期取向（进攻/防御），由此推演安全困境的演化方向。与[[yang-complex-system-card1-core-thesis]]（杨思灵复合系统论）互补——杨思灵的系统分析提供结构性背景，秦立志的因果链提供机制性解释。
