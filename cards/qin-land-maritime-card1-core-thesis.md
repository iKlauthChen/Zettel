# 陆海复合型国家战略转型——核心论题

**出处：** 秦立志《历史上陆海复合型国家的战略转型及其对中国的启示》，《中国周边外交学刊》2016年第二辑（总第四辑），第24—51页。复旦大学国际关系与公共事务学院博士研究生。

## 陆海复合型国家的定义

在陆地上背靠较少自然障碍且濒临开放性海洋的国家（p.24）。

**关键区分**：地理层面的陆海两栖 ≠ 战略层面的陆海兼顾。战略中心只能依托陆权，**战略重心应处于动态分布中**。

三种表现形式：陆权主导、海权主导、陆海兼顾。

## 大战略取向的四种表现形式

| 维度 | 选项 | 主导性 |
|------|------|--------|
| **战略重心** | 海主陆从 / 陆主海从 / 海陆并举 | **核心**——重心变化则大战略必然变化 |
| **攻防政策** | 海陆两方向分别采取进攻性/防御性 | 重心不变时攻防转换也属于转型 |
| **制衡倾向** | 制衡缺位 / 制衡过度 / 适度制衡 | 重心+攻防不变时制衡变化也算转型 |
| **联盟策略** | 以陆制海 / 以海制陆 / 不结盟 | 前三者不变时联盟策略变化也算 |

**例**：中苏同盟名存实亡和中美改善关系→中国联盟战略由以陆制海变为以海制陆→20世纪80年代后从以海制陆转向不结盟（p.25）。

## 影响战略转型的六大因素

| 因素 | 核心内容 | 作用方式 |
|------|----------|----------|
| **1. 战略地理结构** | 战略极/战略通道/战略锋场 | 战略极实力先导、通道变动影响进程和方向、锋场与核心利益区重合度决定转型难度 |
| **2. 攻防平衡** | 陆权/海权 × 进攻有利/防御有利 = 4种类型 | 决定转型的时机、方向和扩张性程度 |
| **3. 地缘威胁** | 海权型 / 陆权型 / 海陆双缘 | 威胁类型决定战略重心取向 |
| **4. 资源汲取** | 外部脆弱性 × 国家政治权力 = 4种场景 | 外部脆弱性与政治权力的组合决定转型成败 |
| **5. 战略学说** | 进攻性/防御性/威慑性 | 学说的进攻性程度决定引发制衡的烈度 |
| **6. 战略决策** | 决策层共识、领导人偏好、对攻防平衡的认知 | **核心作用**——其他5个因素只有经过决策层认知过滤才形成战略输出 |

> 战略决策层在这6个要素中发挥着核心作用，客观的战略地理结构、地缘威胁等要素只有经过决策层的认知过滤，才会形成大战略输出。（p.34）

## 理论与实践意义

秦立志（2016）与[[qin-tan-geostrategy-ingest]]（地缘位势卡组）形成同一作者前后期的深化关系——2016年的本文更接近**经典地缘政治研究**（以海权/陆权二元框架分析大战略），而2021年的地缘位势操作化则是**科学化转向**（以二维坐标测量战略位置）。

**Why:** 此卡确立了全文的四维取向+六因素框架。四维取向界定了"什么是战略转型"，六因素解释了"什么驱动转型"，两者结合构成了一个完整的分析装置。

**How to apply:** 分析任何陆海复合型国家（如中国、印度、法国、俄罗斯）的战略转型时，用四维定位当前状态，用六因素判断转型条件和风险。与[[yang-complex-system-card2-four-subsystems]]（杨思灵复合系统）交叉——秦立志的六因素可嵌入杨思灵"战略要素系统"作为具体分析内容。
