# 朱宁 (2003) ——安全与非安全化：哥本哈根学派安全研究

**来源**: 《世界经济与政治》2003 年第 10 期，pp. 21-26  
**作者**: 朱宁，中国社会科学院世界经济与政治系博士研究生  
**核心论题**: 系统介绍以巴瑞·布赞（Barry Buzan）和奥利·维夫（Ole Wæver）为代表的哥本哈根学派安全研究的核心概念（安全复合体、言语—行为、主体间性安全、安全化/非安全化），并在中国视角下对其欧洲中心主义倾向提出批判。

---

## 卡 1：哥本哈根学派的学术定位

**核心论点**: 哥本哈根学派是冷战后西方安全研究领域最显赫的流派，它以社会建构主义为哲学基础，兼容传统主义安全议程，提出安全复合体理论和非安全化理论。

**学派形成标志**:
- 1985 年哥本哈根和平研究所（COPRI）成立，奥利·维夫参与创始人
- 1990 年《欧洲安全秩序的重塑》——"哥本哈根学派诞生宣言"
- 1993 年《认同、移民与欧洲的新安全议程》——提出"社会安全"概念
- 1996 年 McSweeney 和 Neumann 首次使用 "Copenhagen School" 称呼这批研究者

**理论家族位置**:
- 属于建构主义安全研究的"欧洲分支"，与美国分支（卡赞斯坦为代表）并列
- 在"第三浪"国际理论大辩论中成形
- 不同于 Waltz 新现实主义和 Keohane 新自由制度主义

**与建构主义共享的前提**:
- 国际体系结构 = 物质资源分配 + 社会相互作用（共有知识）
- 行为体的身份和利益是社会建构的
- "敌对"还是"友善"取决于"观念分配"而不是物质力量分配

**代表学者**: Barry Buzan, Ole Wæver, Morten Kelstrup, Pierre Lemaitre, De Wilde

**Why:** 哥本哈根学派是国际关系"语言学转向"在安全研究领域的集中体现——这篇 2003 年的文章是中国学界较早的系统引介，有助于理解后来"安全化"概念在中文学界的传播轨迹。

---

## 卡 2：安全复合体理论（布赞）

**核心论点**: 安全相互依存的常规模式是一种以**地区**为基础的集合——一组单元的安全认知被相互连接在一起。

**安全复合体基本结构三要素**（布赞）:
1. 单元间安排和它们之间的差异
2. 友善模式（amity）和敌对模式（enmity）
3. 主要单元中的权力分配

**关键修正**:
- 早期（古典复合安全理论 CSCT）：以国家为主要单元
- 后期（《新安全论》1998）：单元扩展到各种次集团、组织、共同体（国家、民族、跨国公司等）
- 同时将军、政、经、社、环五个领域一视同仁，不再突显军事-政治领域

**安全复合体的两端光谱**:
- 一端：冲突（恐惧、竞争、共有威胁认知）
- 另一端：**多元安全共同体**（不再在彼此关系中使用武力）

**Why:** 安全复合体理论是哥本哈根学派对地区安全研究最重要的贡献——将安全分析从全球层面下放到地区层面，为解释冷战后欧洲安全（以及扩展至其他地区）提供了框架。

## 卡 3：安全化——核心机制（维夫）

**核心论点**: 安全不是客观既定的状态，而是通过"安全化"（securitization）这一言语行为建构出来的。安全化就是把某事宣称为"存在性威胁"并因此获得打破常规政治规则的权利。

**安全 vs 政治化**:

| | 政治化 | 安全化 |
|---|--------|--------|
| 性质 | 常规政治进程 | 例外政治 |
| 手段 | 政府决策+资源配置 | 打破规则+宣布合法 |
| 空间 | 可公开、可选择、可问责 | 最高决策层优先处理 |
| 状态 | 常规的 | 紧急状态 |

**安全的三重特性**:
1. **自我参照的实践**（self-referential practice）——正是在实践中该问题变成了安全事务
2. **超越一切政治规则和政治结构的一种途径**——"所有政治之上的特殊政治"
3. **主体间性的**——成功的安全化由安全化施动者和听众共同决定

**安全化的内在条件**:
- 遵循安全体制的文法规则
- 建造一个包含"存在性威胁"秘密计划、极限点和摆脱困境可能的话语结构
- 不同领域的专业用语（安全/认同/主权/可持续能力）

**安全化的外在条件**:
- 言语者的社会地位（必须处于权威地位）
- 与威胁相关的"催化条件"

**Why:** 安全化是哥本哈根学派最具生产力也最具争议的概念——它将 Austin 的言语行为理论引入安全研究，安全化不再是描述安全"是什么"，而是分析安全"如何被制造"。

## 卡 4：言语—行为——语言学建构主义分析工具

**核心论点**: 哥本哈根学派将安全定义为一种特殊的**言语行为**（Speech Act），借用 Austin 的论述行语言理论来分析安全的建构机制。

**Austin 模型的应用**:
- 像"我宣布你们是夫妻"一样——只有当被权威执行才是一个有效的言语行为
- 安全化施动者通过言语行为创造了一种新的语境（而不是反映已有语境）
- 安全化言语行为具有**表述性力量**（illocutionary force）

**成功安全化的要素**:
1. **施动者**（Securitizing Actor）——必须处在权威地位
2. **言语行为**本身——遵循文法规则，包含存在性威胁+极限点+解决路径
3. **听众**（Audience）——接受"对共有价值造成存在性威胁"的说法
4. **催化条件**——逼近的坦克、敌对情绪、被污染的水源等事实性元素

**安全分析的问题清单**:
- 谁实施安全化？/ 什么问题造成威胁？ / 指涉对象是谁？ / 何以如此？
- 带来什么后果？ / 在什么条件下实施？ / 成功时人们作何解释？

**Why:** "言语—行为"是哥本哈根学派方法论的独特贡献——安全研究的焦点从"什么是安全"转向"谁、在什么条件下、以何种方式成功地将某事建构为安全问题"。

## 卡 5：主体间性安全

**核心论点**: 安全既不是纯主观的（一种意识到的威胁存在），也不是纯客观的（一种真正的威胁存在），而是存在于**主体间**（intersubjective）的社会构成之中。

**沃尔弗斯（Arnold Wolfers）的遗产**: 安全同时接近客观性（真正威胁）和主观性（意识到的威胁）

**哥本哈根学派的推进**: 安全化实践才是分析的中心——问题是否成为安全事务不由个体独自决定，而是由：
- 安全化行为主体对"存在性威胁"的宣称
- 听众是否接受这一宣称
- 两者之间的主体间互动共同决定

**理论意义**:
- 安全分析的兴趣从"安全是什么"转移到"安全化的实践"
- 安全化必须作为主体间过程来理解——不同的国家和民族定义"威胁"有不同的界限
- 在一个政治共同体合法的安全化，对另一共同体的外部观察者可能看起来是妄想狂

**Why:** 主体间性概念将安全研究从物质主义（新现实主义）和主观主义（心理主义）两个极端中解放出来——安全是一种社会事实，既依赖物质条件又依赖社会共识。

## 卡 6：非安全化——规范性理想

**核心论点**: 安全应当被视为消极的——是常规政治处理问题失败而不得已采取的措施。实践与理论的目标应该是逐步缩小安全范围，最终将政治非军事化。

**维夫的规范性立场**:
- "并非安全越多就越好"
- 安全化会造成以安全为名压制反对派、剥夺公民自由的后果
- "非安全化"是长时间范围内的最优选项
- 不应随意将所有新问题都视为安全问题

**对安全议程拓宽的质疑**:
- 每一个国家在安全外延上拓宽 = 一种"观念上的安全困境"
- 以日本"综合安全"为例：民事问题（两用技术出口）通过安全指涉上升为安全事务，挤压了国家间的弹性空间
- 互相逐高则导致真正的"安全困境"从观念层面作用到物质层面

**来自发展中国家的批评（作者立场）**:
- "非安全化"带有欧洲中心的偏执——维夫忽略了不同历史阶段国家对安全的需求层次不同
- 对丹麦等无生存压力的国家可行，但对朝鲜、伊朗及许多发展中国家来说参与政治独立的代价巨大
- 未被"国家"剥夺的资源有可能被"国际社会"背后的霸权国剥夺
- 王逸舟的补充：安全化带来的动员对社会整合及认同强化有积极一面

**Why:** 非安全化是哥本哈根学派最富有争议的理论贡献——它既是规范性的理想（缩小安全范围），又暴露了该理论隐含的欧洲中心主义前提。

## 卡 7：对哥本哈根学派的中国视角批判

**核心论点**: 哥本哈根学派的核心概念中潜在地包含着危险的政策倾向——它的"非安全化"理想预设了一个无重大生存威胁的世界，这在全球不均衡发展的结构性冲突面前是不充分的。

**批判性反思要点**:

1. **欧洲中心主义**: 维夫的理论带有"强烈的建构主义种族优越感"
   - 丹麦等福利国家的安全需求 ≠ 发展中国家的安全需求
   - 后者所渴望的不过是"生存"而已

2. **安全议程拓宽的悖论**:
   - 布赞以五个领域对抗军事主导固然进步
   - 但拓宽本身造成"观念上的安全困境"——各国竞相拓展安全外延

3. **结构性冲突的被忽视**:
   - "文化"和"历史记忆"被布赞、维夫强调却未予重视
   - **不均衡发展**及其带来的冲突和霸权干涉才是安全问题的全球性根源
   - 物质不均衡 + 观念不均衡 = 双重的掠夺结构

4. **理论的非中立性**:
   - 社会科学理论是社会生活组成的一部分
   - 即使温特也承认自己理论的政策意义
   - 维夫的安全观不仅仅是描述，也决定了国际关系运行的观念框架

**Why:** 这篇 2003 年的文章虽然是引介性质，但结尾的批判相当有力地揭示了哥本哈根学派的隐性政治前提——在"非安全化"的普遍主义修辞下，隐藏着特定历史地理位置的视角偏见。

**How to apply:** 可与[[20260622-0058-fang-guo-metaphor-scenario-nontraditional-security]]（非传统安全话语隐喻图景）的形式语言学分析形成互补——前者从语言学实证入手分析安全化的话语机制（五类隐喻如何塑造认知框架），后者从理论层面揭示安全化的哲学前提（言语行为/主体间性/语言建构主义）。两者共同回答"安全化在语言层面如何运作"这一问题。

**→ [[20260622-0079-yin-liu-national-security-personality-foundations]]**：哥本哈根的"安全化施动者"理论缺一个微观心理层——正是尹继武的人格框架填补了这一空白：安全化行为主体的人格特质（不信任特质、权力动机、认知闭合需求）解释了为什么在某些国家/时期安全化更容易成功。同时，张清敏的"有意偏见"机制（[[20260622-0080-zhang-security-sense-neglected-dimension]]）揭示了安全化的国内政治驱动——安全化不仅是言语行为，还是国内政治需要的产物。

此外，维夫的"非安全化"理想与[[20260622-0060-hu-bo-chen-grey-zone-strategy]]中的灰色地带竞争呈现有趣的镜像关系：非安全化试图将议题从安全领域退回政治领域（缩小安全范围），灰色地带策略恰恰利用安全概念的模糊性将竞争维持在战争门槛以下——灰色地带可被理解为"非安全化的失败"或"安全化不完全"的常态化状态。

[[20260622-0068-qin-tan-geostrategy-ingest]]（地缘位势与安全感知）中"安全感知的双重来源"（决策者认知+精英集团施压）为哥本哈根学派的"主体间性安全"概念提供了具体机制——安全化的受众接受程度不仅取决于言语行为的修辞效力，还受到地缘位势结构性压力的调节。
