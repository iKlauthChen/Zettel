# 高方 & 路斯琪 (2020) ——从文本到世界：一种方法论的探索——韦斯特法尔《地理批评：真实、虚构、空间》评介

**来源**: 《文艺理论研究》2020 年第 4 期，pp. 21-28  
**作者**: 高方、路斯琪（南京大学外国语学院）  
**评介对象**: Bertrand Westphal, *La Géocritique: réel, fiction, espace* (Paris: Minuit, 2007)  
**核心论题**: 地理批评（géocritique）作为一种兼具视角多元化和跨学科性质的比较研究方法论——以"地理中心"（géocentré）取代传统的"自我中心"（égocentré）研究路径，通过对比后现代时空中真实和虚构的不同再现形式，探索文本空间与世界空间的互动本质。

---

## 卡 1：地理批评的起源与核心转向——从自我中心到地理中心

**核心论点**: 韦斯特法尔以**地理中心**（géocentré）为指引，将置于多重视阈交汇中的**场所再现**作为研究对象——有别于传统围绕作家创作展开的"自我中心"（égocentré）路径和将场所仅视为空间背景的文学地理学。

**理论起源**:
- 1999 年韦斯特法尔首次提出"地理批评"概念
- 2000 年《地理批评使用方法》论文集问世——《走向文本的地理批评研究》被视为奠基之作和"诞生宣言"
- 2007 年《地理批评：真实、虚构、空间》由子夜出版社出版
- 受列斐伏尔（精神空间/社会空间生产）、德勒兹与瓜塔里（地理哲学/空间运动）、索亚与詹姆逊（空间转向）影响

**关键问题意识**:
- "地理学家研究地球，社会学家亦然，人类学家、法学家等也不例外。那文学工作者呢？他们的世界是怎样的？"
- 从"如果把与同一场所相关的不同虚构观点联系在一起会产生怎样的结果？"出发

**核心术语**: 
- **Géocritique** — 前缀 géo 源自希腊语 γῆ (gê)，意为土地，强调"人类历史进程中对土地的投入、改造与诠释的方式"
- 地理批评 = 一种诗学，考察"人文空间和文学的相互作用"——不是对文学中的空间再现的考察

**Why:** 标志文学空间研究从"作者中心"到"空间中心"的范式转换——不是一个作者如何写一个地方，而是一个地方如何在不同的文学/艺术再现中被建构。

## 卡 2：后现代空间三大特征——空时性、越界性、指涉性

**核心论点**: 韦斯特法尔以二战后的新时空关系为起点，论证空间间性框架下真实与虚构可以相互转化、相互影响。

### 一、空时性（spatio-temporalité）——**空间的反弹**
- 两次工业革命：空间从属于时间（时间=进步，空间=可被时间操控的对象）
- 两次世界大战打破此格局：殖民空间、种族空间、环境恶化击垮"时间=进步"信条
- 线性时间 → 承载瞬间体验的时间点；线的消弭 + 点的扩张 = 时空相融
- 新时间概念："微时间"（tempuscule）——时间的间隙
- 隐喻体系："分叉"（博尔赫斯 《小径分叉的花园》）、"熵"（品钦）
- 德勒兹"生成是地理的"、福柯"20 世纪是空间的时代"
- 全球化人口流动使空间杂糅、异质、复数化
- **城市**作为主要研究场所

### 二、越界性（transgressivité）——**从文本到世界的可能**
- 越界最初是空间意义上的动作（跨越门槛）
- 从真实空间延伸到精神/想象空间——这一过渡本身就是越界
- 德勒兹-瓜塔里：平滑空间/条纹空间的不断转换 + 解域/结域
- Even-Zohar 多元系统 + Lotman 符号域
- **第三空间**（索亚）：主观/客观、抽象/具象、真实/虚构、重复/差异——一切对立在第三空间相遇并共存
- 文本空间与世界空间在越界性作用下互相进入对方领域

### 三、指涉性（référentialité）——**空间互动的三种类型**
- 可能世界（Possible World）包含真实与虚构之间的三种关系：
  1. **同托邦** — 以逼真性为标准，指涉与再现一致（如现实巴黎与雨果笔下的巴黎）
  2. **异托邦** — 指涉与再现彻底混乱（如莎士比亚将维罗纳置于亚得里亚海沿岸）
  3. **乌托邦** — 彻头彻尾的乌有之地，与真实指涉无稳定关系

**四种扰乱策略**（McHale）：并列 / 嵌入 / 印叠 / 错配

**关键词**: "虚构不是再现真实，而是实现了迄今尚未被表达的新的潜在性"

**Why:** 三大特征搭建了地理批评的本体论框架——论证为什么在后现代条件下真实与虚构的二分不再成立。

## 卡 3：文本空间与世界空间的双向互动

**核心论点**: 后现代语境中的空间与文学不是指涉与再现的单向参照，而是**双向互动**的关系模式。

**互动机制的三个方面**:
1. **文本空间自身的演变**：文学文本本身是一个集指示性、象征性和图像性于一体的符号空间——基于符号的可阐释性，文学空间会在特定空间环境中超越作家表达意图，形成新的符号空间
2. **后现代空间的多变复杂**：二战后时空关系转变 + 政治经济社会文化因素制约 → 后现代空间的形态与意义比以往更复杂
3. **文学对空间的反作用**：文学再现激活真实空间的潜在虚拟性 → 通过想象和阐释进一步创造和丰富空间再现的形式与内涵

**"可再生的关系"**:
- 地理批评不再是对指涉对象的被动投射与机械模仿
- 文学空间是一种可以**作用于外部世界的新动力**
- "真实空间与空间再现之间的互动：一部文本的空间指涉对象本身，已经部分承载了文学参照"

**最终定义**:
> 地理批评从本质上说是一种有关**空间间性**（interspatiality）的研究——文学空间与真实空间的间性问题可被理解为文本与世界、自我与他人共享的**主体间领域**的能动状态。

**Why:** 从"文学反映空间"到"文学建构空间"——地理批评与德勒兹"文学是生成"和"文学是介入"的哲学立场形成深层呼应。

## 卡 4：地理批评的四个核心方法论原则

**核心论点**: 地理批评的多元性集中体现在四个理论核心点上——多聚焦、多感观性、时空分层、互文性。

### 1. 多聚焦（Multifocalisation）
- 以丰富的语料为依托，从整体把握空间/场所特征
- "不可能只通过一门地理学就了解一个国家，也不可能仅通过一部文学作品就从整体上把握所研究场所的特征"
- 例：土生土长的巴黎人/到访的外地人/长居的外国人——各不相同的空间感知
- 作品集合=涵盖本土、异域、异族视角的语料库

### 2. 多感观性（Polysensoriality）
- 打破视觉的绝对优势——场所的味道、声音、肌理
- "每处场所均有属于自己的味道、声音和肌理"
- 文化不仅是视觉感受，还在听觉、触觉和嗅觉上

### 3. 时空分层（Spacio-temporal stratification）
- 空间的生成是时间累积的过程，场所只有在时空关系中才能被感知
- 单一时间性是一种霸权
- **多时性**（polychronie）：将不同的时间性分配给每一处文化空间
- 语料既要共时广度，也要关注空间文化记忆的非共时性

### 4. 互文性（Intertextuality）
- 文本与空间在相互影响中构成互文关系
- 文本诞生于空间又赋予空间生命
- 空间可以诠释文本，也可以通过文本理解空间
- "文本是空间的浓缩，空间则因成为大写的文本而具有可读性"
- **图书馆与世界的隐喻**

**Why:** 四个核心点从方法论上落实了"地理中心"转向——使地理批评从理论构想变为可操作的批评实践。

## 卡 5：地理批评与世界文学——反种族中心主义的诗学

**核心论点**: 地理批评以地球土地的平等性和空间的普遍性来消弭文学的等级性——尝试把有等级的世界文学转变为**无等级的地球文学**——是对传统等级论、中心论和极性论的解构。

**世界文学维度**:
- 韦斯特法尔："我有意拓展面向世界的理论视野，目的在于打开尽可能多的文化参照，用来反抗某种理论上的'种族中心'主义"
- 地理批评方法有助于走出文化禁锢，远离对"他者"的肆意杜撰
- 赋予对抗西方传统二元对立、驳斥主观主义和民族中心主义的诗学属性

**对"世界文学"的重新构想**:
- 文学作品因在民族空间与世界空间中的形态、价值和影响不同而产生等级划分
- 但一旦文学被创造、空间被想象——它们就必然成为世界空间的一部分
- 地理批评借空间的普遍性消弭等级性
- 世界文学 = "地理批评的一个维度"

**实践路径**:
- 地理中心论以**场所**为切入点，不论作家国籍、种族和身份
- 汇集作品构成的重叠空间 → Homi Bhabha 的"**中介空间**"
- 被边缘化的文学作品以空间间性方式融入世界文学的宏观空间
- **绘图学**思路引入被置于不同再现交叉下的空间研究

**Why:** 地理批评最终走向的不仅是文论革新，更是一种"解殖民"的理论姿态——以空间平等性消解西方中心的文学等级秩序。

## 卡 6：对地理批评的质疑与回应

**核心论点**: 地理批评在理论和实践方面仍存在若干问题，但其跨学科优势和方法的开放性使其在空间研究"地理转向"趋势下具有持续生命力。

**来自学界的质疑**:

1. **指涉性的说服力问题**（Khalid Zekri）: 
   - 韦氏"文学再现被包含在世界中"的论断将真实与虚构关系转化为"虚构在社会世界中流通"的问题
   - 真实与虚构"两者之于对方存在真正的自主性吗？"
   - "文学创作是否会再次制造出和经验世界里一样的空间指涉对象？"

2. **语料的西方中心偏向**:
   - 论述主要基于庞杂的欧美语料，对东方作家及其作品少有提及
   - 难免沦为"一种笼统的概括"

3. **翻译角色的忽略**:
   - 在文本走向世界的过程中翻译至关重要
   - 韦氏虽多次谈及翻译的重要性，却未形成系统的理论思考

4. **"地理中心"的隐含等级**:
   - 研究的场所仅限于文学艺术作品指涉较多的大城市 → 显示出新的等级划分

5. **实践之难**（Robert Tally）:
   - 如何界定语料搜集的"度"？怎样判定语料搜集的全面性？
   - "材料浩如烟海，没完没了……你在哪里收手呢？"

**积极回应**:
- 法国《批评》杂志（2008）肯定其"多学科要求"、"丰富的文学语料"、"清晰的脉络"
- 勒阿弗尔大学（2011-2012）两场专题研讨会是地理批评方法的应用实践
- 方法论指向"应用文学"——非证明专业性的条条框框

**Why:** 这些质疑恰恰提示了地理批评的未来发展空间——如果能突破语料范围的限制、将翻译理论纳入框架、解决语料全面性问题，地理批评可能成为一种更具普遍性的文学空间研究范式。
