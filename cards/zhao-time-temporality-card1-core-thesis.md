# 时间、时间性与智慧——核心论题

**出处：** 赵鼎新《时间、时间性与智慧：历史社会学的真谛》，《社会学评论》2019年第7卷第1期，第3—17页。

## 核心命题

人类叙事只有两种基本形式（p.3）：
- **结构/机制叙事**（社会学的特长）——分析结构/机制与社会后果之间的关联
- **时间序列叙事**（历史学的特长）——分析时间序列中关键节点上的人物/事件对后果的转折性影响

**历史社会学的真谛**（而非定义）：有机结合这两种叙事方式。"即使分析当代甚至是最新发生的事情，我们照样可以采取历史社会学视角"——历史社会学不在于研究对象在时间上的远近，而在于方法论上的特殊性（p.3-4）。

## 时间本体作为方法论关键

两种叙事如何结合？关键在于**理解时间的结构**（p.4）：

> 对时间的性质，或者说对时间和社会结构之间的关系的不同理解就成了学者们看问题的不同出发点，或者用哲学语言来说就是学者们不同的本体承诺。（p.4）

## 十种时间观的论点结构

本文的对称性结构：

| 西方五种时间观（批评） | 赵鼎新五种时间观（建构） |
|----------------------|------------------------|
| 1. 无时间的横向比较 | **积累性发展时间**（经济/军事权力） |
| 2. 固定时间下的纵向比较 | **多样性时间**（意识形态权力） |
| 3. 循环时间 | **主宰性政治时间**（政治权力） |
| 4. 进步/终极时间 | **道家时间**（覆盖性法则——强弱转换） |
| 5. 多元史观（无规律多样时间）| **复合时间**（以上四种的复合） |

## 理论抱负

赵鼎新的意图不仅是方法论贡献：

> 树立具有中国特色的时间本体论是发展具有中国特色的社会科学的一个方向。（摘要）

## 对已有时间哲学的超越

赵鼎新拒绝两种极端：
- **结构决定论**（无异时间的社会学横向比较）——忽略事件和人物的转折作用
- **极端多元史观**（无方向的碎片化历史）——只长知识不长智慧

**替代方案**：历史由四种社会权力（经济/军事/意识形态/政治）的不同时间性复合而成，受道家时间（强弱转换法则）的长时段调节——既有规律（非无方向），又没有终极目标（非进步史观）。

**Why:** 此卡确立了赵鼎新全文的论证结构——十种时间观的对称性批评与建构，以及历史社会学的本体论基础。这对于已有的德勒兹时间卡组（李科林/D.W.Smith/Olkowski等）是一个来自社会学的"竞争性时间本体论"。

**How to apply:** 从事任何历史/比较研究前，先反思自己默认的时间观：是无时间的横向比较？进步史观？还是多元碎片？然后有意识地运用赵鼎新的复合时间框架来组织叙事。与[[wang-meng-time-space-world-politics-methodology]]（时空之维）交叉——王存刚讨论的是时间在IR方法论中的功能，赵鼎新讨论的是时间本体论本身——二者在"时间性"层面可产生深层对话。
