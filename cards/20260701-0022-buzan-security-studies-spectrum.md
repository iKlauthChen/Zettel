# 国际安全研究的概念谱系——布赞

## 安全概念的三组框架

1. **补充性概念**：战略、威慑、遏制、人道主义——揭示安全具体含义
2. **平行性概念**：权力、主权、认同——揭示广义政治层面的安全
3. **竞争性概念**：和平、风险、意外——替代安全概念

## 认识论三类型

| 类型 | 核心 | 脉络 |
|------|------|------|
| **客观安全** | 是否存在确切威胁，物质界定 | Wolfers(1952)→Walt(1987)→Jervis(1976) |
| **主观安全** | 是否有被威胁的感觉，含认知/误解 | 未摒弃客观定义，经主观"过滤" |
| **话语安全** | 安全是言语行为，主体间建构 | 哥本哈根学派安全化理论核心 |

## 安全研究11学派谱系

常规建构主义 / 批判建构主义 / **哥本哈根学派（安全化）** / 批判安全研究 / 女性主义 / 人的安全 / 和平研究 / 后殖民主义 / 后结构主义 / 战略研究 / （新）现实主义

## 地区安全复合体理论（布赞维夫）

**定义**：一组单位，其主要安全化/去安全化进程紧密相联，不能彼此分开分析。

**三种社会模式**：冲突形态→安全机制→安全共同体，对应Wendt的霍布斯/洛克/康德三种结构。

**亚洲**：东北亚最先形成RSC（1895年日本击败中国），二战后代际叠加遗产。东亚在可预见的将来不太可能形成安全共同体（缺乏共同文化/制度）。

## 我的判断

布赞的安全研究谱系是迄今为止最系统的安全概念分类学，但"11学派"的划分可能同时具有过度和不足：过度在于有些"学派"之间的差异（如常规建构主义和批判建构主义）在实际研究中越来越模糊；不足在于它遗漏了安全化理论内部日益重要的分支（如以Balzacq为代表的"社会学转向"）。布赞维夫的RSC理论的一个被低估的贡献是将"安全地区化"与全球层面的极结构区分开来，为分析中美竞争在地区的差异化表现提供了比华尔兹结构现实主义更精细的工具。
