# 尹继武 & 刘颖哲 (2025) ——国家安全的人格基础：一项理论性分析

**来源**: 《国际安全研究》2025 年第 3 期，pp. 86-106  
**作者**: 尹继武、刘颖哲（中国人民大学国际关系学院）  
**核心论题**: 为主流国家安全理论补充微观心理基础——系统构建**人格特质**（personality traits）影响国家安全信念、安全偏好、安全决策和安全行为的理论框架，辨析领导人个体人格与群体信念聚合在国家安全诸环节中的作用机制。

---

## 卡 1：人格-国家安全理论框架总览

**核心论点**: 国家安全理论需要微观心理基础——人格特质经由多主体机制影响安全体系的四个环节（信念→偏好→决策→行为），形成从微观人格到宏观安全的因果链。

**理论定位**:
- 主流国家安全研究：注重宏观主体（国家/体系）和物质性要素 → 忽视人性维度
- 政治心理学补充：为国家安全的"主观性"（威胁感知、安全认知）提供微观基础
- 人格 ≠ 决定因素，而是**调节变量**——介于外部刺激和政治反应之间

**人格特质定义**: 个体内部心理特质和机制的集合，源于遗传与环境共同作用。**相对稳定、可观察**的性格和倾向，使行为表现出跨时间/跨情境一致性。

**国家安全四环节的人格影响矩阵**:

| 环节 | 个体（领导人） | 群体（精英/民众） |
|------|-------------|----------------|
| **安全信念** | 人格特质三维度 → 安全思维 | 国民性格/战略文化 → 安全文化 |
| **安全偏好** | 动机需求+敏感性/包容性 → 目标与手段偏好 | 价值取向聚合 → 意识形态 |
| **安全决策** | 自我防御机制(危机)+领导风格(常规) | 小集团思维/多元思维/群体极化 |
| **安全行为** | 人格-情境交互效应 | 政治化/工具性运用人格 |

**Why:** 为国家安全理论补充了目前最系统的心理学"人"的维度——结构清晰、可操作、可检验，为后续实证研究提供了概念框架。

## 卡 2：安全信念的人格基础——三大人格维度

**核心论点**: 领导人的人格特质通过**人与环境、人与工作、人与他者**三大维度塑造其整体安全观；群体信念则凝结为国民性格与战略文化。

**领导人人格三维度**:

**1. 环境维度——对约束的态度**
- 环境不敏感型：浮夸自恋、不羁善变 → 坚持信仰体系、挑战客观约束
  - 例：西奥多·罗斯福（舆论力量+强硬形象）、特朗普（Truth Social+情绪张力）
- 环境敏感型：低调谦逊、顺从尽责 → 尊重外部约束、谨慎行事

**2. 工作维度——对政治事务的态度**（巴伯分类）
- 精力投入（积极/消极）× 从政感受（正面/负面）→ 四种总统性格
- "积极-负面"型（胡佛）→ 权力追求+无满足感 → 难以有效应对安全挑战
- "积极-正面"型（罗斯福）→ 热情亲民+随机应变 → 增强国家认同

**3. 他者维度——关系处理倾向**
- **不信任特质**：零和博弈思维，放大外部威胁
- **内群体偏见**：借助外部威胁动员内部团结
- **宜人性**（大五人格）：温和谦逊，易获得支持

**群体信念**:
- **国民性格**: 东方（整体式思维/互依自我/集体主义）vs 西方（分析式思维/独立自我/个人主义）→ 形成"安全状态"vs"安全威胁"范式
- **战略文化**: 各国古代典籍与战略文本积淀的稳定战略观念（中国"单边默契"、俄罗斯"战略纵深"、美国四种战略传统）

**Why:** 将政治心理学中分散的人格研究（自恋、大五、不信任、控制点等）系统化为国家安全分析的三维框架——环境/工作/他者，可直接操作化为指标。

## 卡 3：安全偏好的人格动力——价值取向与手段偏好

**核心论点**: 人格特质通过需求动机塑造安全价值取向，通过敏感性/包容性塑造安全手段偏好——形成"追求什么目标"和"以什么方式追求"两条通道。

**价值取向的人格动力**:

**1. 领导人动机三类型**（Winter）
- **权力动机** → 控制特质、亲力亲为、追求名誉声望
- **成就动机** → 积极自律、任务导向、利用环境信息调整策略
- **关系动机** → 软弱、重视内部凝聚、寻求志同道合者
- 动机在不同环境下产生不同表现（权力动机者：危机→冒险展现决断力；和平→亲社会行为赢取声望）
- **认知闭合需求**（need for cognitive closure）调节动机发挥：高认知闭合=低概念复杂性=动机驱动偏见（沙龙案例）

**2. 民众价值取向**
- 威权主义人格：权力为中心，"对强者卑躬屈膝，对弱者肆意欺凌"
- 尽责性 → 右翼偏好；经验开放性 → 左翼偏好；厌恶敏感性 → 保守倾向
- **极端化风险**: 竞争性价值主张趋极化→超出制度框架→政权/本体安全危机（当代欧洲右翼民粹崛起）

**手段偏好**:

**1. 武力使用意愿**——受三种敏感性影响:
- **风险敏感性**: 内控型领导人更可能采取军事干预（里根 vs 老布什）
- **时间敏感性**: 对短期成本不敏感者更可能坚定立场
- **形象敏感性**: 高自我监控者愿为声誉而战

**2. 谈判合作意愿**——三层包容性:
- **环境包容性**: 自我克制→尊重规则；挑战约束→缺乏承诺
- **信息包容性**: 战略远见+概念复杂性→推动谈判进程
- **外群体包容性**: 低偏见→多边主义/情感信任

**Why:** "偏好"作为信念和行为之间的中介环节——回答了安全研究的一个核心问题：为什么面对相同的客观威胁，不同国家（领导人）会选择截然不同的应对策略。

## 卡 4：安全决策中人格的调节效应——危机/常规与个体/群体

**核心论点**: 人格因素在安全决策中通过**危机中的自我防御机制**和**常规中的领导风格**影响个体决策，通过**群体一致性倾向**和**群体极化/认同**影响集体决策。

**个体层面**:
**危机决策——自我防御机制**（拉斯韦尔继承弗洛伊德）
- 高强度精神压力激发防御机制 → 认知偏差（防御性回避：草率决策或拖延）
- 伴随消极情绪（焦虑/恐惧/愤怒）→ 苏伊士运河危机中的艾登
- **非始终负面**:适度压力也能促进自我资源整合

**常规决策——领导风格**
- **政策制定风格**: 信息收集+处理方式（杜鲁门的历史类比偏差）
- **人际关系风格**: 宜人性高→制度化内部沟通；偏执多疑→排斥异见（艾森豪威尔卓越组织能力 vs 反面）

**群体层面**:
**决策制定**:
- **小集团思维**（groupthink）：强势领导+服从性高的官员 → 过度乐观幻觉（猪湾事件）
- **多元思维**（polythink）：优柔寡断+低凝聚力团队 → 决策瘫痪（奥巴马"对手团队"）

**决策执行**:
- **群体极化**: 低自我意识者在同质信息环境中追随优势意见 → 网络时代极端情绪化舆论
- **群体认同**: 神经质高的人记忆外群体负面信息；威权主义人格更易偏见；自尊心强者支持冲突决策 → 极端认同推动民族主义/孤立主义

**Why:** 将人格研究从"谁在掌权"推进到"如何决策"——决策过程的结构性机制（小集团思维/群体极化）而非仅仅结果归因。

## 卡 5：人格-情境互动与安全行为

**核心论点**: 人格对宏观安全行为的影响是**条件性的**而非决定论的——在特定时空情境下发挥作用，并且会随时间变化和政治化操作而动态调整。

**人格的时空维度**:
**1. 特定人格特质在特定条件被激活**:
- 高体系压力 → 自恋型领导人寻求力挽狂澜（对抗高地位对手）
- 对手暴力挑衅 → 自我主导型领导人更可能武力回应
- 国内经济问题突出 → 不信任感强+低概念复杂性领导人更可能发动"转移注意力战争"
- 土耳其厄扎尔案例：自信特质使其在体系压力和制度约束下主动突破限制

**2. 人格随时间变化**:
- 生理条件（衰老→固执、认知灵活性下降、更高风险策略）
- 经验学习（小布什第二任期→更好谈判意识；猪湾事件→肯尼迪调整）
- 权力结构（长期政治斗争→同理心受损、权力动机强化、封闭式决策）

**多主体互动与人格政治化**:
- **性情归因**:冲突情境下将他者行为归因于人格特质，弱化外部约束
- **工具性人格表现**:领导人有意展现特质动员支持（泽连斯基——战时对内展现英雄形象，对外展现温暖关怀）
- **人格作为情报**:新领导人上台→人格分析成为重要安全情报（肯尼迪上台前的美苏相互分析）

**Why:** 人格分析最容易失误的地方就是"化约论/决定论"——本文的人格-情境互动框架提供了更为精细的分析策略。

**链接方向**:
- **→ [[20260622-0081-zhu-ning-copenhagen-school-security]]**：尹继武的人格框架与哥本哈根学派的"安全化"理论形成互补。哥本哈根学派分析"什么条件使安全化成功"（言语行为+听众接受+权威地位），人格理论回答"谁在什么心理驱动下发起安全化"。安全化"施动者"的人格特质（如不信任特质、权力动机、认知闭合需求）可以解释为什么有些领导人更倾向于将议题安全化。
- **→ [[20260622-0058-fang-guo-metaphor-scenario-nontraditional-security]]**：人格特质影响决策者对隐喻框架的选择——同一外部安全事件，自恋型领导人与宜人性高的领导人可能选择完全不同的隐喻策略（战争隐喻 vs 对话隐喻）。这说明隐喻图景不仅受制于社会文化语境和受众预期，还深受决策者个体人格的调节。
- **→ [[20260622-0060-hu-bo-chen-grey-zone-strategy]]**：灰色地带竞争中的"叙事竞争"需要分析受众(民众/国际社会)的群体人格聚合：当一国公民的威权主义人格和神经质水平较高时，灰色地带策略中的"妖魔化"叙事（道德隐喻）效果更强。人格-情境互动机制为灰色地带竞争的效果差异提供了微观解释。
- **→ [[20260622-0080-zhang-security-sense-neglected-dimension]]（张清敏安全感）**：尹-刘的"不安全感的深层来源"可在张清敏的框架中找到认知机制层面的解释——人格特质影响决策者的基本信仰和启发式思维偏好，进而调节安全感的产生。张清敏的四种错误知觉机制（刻板印象/归因/类比/隐喻）为尹-刘的"人格-情境互动"提供了具体的操作化路径。

**→ [[20260622-0068-qin-tan-geostrategy-ingest]]**：地缘位势（安全感知=结构压力×威胁紧迫性×精英集团施压）中"精英集团施压"环节的人格基础即由尹继武的群体信念聚合机制（威权人格服从、群体极化、小集团思维）来填充。两框架可对接：地缘位势提供结构性自变量，人格提供机制性中介变量。
