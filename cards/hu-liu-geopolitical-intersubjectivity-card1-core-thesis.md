# 论地缘战略的主体间性——核心论题

**出处：** 胡志丁、刘卫东《论地缘战略的主体间性——兼论中国地缘战略抉择》，《人文地理》2016年第31卷第3期，第122—127页。

## 核心命题

地缘战略的构建、制定和实施都必须遵循**主体间性**（intersubjectivity）。

> 主体间性就是主体之间的交互作用。主体间性具有物质性、实践性、历史性和社会性，其存在于集体主体之中，可以通过"教化"获得。（摘要）

## 核心分析框架

> 真正成功有效的地缘战略应该是充分考虑主体间物质、精神交往、历史、社会等交互关系的战略。（p.126）

```
  经典地缘战略理论的隐含假设         主体间性视角的修正
  ┌──────────────────────┐       ┌──────────────────────────┐
  │ 客观的、普适的地理法则 │  →   │ 特定"自我"与"他者"的二元 │
  │ 适用于所有时空        │       │ 结构——不可复制           │
  └──────────────────────┘       └──────────────────────────┘
```

## 主体间性的四特征

| 特征 | 含义 | 在地缘战略中的体现 |
|------|------|-------------------|
| **物质性** | 交互关系以物质交往为基础 | 美苏军事实力对比决定了冷战形式而非热战（p.125） |
| **实践性** | 交互关系通过实践产生 | 苏联的扩张历史使其形成本能的不安全感→对外政策扩张性 |
| **社会性** | 主体是社会性存在 | 美苏国家性质差异（社会主义vs资本主义）建构了敌对关系 |
| **历史性** | 交往关系是具体历史的 | 俄国的扩张史、二战惨痛经历→不安全感→对抗性 |

## 对经典地缘政治理论的重新解读

胡志丁和刘卫东用主体间性重新审视了四个经典理论，揭示了每个理论的"自我-他者"预设：

| 理论 | 构建者（自我） | 隐含的他者 | 关键证据 |
|------|---------------|-----------|----------|
| **国家有机体论**（拉采尔） | 德国 | 其他西欧国家 | 德国人口从法国79.7%增长到法国1.56倍（1820-1913）→人口压力需扩张 |
| **陆权论**（麦金德） | 英国（海上国家） | 俄国（陆上强国） | 英国作为海上霸主只受陆上攻击威胁——俄国中亚扩张 |
| **海权论**（马汉） | 美国 | 英国（当时海上霸主） | 1890年代美国海军仅有142艘过时木船；马汉对英西/英荷海战的分析 |
| **边缘地带论**（斯皮克曼） | 美国 | 欲控制欧亚大陆边缘地带的国家 | 两次世界大战经历——真正威胁来自边缘地带而非心脏地带 |

## 主体间性的四特征

> 地缘政治理论预设了"自我"和"他者"。"自我"与"他者"地位或称两者的"二元结构"的不可互换性决定了地缘政治理论的不可复制性。（p.123）

## 核心案例：同一围堵战略的不同结果

### 美国围堵苏联 → 苏联解体
- 美苏实力相当，国防开支接近（表4）
- 苏联有扩张历史（彼得大帝→叶卡捷琳娜→斯大林）
- 国家性质对立（社会主义vs资本主义）
- **主体间性互动充分**：遏制→反遏制→军备竞赛→拖垮

### 美国围堵中国 → 中国持续崛起
- 中美实力差距大（1996年中国军费86亿美元 vs 美国2640亿美元；2012年1060亿 vs 6880亿美元）
- 中国无扩张称霸历史
- 中国坚持和平发展，不"应战"
- **主体间性互动缺位**：美国围堵→中国冷处理→围堵失效

> 围堵地缘战略要起效必须要"围堵方"和"被围堵方"互动。然而改革开放以来，中国制定的外交政策是独立自主的和平外交政策……在这样的对外大政方针指引下，中国在反围堵方面行动肯定是言语上要多于实际行动。（p.126）

## 中国地缘战略三原则（p.126）

1. **"冷处理"**：避免中美直接正面对抗
2. **"平行"地缘战略**：不与美国地缘战略直接对抗，保证中美没有正面冲突——"一带一路"是最佳体现
3. **适度发展海军**：海陆兼备决定不宜大规模发展海军；海军目标限于保障海洋权益和海外贸易安全，而非战胜美国海军

**Why:** 本文将西方哲学中的"主体间性"概念（胡塞尔/哈贝马斯等）引入地缘战略分析，提供了一个不同于传统权力政治和地理决定论的视角：地缘战略的有效性取决于"自我"与"他者"之间的互动关系，而非单方面的战略规划。

**How to apply:** 分析地缘战略时，不仅问"我方的目标是什么"，还问"对方的主体性特征是什么"以及"双方的互动关系将如何演变"。美国围堵苏联成功而围堵中国失败的关键不是美国的战略不同，而是**被围堵方的主体性（实力/历史/认知）不同**。与[[yang-complex-system-card5-false-as-true]]（杨思灵"执假为真"）交叉——两者都关注战略效果中的"主体间性"维度：杨思灵讨论虚假意象如何在互动中被生产，胡志丁讨论战略本身的效果取决于互动双方的特征。
