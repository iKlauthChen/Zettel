# 海权作为"决定性战略砝码"

**出处：** 姜鹏《海陆复合型地缘政治大国的战略选择与崛起成败》，第29—33页。

## 海权的战略角色

姜鹏用了一系列比喻来描述海权国在陆权竞争中的角色：

| 比喻 | 含义 |
|------|------|
| **"托盘中的砝码"** | 海权国不一定是战场上的主力，但可以倾覆力量平衡 |
| **"战略助攻手"** | 海权国通过封锁/切断海外资源，使陆权对手难以持久 |
| **"决定性重量"** | 引自Sullivan(1994)——海权的战略不是正面决胜，而是在关键处施加重压 |
| **"离岸平衡手"** | 海权国根据威胁认知决定站在哪一边 |

**俾斯麦的精彩回答**：当被问及如果英国军队在德国海滩登陆将如何应对，俾斯麦说："我会叫当地的警察把他们都抓起来。"（p.29）——这一回答精准说明了海权国在陆上的脆弱性，但同时也反证了海权国在**海上**的主宰性力量。

## 历史验证：与海权结盟的"不败纪录"

**表3的核心发现（p.30）**：

| 战争 | 与海权结盟的一方 | 结果 |
|------|-----------------|------|
| 三十年战争 | 波旁王朝（与荷兰/英国结盟） | 胜 |
| 西班牙王位继承战争 | 哈布斯堡家族（与英国结盟） | 平局 |
| 七年战争 | 普鲁士（与英国结盟） | 平局 |
| 拿破仑战争 | 俄普奥（与英国结盟） | 胜 |
| 克里米亚战争 | 拿破仑三世法国（与英国结盟） | 胜 |
| 普法战争 | 普鲁士（争取英国友谊中立） | 胜 |
| 一战 | 法俄意（与英/美结盟） | 胜 |
| 二战 | 中法苏（与美/英结盟） | 胜 |
| 冷战 | 西欧+中国（与美国结盟） | 平局 |

> 凡是同海权国结盟的陆权国在"区域陆权"竞争中即便没有赢得全面胜利，也会至少取得平局的结果。而那些背负海陆两线巨大战略负担的崛起国则可能因有限的战略资源与无限的安全困境之间的矛盾而面临被联合扼杀的命运。（p.27）

## 海权国的作用机制

姜鹏通过考察英法七年战争、拿破仑战争等案例，提炼出海权国在陆权竞争中的三个具体作用：

**1. 远洋打击与近岸封锁**（p.29）
- 切断敌对同盟获得海外资源的可能
- 摧毁敌对国家孤悬海外的战略基地
- 在长期消耗战中通过控制航道决定资源配置

**2. 消耗战中的决定性支撑**
- 海权国可以为陆权盟友提供长期战争的资源支持
- 使原本势均力敌的陆权平衡发生倾覆

**3. 制衡威胁意图的组合逻辑**
> 海权国参与制衡的根本逻辑不仅是依据权力结构大小这样简单……而是更关切带有威胁意图的权力本身。（p.31，引自Walt）

**核心**：海权国更在意哪一方对其海上霸主地位与现有秩序框架**最具挑战意图**。这意味着如果崛起国能够通过战略选择（如主动限制海权发展）表明自己无意挑战海权国地位，就能大幅降低被制衡的风险。

## "海上生命线迷思"的批评（p.32-33）

姜鹏批评了"如果不能掌握海域制海权，海上生命线就有被扼杀风险"的流行观点，提出三点：

1. **和平时期无法靠单一国家覆盖**——全球航线安全一直由霸权国及其联盟体系多边维系
2. **孤悬海外的军事基地可能是负担而非资产**——一战德国在太平洋的基地最终成为"权力资源分散的重要负担"
3. **控制某个海峡无法确保立体的运输网络**——即便控制了马六甲，浩瀚印度洋上的航线如何保证？

> 如果没有取得与海权霸主对等的实力，那么即便是拥有全球第二的海上军事力量，在同海权霸主及其全球联盟体系的对抗中也是无济于事的。（p.30）

## 对中国"不结盟"政策的反思（p.35）

姜鹏批评了"不结盟"政策的局限性，主张中国应更务实地处理与海权国（美国）的关系：

> 奉行"不结盟"政策的中国仍然在实力上远逊于美国的全球联盟体系。在这种情势下，奉行"区域陆权战略"，并将中国在体系中的角色与身份塑造成消极的海上安全"搭车者"、"建设者"或合作安全的"联盟者"往往比不假思索地坐实来自地缘政治竞争对手们无端指责——现有秩序"潜在挑战者"——更能够降低来自体系的安全压力。（p.35）

## 战略自制：尤利西斯与塞壬（p.33）

姜鹏用沃特豪斯名画《尤利西斯与塞壬》的隐喻来结束：

> 有的时候——对于崛起国来讲——把自己某些方面束缚起来往往能够获得更大的自由与安全。（p.33）

> 判断一个民族是否拥有成熟的政治思维的一个重要标志就是观察其在成就面前是否拥有冷静的耐心与充足的节制力。（p.33-34）

**Why:** 姜鹏的"海权作为决定性砝码"机制与秦立志的德国失败分析形成互补——秦立志从德国国内决策角度解释失败，姜鹏从**体系互动**角度解释：德国不是因为国内决策失误而失败，而是因为选择了"海陆并举"战略→触发同色竞争→英法俄形成反德海陆联姻→德国被海陆夹击而败。

**How to apply:** 评估中国（或其他崛起国）的战略处境时，用"同色竞争"检视：中国与周边国家是否构成"同色"（都追求陆权？都追求海权？）？与霸权国美国是否构成"同色"（都在扩张海权？）？如果是，则需要通过战略约束来改变颜色定位。与[[20260624-0088-qin-land-maritime-card3-germany-case]]（秦立志德国案例）交叉。
