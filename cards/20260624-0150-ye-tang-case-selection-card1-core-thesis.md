# 基于因果机制的案例选择方法——核心论题

**出处：** 叶成城、唐世平《基于因果机制的案例选择方法》，《世界经济与政治》2019年第10期，第22—47页。

## 核心问题

> 对于因果机制的重视是社会科学方法最为重要的革命之一，然而迄今多数研究的案例选择方法却仍然是基于因素的，这就造成了案例选择和案例研究方法之间的脱节。（摘要）

**解决方案**：提出一个**基于因果机制的案例选择方法**（MBCSM）。

## 选择性偏差的重新定义

**传统观点**（定量学派）：选择性偏差是必须避免的，案例选择应追求随机性。

**本文观点**（p.23）：
> 对于"提供一个基于因素+机制的解释"或更加特定的"发现或检验机制"的目标来说，**"选择性偏差"不仅是不能避免的，而且是必要的。**

## 三种案例与因果解释效果（图1，p.29-30）

```
Y ↑
  │         ● A(X=1,Y=1) 正面案例
  │           
  │    ● C(X=0.5,Y=0.5) 半负面案例
  │
  │                       ● B(X=0,Y=0) 传统负面案例
  └─────────────────────────→ X
```

| 案例 | 含义 | 信息量 |
|------|------|--------|
| **A（正面案例）** | X=1,Y=1——机制完整展现 | 对因素和机制都有强劲贡献 |
| **C（半负面案例）** | X=0.5,Y=0.5——机制进行到一半被中止 | 对因素和机制都有强劲贡献——核心创新 |
| **B（传统负面案例）** | X=0,Y=0——机制从未启动 | 对因素有贡献，对机制贡献非常有限 |

## 半负面案例的核心作用

> 所谓"半负面案例"，是指那些"存在可以让结果出现的机制，但却因为各种因素的作用而使得机制中止，最终结果未能出现"的案例。（p.28-29）

**碳氧化类比**（p.32-33）：
- 正面案例（氧气充足）：C + O₂ → 2CO → 2CO₂
- 半负面案例（氧气不足）：C + O₂ → 2CO（**停留在中间产物CO**）
- 传统负面案例（无氧气）：C（无变化）

**关键**：半负面案例中的中间产物CO极大提高了化学反应机制的可信度——它证明了机制确实存在，只是在特定阶段被抑制。

## 四步操作规则（p.35-39）

| 步骤 | 内容 | 核心工具 |
|------|------|----------|
| **步骤1** | 确立理论适用的时空范围，规划样本总体 | 范围条件（scope condition） |
| **步骤2** | 基于因素的类型学检验——确保多数案例符合理论预期 | 布尔代数/类型学表格（2×2） |
| **步骤3** | 从正面案例中筛选具有相同机制的**路径案例**（pathway case） | 过程追踪——识别各阶段标志性变化 |
| **步骤4** | 选择半负面案例进行比较——识别那些"机制被部分激活但中止"的案例 | 因果过程观察值比较 |

**步骤2的八种类型（表2，p.37）**：

| 类型 | X₁ | X₂ | Y | 性质 |
|------|----|----|----|------|
| 1 | 1 | 1 | 1 | ✅正面案例 |
| 2 | 1 | 0 | 0 | ✅负面案例 |
| 3 | 0 | 1 | 0 | ✅负面案例 |
| 4 | 0 | 0 | 0 | 不可能案例（结果几乎不会出现） |
| 5 | 1 | 1 | 0 | ❌异常案例（偏离预期） |
| 6 | 1 | 0 | 1 | ❌异常案例 |
| 7 | 0 | 1 | 1 | ❌异常案例 |
| 8 | 0 | 0 | 1 | ❌异常案例 |

## 四种案例类型的层次关系（图2，p.34）

```
      所有案例
     /        \
  同质案例    非同质案例（排除）
   /    \
符合预期    异常案例
 /    \
正面   负面（含不可能案例）
          /    \
    半负面    传统负面
   （机制中止）（机制未启动）
```

**Why:** 本文是唐世平团队方法论系列的核心之作——它明确提出了一个不同于定量（随机性）和传统定性（因素比较）的第三条案例选择路径：**基于机制的、有意的"选择性偏差"**。对于任何从事案例研究的IR学者，这是一个从根本上改变研究设计的方法论工具。

**How to apply:** 设计案例研究时，在传统的正反案例对比中加入**半负面案例**——那些"差一点就发生"的结果（如差点击败霸权但仍然失败的挑战者、接近但未能实现民主转型的国家、几乎但未爆发冲突的危机等）。
