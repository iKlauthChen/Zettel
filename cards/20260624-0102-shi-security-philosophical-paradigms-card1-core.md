# 国际安全的基本哲理范式

**出处：** 时殷弘《国际安全的基本哲理范式》，《中国社会科学》2000年第5期，第177—188页。

## 核心框架

时殷弘借鉴马丁·怀特（Martin Wight）三大国际关系思想传统——**现实主义、理性主义、革命主义**——来构设国际安全的基本哲理范式。

## 国际安全的哲理定义：自然法基础（p.178-179）

借鉴格老秀斯的自然法观念（五项根本义务）作为安全定义的哲理基础：
1. **不侵占他人所有**（包括生命、心灵、躯体、外在所有物）
2. 将所侵占的归还原所有者
3. **信守承诺**
4. 补偿由自己的过错给他人造成的损失
5. 按照过错施以恰当的惩罚

**个人-国家类比**：个人安全→国家安全；个人社会→国际社会。

**安全的最简界定**（借用洛克式）：个人安全=生命、自由、财产权利无虞侵害；国家安全=主权、独立、领土完整；国际安全=国家普遍安全前提下国际社会处于和平、有秩序的状态。

**布尔的国际秩序三基石**（p.179-180）：
1. 国家免遭外部暴力侵害
2. 国际承诺和协议得到遵守
3. 国家主权和领土完整得到保持

+ 国际社会本身的保存（阻止霸权+防止非国家行为体破坏国家地位）

## 三大传统的安全观

### 1. 现实主义传统：均势（p.180-183）

| 维度 | 内容 |
|------|------|
| **哲学起点** | 霍布斯自然状态——国家间关系=持续的战争姿态 |
| **安全机制** | **均势**——阻止霸权的国际力量分布是起码国际安全的首要保障 |
| **自动生成论** | 霸权追求必然自动招致制衡——时殷弘对其提出严肃质疑 |

**对均势自动生成论的批评**（p.182）：历史上对霸权的反应远非总是制衡，还包括容忍、规避、屈从和"搭车"；制衡努力并不总随霸权威胁增大而增强；霸权国可以提供利得使别国宁愿搭车而不制衡。多极格局下制衡更不容易。

### 2. 理性主义传统：商业和平+民主和平+法制和平（p.183-186）

| 机制 | 内容 | 时殷弘的评判 |
|------|------|-------------|
| **民主和平论** | 民主国家间极少战争 | 民主国家数量过少；民主国家间的战争危机并非罕见；大众也可能怀抱极端意识形态 |
| **商业和平论**（互相依赖和平论） | 自由贸易降低冲突概率 | 互相依赖既可能促进协调也可能引发争斗——汉密尔顿早有有力驳斥 |
| **集体安全/法制和平** | 国际组织保障普遍安全 | 假设所有国家在任何情况下都有共同利益和共同行为——"有违常理" |

**理性主义的合理成分**（p.185-186）：但时殷弘也承认，二战后西方发达国家间的长期和平、区域安全组织的作用——自由主义国际主义取得了"局部成功和有限认证"。

### 3. 革命主义传统：安全范畴的急剧扩展（p.186-187）

革命主义视野下的国际安全≈世界安全/人类安全，具有强烈的道德内涵——"涤荡所有人间非正义"。

**世界秩序学派**（1970s-80s）首次发动：核战争危险、人口危机、资源危机、生态危机、南北经济差距、人权压制等。

**安全定义的扩展轨迹**（p.186-187）：
> 乌尔曼(1983)→马修斯(1989)→种族冲突/难民/恐怖主义/金融危机/核扩散/电脑黑客袭击 → 国际安全的含义和所涉范围大为扩大

## 三层次安全框架（p.187）

```
       全球安全（革命主义视野：全人类共同体）
           ↑↓
       国际安全（理性主义+现实主义：主权国家间）
           ↑↓
       国家安全（现实主义：主权/独立/领土完整）
```

**三条兼顾协调原则**：
1. 国家个体利益与国际社会总体利益间尽可能兼顾和协调
2. 人类共同体利益与国家或国际社会利益间尽可能兼顾和协调
3. 确实难以兼顾时，当依具体情况而定——但"全球安全尚无足够有力和独立的权威代表，因而需予格外的关照"（p.187）

**Why:** 本文提供了理解国际安全的"元框架"——三大传统不是互相排斥的立场，而是不同层次安全问题的哲理基础。现实主义适合解释国家安全层次的均势逻辑，理性主义适合解释国际安全层次的制度合作，革命主义适合解释全球安全层次的跨国挑战。三者各有其适用的范围和局限。

**How to apply:** 分析任何安全问题时，先问：这一问题在哪个安全层次（国家/国际/全球）最突出？则需要对应的哲理传统来理解——国家安全↔现实主义、国际安全↔理性主义+现实主义、全球安全↔革命主义。与[[20260622-0081-zhu-ning-copenhagen-school-security]]（哥本哈根学派安全化）交叉——时殷弘从哲理层面（三大传统）讨论安全，哥本哈根学派从机制层面（安全化）讨论安全——两者互补：时殷弘提供了安全的思想史基础，哥本哈根学派提供了安全的操作机制。
