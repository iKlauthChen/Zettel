# 涌现结构的互反因果性模型为社会科学提供了比方法论个体主义和整体论更精确的描述语言

Bonta 和 Protevi 将涌现（emergence）定义为一个系统能够产生"聚焦性系统行为"（focused systematic behavior）的过程，其中全局结构通过约束组成成分的局部作用来实现自身。他们借鉴 Thompson 和 Varela（2001）的"互反因果性"（reciprocal causality）概念来描述涌现的运作方式：局部到全局的"上向因果性"（upward causality）产生了系统化的聚焦行为；全局到局部的"下向因果性"（downward causality）约束了组成成分的局部交互。

> "An emergent structure is that which enables focused systematic behavior through constraining the action of component parts."

他们谨慎区分"认识论涌现"（epistemological emergence——只因为我们无法计算系统行为的聚合结果）和"本体论涌现"（ontological emergence——涌现是世界的真实特征）。只有后者才值得因果承诺。Bonta 和 Protevi 倾向于将本体论涌现与之前章节勾勒出的复杂系统本体论（虚拟/现实、吸引子、分岔）联系起来，但承认证实本体论涌现的存在标准仍然是一个未决的科学问题。

## 我的判断

"互反因果性"是本书对社会科学最实用的贡献之一。它为"结构限制能动性，能动性改变结构"这个老问题提供了精确的语言：上向因果是说"个体行为的聚合会产生系统层面的秩序"（这微观经济学也能说），下向因果是说"那个秩序一旦形成就反馈回来约束个体的可能行为"。后者才是整体论的核心直觉，但用非神秘化的因果语言表达出来了。不过，正如作者坦率承认的，区分"仅仅是认识论的"和"真正的"涌现仍然困难。我的立场是：在社会系统中，即使涌现最终只是认识的而非实在的，互反因果性作为分析启发法（heuristic）仍然是有用的——它至少要求分析者不能只从宏观数据读趋势或只从微观行为推断。
