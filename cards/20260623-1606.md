# Multiplicities: Deleuze's Riemannian manifold vs Badiou's Cantorian set theory — the core ontological disagreement about how to think space and difference

## 原文要点

> "It was a decisive event when the mathematician Riemann uprooted the multiple from its predicate state and made it a noun, 'multiplicity'" (ATP 504-5). Whereas the traditional Euclidian multiple is simply a countable heap of the same things, "Riemannian space is pure patchwork. [...] Heterogeneous, in continuous variation, it is a smooth space, insofar as smooth space is amorphous and not homogeneous" (ATP 485).

> A Deleuzian multiplicity is defined not by its elements [...] It is defined by the number of dimensions it has; it is not divisible, it cannot lose or gain a dimension without changing its nature.

> For Badiou, instead, a more meticulous excision of oneness must build up some kind of formal system. He claims to be able to guarantee a fully immanent atheist and revolutionary constructibility of social reality, centered around the unprecedented event that arrives from outside the purview of philosophy.

## 我的判断

Saldanha 对德勒兹-巴迪乌之辩的处理是全书最有理论张力的部分之一。德勒兹的多重性来自黎曼流形（Riemannian manifold）：一个 n 维空间，其变形和关系是内在于空间的，不能在不改变其性质的情况下被分割或增减维度。而巴迪乌的多重性来自康托尔集合论（Cantor's set theory）：存在通过纯数学的"计数即一"来构成，集合的元素同质且边界清晰。德勒兹认为集合论是"公理性的"，属于国家科学和资本的本体论工具；巴迪乌指责德勒兹低估了康托尔对哲学的变革意义——存在无穷的阶数，但不存在"一切的一"，这意味着德勒兹的一元论（One-All）实际上残余了泛神论。这个争论对空间理论有直接含义：如果空间由黎曼流形构成，空间就是连续的、质的、异质的（平滑空间）；如果空间由集合构成，空间就是离散的、量的、可数的（条纹化空间）。Saldanha 在这一点上似乎倾向于德勒兹但留下了开放空间："Perhaps A Thousand Plateaus too easily conflates axiomatics with the status quo and should have considered the minor uses of mathematical logic." 这是一个诚实的让步。
