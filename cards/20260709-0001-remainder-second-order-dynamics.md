# 剩余的二阶动力学：被排除者通过自我叙事再生产剩余

## 论题展开

从特朗普2026年4月公开点名章家敦（"根本不知道自己在说什么"），到余茂春以"长期预案"论自我辩护，两个事件共同揭示了一个被忽略的理论维度：**剩余不仅是安全话语排除的产物，也是被排除者通过自我叙事主动再生产的产物。**

"剩余安全"的基础框架（[[20260624-0075-original-batch4-surplus-security|核心命题]]）已经论证：任何系统的运作都产生超出其自我维持所需的"多余产物"，这个产物既是系统的驱动力，也是系统的症状。但被排除者在其中的角色被默认为**被动的**：他们是系统排除的剩余，是安全体制运作后留下的碎片。

余茂春的"长期预案"论表明情况更复杂。当被排除者说"我们的观点不是错误的预言，而是为未来储备的预案选项"时，他们在做三件事：

1. **证伪免疫**：将预测失败重新解释为"备而未用"——每次崩溃论落空都转化为"危机推迟而非消除"，预言不被证伪。
2. **价值重述**：将"被排除"本身重述为战略性位置——"我们不在权力中心，所以我们能自由储备权力中心不敢储备的选项"。排除不是失败，而是优势。
3. **存在再生产**：通过元叙事确保自身在被排除后仍有正当性继续发声，由此产生新的剩余——一个不被采纳的"预案"堆积层。

这个过程构成一个**自指循环**：

```
安全话语 → 二阶排除（排除过于透明的威胁认定者）
  → 被排除者生产证伪免疫的元叙事 → 新叙事需要新的排除操作来维持边界
  → 更多剩余（未被启用的储备威胁叙事/被忽视的战略建议）
  → 循环继续
```

章家敦每次发声都在生产一个新的"未被采纳的崩溃时刻"；余茂春的每段辩解都在生产一个新的"被主流忽视的战略储备叙事"。**剩余是自我增殖的，不仅从上到下（系统排除），也从下到上（被排除者的自我再生产）。**

## 我的判断

**"剩余的二阶动力学"与"认识权威"（[[20260626-1224-安全化失败当认识权威不支持|安全化失败案例]]）有结构性关联但指向不同问题。** Lieberman被排除是因为他的判断不符合AMAN的情报评估——他的威胁认定缺乏认识权威。而章家敦/余茂春被排除则是相反的问题：他们的威胁认定积累了太多"过期未交的预言债务"，以至于引用他们会**暴露政策依据的脆弱性**。前者是"不够权威"，后者是"权威已经破产"。两种排除路径指向同一个结构：安全话语需要维护一个**可信的威胁认定基础设施**，任何威胁认定者一旦威胁到这个基础设施的信誉，就会被排除——无论他是"不够格"还是"太透明"。

这个二阶动力学对剩余安全理论的意义在于：**剩余不仅是被排除的"威胁"，也是被排除者通过生存策略维持自身在场的方式。** 这意味着剩余安全理论需要追踪两个方向：向下的排除操作（安全话语排除什么）和向上的剩余再生产（被排除者如何通过自我叙事维持自身）。前者是**剩余的一阶**（系统排除的产品），后者是**剩余的二阶**（被排除者的自我再生产）。

一个开放的边界问题：被排除者的自我叙事是否可能超越证伪免疫，变成一种**主动的剩余生产**——即被排除者不再是为了维持存在而辩护，而是**为了制造新的剩余而不断产出预言**？如果如此，那么这个二阶动力学就不仅是防御性的，还是**生产性的**——被排除者从"系统剩下的碎片"变成了"剩余的生产者"。
