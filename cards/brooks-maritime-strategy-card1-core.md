# 海军力量与国家安全：美国海洋战略

**出处：** Linton F. Brooks, "Naval Power and National Security: The Case for the Maritime Strategy," *International Security* 11:2 (Fall 1986), pp. 58–88.

**作者背景**：美国海军上校，时任国家安全委员会国防项目主任。本文代表美国海军官方立场。

## 海洋战略的定义（p.58-59）

> 将传统的海军思维编织成一个连贯的概念——**在冲突初期、以强力、全球性、前沿部署的方式使用海上力量**，以威慑与苏联的战争，并在威慑失败后实现美国的战争目标。

战略=海军+海军陆战队领导层的专业共识，正式编纂于1982年内部分类文件，1986年1月在《美国海军学会会刊》公开。

## 三阶段战争图景（p.64-66）

| 阶段 | 内容 | 目的 |
|------|------|------|
| **第一阶段：威慑/向战争过渡** | 全球快速前沿部署（反潜部队突击部署、多航母战斗群集结、两栖部队登船）+ 征召预备役/海岸警卫队转隶海军 | 展示国家意志 + 全球性（不愿放弃任何区域）+ **可逆的威慑信号** |
| **第二阶段：夺取主动** | 在关键海域建立海空控制（挪威海、东地中海、太平洋接近区）→ 对苏联潜艇（含弹道导弹潜艇）发起攻击性反潜战 | **阻止苏联在其偏好的单战区集中兵力** |
| **第三阶段：将战斗推向敌境** | 航母航空兵和海军陆战队直接打击岸上目标 + 继续反潜战（含苏联SSBN） | 改变核力量对比 → **战争终止杠杆** |

**总体目标**（p.66-67）：
1. 拒绝苏联按其设想的战争方式作战——全球施压，表明战争既非短期也非局部
2. 摧毁苏联海军
3. 影响陆战——通过牵制其重新部署、确保己方增援和补给、直接运用航母和两栖力量
4. 以可接受的条件**终止战争**——通过威胁直接攻击本土或改变核力量对比

## 苏联海军战略的假设（p.62-63）

- 苏联希望一场短期的、决定性的胜利 → 集中在单战区（欧洲）
- 苏联海军角色：保护SSBN（弹道导弹潜艇）+ 保卫本土 → **防御性堡垒**
- 对西方海基核资产的摧毁（航母、SSBN、巡航导弹舰艇）是优先任务
- 传统任务（攻击增援/补给船队、支援陆军）明显是次要的

## 关键假设——对后续海军战略的持续影响（p.67-68）

| 假设 | 含义 |
|------|------|
| **美国在NATO之外也有重要军事利益** | 日本、韩国——在全球战争中需要保卫 |
| **中欧不会立即崩溃** | 海上力量需要时间生效——如果中欧几天内沦陷，海洋战略以及任何海权的替代方案都无法阻止——但**中欧的丧失是战役的失败，而非战争的失败** |
| **海基空中力量的最佳使用不在中欧** | 航母增援空军在中欧的直接贡献有限→用在侧翼和太平洋牵制苏联 |
| **核武器使用并非不可避免** | 海洋战略通过改变"核力量对比"使核升级对苏联越来越不具吸引力 |
| **保护海上交通线的方式必须不同于过去** | WWII贸易护航模式不适用——不能容忍二战式的消耗，也不能提供专用的战略原材料运输 |

## 对后续海洋战略/海权理论的持久贡献

Brooks（1986）中阐述的海洋战略的核心逻辑——**前沿部署、全球性、常规威慑、战争终止杠杆**——对后来美国海军战略有持续影响：

| 维度 | 持久遗产 | 当代对应 |
|------|----------|----------|
| **反介入/区域拒止（A2/AD）** | 海洋战略中"在苏联飞地外压制潜艇"的逻辑 | 今日反航母打击群/反介入环境 |
| **战争终止杠杆** | 通过威胁战略资产（SSBN）改变对方计算 | 核威慑/战略稳定辩论 |
| **全球性vs单战区** | 拒绝让对手选择战争的地域范围 | 印太战区vs欧洲战区 |
| **联盟效应** | 联盟的完整性不能牺牲 | 日美同盟/北约 |

**Why:** Brooks的文章提供了美国海军在冷战高峰期如何思考"海权 vs 陆权"问题的第一手官方观点。它不是地缘政治理论，而是**将地缘政治理论（马汉/麦金德/斯皮克曼）转化为可操作的国防战略**的实践文本。"战争终止杠杆"通过威胁战略核资产来获得谈判筹码的思路，在当时的战略研究中是一个创新——事后看也为理解核威慑的"第三轨"提供了文本。

**How to apply:** 在分析当代大国海上竞争（如中美）时，Brooks的核心逻辑——**海军战略不是独立的，而是国家大战略的海上组成部分**——仍然适用。但苏联案例中的许多假设（苏联海军的防御性堡垒化、苏联对单战区的偏好、北约一体化）在中国案例中并不自动成立。
