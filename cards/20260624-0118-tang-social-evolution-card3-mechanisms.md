# 从进攻性到防御性现实主义世界的四重转型机制

**来源**：唐世平 (2009). 国际政治的社会进化——从米尔斯海默到杰维斯. *当代亚太*, (4), 5-31.

## 根本机制：征服作为自我挫败

在进攻性现实主义世界中，国家获得安全的唯一方式就是削弱其他国家的安全——通过征服来获取安全。但这一机制将不可避免地自毁：

**"征服或被征服"的连锁反应**：
1. 一国追求征服且一些征服将取得成功
2. → 国家的数目将**减少**
3. → 国家的平均大小（土地、人口、资源）将**增加**
4. → 广阔的土地 = 更大的战略纵深
5. → 更多的人口 = 更大规模的军队
6. → 更多的财富 = 更多的军事资源
7. → **防御能力增强**
8. → **征服将变得更加困难**

> "当国家按照进攻性现实主义的中心逻辑——通过征服来获取安全——行动一段时期之后，他们的行为就将逐渐但不可避免地使进攻性现实主义的中心逻辑更加难以实现。"

**历史趋势**：
- 公元前1000年：世界上约60万个独立政治实体
- 现在：约200个
- 国家数量减少而国家规模增加的**核心机制是征服**——在相当长的时期里几乎没有间断的征服

## 三个辅助机制

### 机制一：对进攻性现实主义国家的不利选择

追求侵略的国家更可能在战争中被消灭。长期来看，系统选择更温和的国家。这类似于自然选择——进攻性国家被系统"淘汰"的频率更高，留下防御性国家。

### 机制二：对征服困难性的认知（社会学习）

> "国家从历史中学习——随着征服变得越来越困难，国家逐渐意识到征服的成本超过了收益。"

这是**拉马克遗传**的典型表现——国家通过学习意识到侵略的代价，这一认知在社会中传播并被制度化。随着征服的困难性深入人心，防御性现实主义思维逐渐成为主导。

### 机制三：主权和民族主义观念的产生及扩散

主权观念使征服失去合法性，民族主义使被征服人口更难统治：
- 威斯特伐利亚体系确立了主权国家体系——领土完整的规范
- 民族主义使征服带来的"同化成本"急剧增加
- 二者共同提高了征服的门槛，使防御性现实主义世界制度化

## 四重机制的协同

```
根本机制（征服→国家减少→防御增强）
    ↓ 强化
辅助机制1（侵略者更多被淘汰） → 辅助机制2（学习到征服困难） → 辅助机制3（主权/民族主义规范）
    ↓ 锁定
防御性现实主义世界
```

**Why**: 四重机制的创新在于将"征服自我挫败"的结构逻辑与社会学习、规范扩散结合起来。根本机制是**物质性的**（国家规模增大导致防御占优），辅助机制是**精神和制度性的**（学习、主权规范、民族主义）。物质力量和精神力量共同作用驱动系统转型。这一框架超越了纯粹的物质决定论（现实主义）和纯粹的观念决定论（建构主义）。

**How to apply**: 用于分析当前国际体系的未来演化方向——如果防御性现实主义世界是通过征服困难性和规范扩散建构的，那么新技术（如A2/AD、网络战、太空武器）可能改变征服的成本计算，而新规范（如"保护的责任"）可能挑战主权规范。中国在南海的人工岛礁建设和填海造地可视为改变"征服困难性"计算的尝试。
