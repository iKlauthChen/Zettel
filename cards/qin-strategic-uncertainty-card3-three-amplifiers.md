# 威胁预期的三大放大机制

**出处：** 秦立志《战略不确定性与安全困境的生成机制》，第83—84页。

## 机制概览

三大干预变量可单独也可交互作用，放大初始阶段形成的负面威胁预期，导致原本低烈度的安全困境演变为高烈度安全困境。

### 机制一：因果漏斗（Causal Funnel）

```
  在漏斗中回溯得越远：
      ┌─────────────────────────────────┐
      │ 客观因素（实力对比等）重要性增大 │
      └─────────────────────────────────┘
                     ↓
  在接近窄窄的口子时：
      ┌─────────────────────────────────┐
      │ 主观因素（意图层面）重要性增大   │
      └─────────────────────────────────┘
```

**运作逻辑**：当威胁预期负面化时，对因果漏斗的回溯更加偏重客观情势（实力对比、军事力量等硬指标），而非主观意图研判。这导致从"判断对方决心"的不确定性转向"只盯住实力变化"的确定性，进而夸大对手威胁。

**关键危险**：过于侧重实力维度，忽略意图维度的复杂性（p.83-84）。

### 机制二：回音壁效应（Echo Chamber Effect）

> 关于互动双方矛盾不可调和的观点，会在体系与国内政治系统内被迅速吸收，然后再通过某种渠道或形式反射出来，导致决策者周围充斥必有一战的信息。（p.84）

**双重发生条件**：
1. **行为体之间**：负面观点在互动中不断强化
2. **行为体内部**：基于此前冲突行为的记忆强化复仇或对抗决心

**扭转条件**：地缘政治的反作用力压倒国家承担制衡风险的作用力时，才可能让大国或迟或早选择战略收缩（p.84）。

### 机制三：镜像反应（Mirror Image Reaction）

> 国家倾向于将对手的行为归结为蓄意的，将己方的行为归结为情势性的。（p.84）

**后果**：双方激化行动—反应模型，都将对方的防御行动视为进攻行动，形成"恶意缺乏的安全困境"——即使双方都是防御性导向，也能产生类似进攻—对抗的效果。

## 三机制的交互效应

三者并非独立运作，而是彼此强化的增强回路：

```
            因果漏斗（盯实力）
                  ↕
  回音壁效应 ←────→ 镜像反应
  （信息强化）      （归因偏差）
```

- 因果漏斗为回音壁效应提供内容素材（只有实力数据而无意图分析的信息环境）
- 回音壁效应为镜像反应提供舆论温床（人际/媒体/精英舆论都在强化"对方恶意"论）
- 镜像反应反过来深化因果漏斗中的归因倾向（将对方所有行动都视为恶意）

**Why:** 三大放大机制解释了安全困境为何会"螺旋升级"——不是单一决策失误，而是系统性的认知和信息偏差在互动中自反馈强化。

**How to apply:** 在安全分析中，定期"交叉审查"三个维度：是否过度聚焦对方军力而忽略了其意图复杂性（因果漏斗）？决策者周围是否存在信息同质化环境（回音壁）？己方是否将对方行动全部归为蓄意而将自己的归为情势（镜像）？与[[yin-liu-national-security-personality-foundations]]（人格基础）中的"自我防御机制"交叉——放大机制在宏观互动层面发挥作用，人格在微观决策层面发挥作用。
