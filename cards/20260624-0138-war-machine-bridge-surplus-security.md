# 战争机器桥接卡——DG战争机器 vs 安全研究的战争机器

**2026-06-24** 从全套现有战争机器卡（Reid 6卡 + 恐怖/第七型 8卡 + 萨丹哈空间四重奏 + 胡波灰色地带7卡 + Dibb拒止威慑5卡）综合写成的桥接卡，存入 zettel/cards/。

## 核心论证

**严格区分两种"战争机器"：**
1. DG战争机器——不是军队，而是一种先于国家、外在于主权的游牧力量——逃逸线、偶然性、创造性的去层
2. 安全研究战争机器——国家的军事-威慑-暴力体制——预警、威慑、强制、暴力工具

两者的**混淆**是最大障碍。两者的**关系**通过"占有"（appropriation）概念连接：国家试图占有战争机器，但占有永远不完全。剩余安全从这个占有间隙中获得本体论锚定——安全化（国家占有威胁定义权的操作）也必然有剩余。

## 主要产出

- **七型连续谱**（I游牧→II反国家→III国家→IV法西斯→V全球资本→VI第七型恐怖→？VII灰色地带）
- **三条接入线路**：①占有-剩余的结构对称性 ②克劳塞维茨偶然性作为安全威胁的非实际化剩余 ③非全与战争机器的结构平行
- **理论修正**：暴力不是战争机器的本质，而是国家占有的产物；灰色地带不是占有的放松而是"负占有"（策略性不占有）

**Why:** 在阶段札记中标记为最紧急的缺口——战争机器簇迄今完全没有进入剩余安全框架，而剩余安全最终要面对暴力、主权、例外状态这些最硬的问题时，这条桥接线不可绕过。

**How to apply:** 从此卡出发可以：
- 修正"剩余安全"的定义：剩余不是遗漏，而是安全化占有的结构性溢出
- 重新定位灰色地带操作：国家"负占有"（不占有战争来维持对暴力的潜在调用权）
- 将非全与战争机器不可占有性区分开——前者是结构性空位，后者是差异的创造性过剩
- 为战争机器的类型学接入安全分析提供可操作框架（哪一型战争机器在哪种条件下被安全化占用？哪一型不可被占用？）
