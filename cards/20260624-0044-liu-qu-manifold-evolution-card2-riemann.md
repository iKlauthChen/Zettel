# 黎曼的 n 维流形：从哲学启发的描述性定义到空间统一

**来源**：刘宇辉 & 曲安京 (2015). 流形概念的历史演变. *自然辩证法研究*, 31(5), 78-82.

## 背景：非欧几何的危机

- 罗巴切夫斯基和波约已公开发表非欧几何，高斯内心承认但未公开主张
- 社会尚未完全接受非欧几何
- 黎曼演讲的目的之一：**澄清空间是什么**，统一已出现的各种几何
- 主张几何学**全局观**：作为"任何种类的空间里任意维度的流形研究"

## n 维流形的构建

1854年哥廷根演讲《关于几何基础的假设》分三部分：
1. n 维流形概念
2. 流形上的度量关系
3. 对空间的应用

### 构建方法（受赫巴特哲学影响）

- 称 n 维流形为 **n 元延伸量** (n-fach ausgedehnte Größe)
- 分为**连续流形**与**离散流形**
- 连续流形理论分两个层次：
  - 与位置相关的**区域关系**（现代术语：拓扑理论）
  - 与位置无关的**大小关系**（现代术语：度量理论）
- 构造类似**归纳法**：n+1 维流形通过 n 维流形同一维流形递归构造；低维可通过高维固定某些数量简缩而成
- 每个 n 维流形有 n 个自由度，每点位置可由 n 个数值（局部坐标）表示

### 受赫巴特启发

> "从一个维度前进到另一个维度所依据的方法，很明显是一个始终可以继续发展的方法，然而现在还没有人会想到按空间的第三个维度去假设空间的第四个维度。"

黎曼突破了三维限制，按递归方法构造 n 维流形。

## 量块（Quanta）与邻域概念的萌芽

- 流形中的比较——离散情形由数数给出，连续情形由测量给出
- 测量要求参与比较的量能迭加，需要"量块"作为标准
- **量块体现了拓扑学中的邻域概念特征**；"参与比较的量能够迭加"要求两个量块重叠部分有统一测量标准——即保证任意两个局部坐标系的相容性
- 这后来由希尔伯特发展为 n 维流形局部与 n 维欧氏空间的**同胚**

## 黎曼度量

- 在流形每一点赋予一个正定二次型（推广高斯第一基本形式）
- 借助高斯曲率给出**黎曼曲率**概念
- 陈述了一系列曲率与度量的关系

## 局限

- 引入坐标方法不清晰，缺乏用邻域或开集覆盖流形进而建立局部坐标系的思想——这是当时缺乏集合论和拓扑学工具导致的

## 意义

黎曼的流形概念与格拉斯曼的"扩张"概念和施莱夫利的"连续体"概念基本一致，但黎曼的演讲系统地统一了各种几何，使空间概念从欧氏三维拓展到 n 维，并区分了流形的拓扑层面与度量层面。

**Why**: 黎曼的关键创新在于：（1）将流形从三维空间中解放出来（n维）；（2）明确区分了流形的拓扑结构与度量结构——这是理解现代几何学"先有流形再赋予几何结构"路径的源头；（3）量块/邻域概念萌芽预示了后来的点集拓扑。

**How to apply**: 链接到德勒兹"virtual manifold"或Saldanha空间化中的流量/地图进路——黎曼把空间看作可以附着不同度量的拓扑基础，类似德勒兹"虚拟"作为多重性的基底。可用来说明"空间不是先验容器，而是由邻域关系构成的延展量"的思想史转折。
