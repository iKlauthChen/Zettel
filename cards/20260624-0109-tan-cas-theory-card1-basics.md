# 复杂适应系统（CAS）理论——核心框架

**来源**：谭跃进, 邓宏钟 (2001). 复杂适应系统理论及其应用研究. *系统工程*, 19(5), 1-6.

## 理论起源

CAS理论由**霍兰德（John Holland）**教授在多年研究复杂系统的基础上提出。Holland 也是遗传算法（Genetic Algorithm）的创立者。CAS的提出背景是圣达菲研究所（SFI）对经济作为演化复杂系统的跨学科研究。

## 核心思想

> "CAS的复杂性起源于其中的个体（Active Agent）的**适应性**——正是这些个体与环境以及与其他个体间的相互作用，不断改变着它们自身，同时也改变着环境。"

**最重要的特征：适应性（Adaptation）**
- 个体能够与环境以及其他个体进行交流
- 在交流过程中"学习"或"积累经验"，不断演化
- 根据学到的经验**改变自身的结构和行为方式**
- 各个底层个体通过相互间的交互、交流，可以在上一层次、整体层次上**突现**（emerge）出新的结构、现象和更复杂的行为

## 个体（Agent）的特性

所有个体都处于一个共同的大环境中，但各自根据**局部小环境**并行地、独立地进行适应性学习和演化。这种适应性和学习能力被认为是**智能**的一种表现形式，因此也常被称为"智能体"。

个体为了生存不断调整行为、修改规则——大量适应性个体在环境中的行为反过来影响和改变环境，环境则以"约束"形式对个体产生作用。个体和环境处于一种永不停止的相互作用、相互影响、相互进化过程之中。

## ECHO模型

Holland在遗传算法基础上建立了**"回声"（ECHO）模型**——用以模拟和研究一般的复杂适应系统的行为。圣达菲研究所基于ECHO模型建立了相应的建模工具——**SWARM仿真平台**。

## CAS的一般特征

| 特征 | 描述 |
|------|------|
| **层次性** | 系统具有明显层次，各层之间界线分明 |
| **层间独立性** | 直接关联少，个体主要与同层次个体交互 |
| **智能性/适应性/主动性** | 个体可自动调整状态/参数以适应环境、协同/合作/竞争（"适者生存"） |
| **并发性** | 个体并行地对环境刺激作出反应和演化 |
| **随机性** | 模型中可引进随机因素以增强表达能力 |

**Why**: 此文是CAS理论在中国的早期系统性导入（2001年），由国防科技大学系统工程研究所发表。核心洞见是CAS将复杂性起源定位于**个体的适应性**（而非外部冲击或随机扰动）。适应性个体的局部交互→整体涌现，这一机制是理解复杂系统演化的钥匙。区别于宋学锋（2003）的一般复杂性理论，CAS更强调**主体有目标、有学习能力**——这使其特别适合有人参与的系统（经济/军事/社会）。

**How to apply**: CAS理论可以直接应用于国际体系分析——国家作为"适应性主体"，在无中央政府的国际环境中根据局部信息学习、调整行为，通过交互涌现出国际体系层面的秩序（如均势、制度）。SWARM仿真平台可应用于冲突升级模拟、军备竞赛动力学建模等。
