# 发起动力——为何国家愿意为盟友"下场"？

**来源**：刘霖炜, 葛汉文 (2024). 延伸—即时威慑理论：历史案例与当下意义. *世界经济与政治论坛*, (1), 43-64.

## 核心悖论

延伸—即时威慑的发起方或防御方自身领土没有直接面对战争威胁——**缺少最直接的理由卷入一场本与其无关的国际危机**。因此，防御方需要有足够动力亲自下场保护盟友，同潜在进攻方进行一场意志与实力的较量。

## 三大动力因素

### 1. 国家的威望

**罗伯特·吉尔平**：国家的威望是一种**观念上的权力**——其他国家对一个国家行使其权力之潜力、能力和意愿的看法。

机制：
- 强国通过提供公共产品等方式实现对弱国的领导 → 投射权势 → 形成思想意识上的霸权
- 冷战期间美苏都需要在盟友面临战争风险时愿意使用武力支持盟友
- 否则安全承诺的可信性将打折扣 → 国家形象和威望受损

**典型案例**：朝鲜战争和越南战争——美国之所以出手保护韩国和越南共和国，"维持威望"是政策制定和实施极其重要的考虑因素。

### 2. 与被保护方的政治经济联系

**延伸威慑的内在矛盾**：大国为某个弱国对抗另一大国可能导致**失大于得**。

因此防御方需要从被保护国获取足够多的好处（政治上追随或经济上获益）才有动力伸出援手：

- 美国：欧洲复兴计划 + 北约 → 西欧与美国的绑定
- 苏联：经济互助委员会 → 社会主义阵营产业分工
- 一旦盟友被击败 → 失去政治追随者和经济伙伴

**最值得关注要素：军贸合作**
- 被保护国在防务上越是依赖防御方的军事援助 → 越难以独自应对进攻方威胁 → 防御方履行安全义务的必要性越强

### 3. 一般威慑时期的军事态势

国家在决定介入某一危机之前大多会评估其预期效果：
- 由于成本和风险巨大，履行延伸威慑承诺不应付出过高代价
- 如果一般威慑期间被保护国处于绝对劣势（即使防御方支援也难以应对或代价过高）→ 防御方可能弃之不顾
- 如果被保护国具备一定防御能力 → 可以顺势转变为威慑力量 → 减轻防御方成本 → 注入信心

## 三要素的交互

威望（主观观念）、政治经济联系（物质利益）、军事态势（可行性与成本考量）三者共同决定了防御方的介入意愿。三个要素同时走低时，防御方最可能弃盟友不顾；同时走高时，防御方最可能全力介入。

**Why**: 延伸—即时威慑的"发起动力"问题实际上揭示了联盟政治的一个核心悖论：联盟之所以需要，（对弱国而言）是为了获取保护；联盟之所以可能，（对强国而言）是因为保护者的利益往往多于被保护者自身的安全。三动力框架为评估"保护承诺可信度"提供了可操作的分析维度——不只是看盟友本身的价值，还要看盟主在全球战略中的威望需求和军事态势。

**How to apply**: 用于评估美国在台海冲突中"保护台湾"的承诺可信度——(1)威望层面：放弃台湾对美国亚太领导地位的冲击有多大；(2)政治经济联系：台湾在全球半导体供应链中的核心地位；(3)军事态势：第一岛链美军部署与解放军的即时力量对比。
