# 中国模式——有所保留的全球化：国家主导+市场机制+战略自主

**来源**：Nathan, A. J., & Scobell, A. (2013). Globalization as a Security Strategy: Power and Vulnerability in the "China Model". *Political Science Quarterly*, 128(3), 427-453.

## 核心特征

中国并未因加入WTO而转向完全西方式的经济。相反：

> "中国决策者创造了一种独特的国家引导但市场化的体制，维持了自我控制的关键要素。"

"中国模式"从全球贸易和投资中汲取力量，但没有损害：

- **国内市场**在经济增长中的主要角色
- **私有和外资部门**的繁荣但非主导地位
- **国家控制经济制高点**的能力

## 国家主导的具体表现

| 领域 | 国家控制方式 |
|------|------------|
| 土地 | 城乡土地仍归国家所有 |
| 能源/水/银行/铁路 | 政府直接管理 |
| "私有化"国有企业 | 党指派高层管理者+党委+政府指导银行信贷 |
| 战略行业 | 约千家最大国企整合为"国家队"（能源/电信/重工/国防/矿业/媒体/银行/交通） |
| 价格 | 间接政策杠杆决定土地、劳动力、住房、能源和信贷价格 |
| 汇率 | 浮动范围受限，政府通过买卖外汇设限 |
| 资本账户 | 人民币不可自由兑换——强大屏障防止国际投机 |

## 外贸易依赖的再审视

**表面**：中国外贸占GDP比例（51.9%，2008年），对于一个大国经济体来说是高的。

**实质**：
- 中国外贸比例排名仅第19位（低于印尼54.5%，略高于法国51.8%）
- "两头在外"战略（liangtou zaiwai）：原材料来源和市场都在国外
- 全球供应链中，中国获得的主要是**劳动组装成本**而非全部产品价值
- GDP中净出口的贡献率从未超过几个百分点

> "中国增长并非'出口驱动'——其国内市场的规模（加上政府刺激）足以避免2008年全球衰退时增长率的相应下滑。"

## "走出去"战略

- 利用其他经济体的对等开放，推动中国企业参与全球竞争
- 以国家银行优惠信贷协助企业成功
- 通过基础设施（道路、管道、港口、铁路）将运输网络延伸到越南/缅甸/尼泊尔/斯里兰卡/巴基斯坦/土库曼斯坦/乌兹别克斯坦/哈萨克斯坦/蒙古
- 这些项目不仅便利能源进口和内陆跨境贸易，还**创造了有潜在军事用途的后勤设施**

## 从援助受援国到援助提供国

- 1982年：建立对外援助司
- 1990年代：三家国际责任银行（国开行/农发行/进出口银行）
- 对外援助从1996年的5亿美元跃升至2007年的30亿美元以上

**Why**: "中国模式"概念揭示了全球化不是单向的"西化"——中国通过选择性融入和保持关键领域的国家控制，实现了"嵌入但不受制"的独特状态。这对"全球化=自由主义化"的线性叙事构成了挑战。国家在保持经济制高点控制的同时利用市场效率，这种混合体制在对冲全球化安全风险中起到了核心作用。

**How to apply**: 用于分析中国在面对西方"脱钩"压力时的制度韧性和战略选择弹性——中国模式既不是纯粹的国家资本主义也不是自由的市场经济，其混合性使中国比"纯市场经济"（如美国）或"纯计划指令经济"（如朝鲜）更有能力在全球化与安全自主之间导航。
