# 两种"战争机器"的严格区分与桥接——从Reid/德勒兹的游牧本体论到剩余安全框架

## 一、核心命题

**德勒兹-瓜塔里(DG)的"战争机器"不是安全研究中的"战争机器"。** 这一混淆是当前跨学科对话中最大、最持久的障碍。安全研究在直觉上将"战争机器"理解为军事机构、国防工业或暴力工具的复合体——即国家权力的武装延伸。DG 的"战争机器"恰恰相反：它不是国家拥有的东西，而是国家试图占有却永远无法完全捕获的外部性力量。

**"剩余安全"概念的接入点就在这个占有失败后留下的间隙中。** 如果战争机器是不可完全占有的，那么安全化——作为国家占有"威胁"的合法暴力的操作——也必然是永远不完全的。安全化操作留下的剩余，正是战争机器不可占有性的一个特异化表现。

## 二、严格区分：两种战争机器的平行对照

### DG 的战争机器（通过 Reid 重构）

| 维度 | 规定 |
|------|------|
| **本体论地位** | 先于国家、外在于主权——不是国家制造的，是国家试图占有的（杜梅齐尔：战神因陀罗违犯主权之法） |
| **本质** | 一种游牧的存有论方式（a nomadic way of being），不是暴力工具 |
| **核心机制** | 逃逸线（line of flight）——创造性的去层、新连接的打开 |
| **与战争的关系** | 其"战争"内涵不是军事暴力，而是对既有秩序的拓扑学挑战——"战争"在此是一个哲学概念，指代差异的创造性释放 |
| **绝对战争** | 一个永远不被实际化的理念——"非实际化的剩余"（real but nonactual），与拿破仑的"向空白的冲刺"同构（Reid切卡1611） |
| **与法西斯/恐怖的关系** | 逃逸线 ≠ 毁灭线——肯定性战争（主动创造）≠ 虚无主义战争（怨恨毁灭）（Reid切卡1613-1614） |
| **最新发展** | 第七型（全球恐怖主义）——游牧形式+教条欲望的双重特征（切卡0497） |

### 传统安全研究的"战争机器"

| 维度 | 规定 |
|------|------|
| **本体论地位** | 国家主权的工具——合法暴力的制度化载体 |
| **本质** | 军事-工业-情报复合体——从国防部到军火商到战略规划机构的制度性装配 |
| **核心机制** | 威慑、强制、打击——以暴力或暴力威胁实现政治目标 |
| **与战争的关系** | 战争是实现国家利益的手段（克劳塞维茨的标准解读："战争是政治以其他方式的延续"） |
| **预警与决策** | 基于克洛诺斯时间——预警期、响应窗口、威慑信号的可计算性（Dibb & Brabin-Smith切卡：预警时间坍缩是这一范式的根本危机） |
| **灰色地带** | 传统战争机器的失效场景——当"三灰"条件（目标/形态/手段的模糊化）使战争门槛消融，国家战争机器的旧工具箱失效（胡波切卡） |
| **核心困境** | 安全困境：追求自身安全产生体系不安全——战争机器的扩展产生新威胁 |

### 对照表

| 比较项 | DG战争机器 | 安全研究的战争机器 |
|--------|-----------|------------------|
| 拥有者 | 无人拥有——或说游牧者"是"战争机器 | 国家——是国家的制度和工具 |
| 核心功能 | 创造性的去层、逃逸、变异 | 暴力、威慑、防御、强制 |
| 对国家的关系 | 外在于国家——国家试图占有 | 内在于国家——是国家行为体的属性 |
| 暴力的地位 | 衍生性的——不是本体论定义 | 核心定义性的——没有暴力不成战争机器 |
| 时间性 | 艾昂（非实际化的虚拟）、偶然性的爆裂 | 克洛诺斯（可计量的预警-响应周期）、凯若斯（及时打击窗口） |
| 评估标准 | 创造力的释放 vs 毁灭线的坠落 | 威慑可信度、效费比、胜算 |

## 三、关键枢纽：占有（appropriation）作为结构性连接

两种战争机器不是毫无关系的——它们通过**占有**（appropriation）这一操作连接在一起。

### 占有的逻辑结构

```
[DG战争机器: 游牧的创造性力量]
        ↓
[国家试图占有战争机器]
   → 成功一面：制度化、规范化（军队、国防政策、战略规划）
   → 失败一面：剩余——不可占有的游牧能量
        ↓
[两种失败模式]
   A) 法西斯式毁灭线：占有失败导致战争机器反向毁灭国家（Reid切卡1613）
   B) 第七型恐怖：被国家驱逐的游牧能量以宗教/意识形态暴力形式回归（切卡0497）
        ↓
[剩余安全框架的接入点]
   C) 安全化的不完全性：安全化操作试图占有"威胁话语"的游牧能量，
      但总是留下不可化约的时间差异——这就是剩余安全的来源
```

Reid 切卡 1612（德勒兹 vs 福柯）提供了一个更精确的表述：
- **福柯**：越轨从不外在于权力——欲望的释放和战争的暴力都是权力运作的内在组成部分
- **德勒兹**：占有是不完全的——总有一个不可化约的剩余保留在国家的编码之外

**剩余安全的操作定义从这一区分获得本体论锚定**：如果德勒兹是对的——占有永远有剩余——那么安全化（国家占有"威胁"合法定义权的操作）也永远有剩余。剩余不是"遗漏了什么"，而是占有所产生的结构性溢出。

### 杜梅齐尔的神话学证据（Reid切卡1610）

印欧三元结构中战神（因陀罗）从属于主权之神（密特拉/伐楼拿），但战神**违犯主权之法而不受惩罚**。这一悖论结构精确对应了占有操作的本体论形式：
- 战争既在主权的体系之内（它是三元结构的一环）→ 国家可以调动军队
- 战争又外在于主权的规则（战神不受主权之法约束）→ 国家无法完全驯化暴力

Colebrook（切卡0499）将这一结构推向极致：战争机器先于国家——主权不是暴力的唯一来源，而只是试图占有原初的力的关系。在这一框架下，"反恐战争"不是主权例外，而是国家对战争机器的双重压制：既压制原初的创造性力量（通过安全化），又将其捕获为自身的暴力工具。

## 四、类型学连续谱：从 DG 战争机器到安全研究战争机器到剩余安全

为了让两种战争机器的关系从抽象走向可操作，我构建一个六/七型**连续谱**（基于 D&G 六型 + Purakayastha&Das 第七型 + 剩余安全衍生型）：

| 型 | 名称 | 核心特征 | 实例 | 与安全研究的交叉 |
|----|------|---------|------|----------------|
| **I** | **游牧战争机器** | 不以战争为目的；创造性的去层和逃逸 | 游牧部落、思想运动、艺术革命 | 安全研究完全忽视——因为不涉及暴力 |
| **II** | **作战战争机器** | 以对抗国家为目标——战争作为对国家的反抗 | 游击战、解放运动 | 被安全研究归入"非对称威胁"或"叛乱" |
| **III** | **国家战争机器** | 战争作为国家实现政治目标的工具——这是安全研究唯一承认的"战争机器" | 正规军、国防体系 | 安全研究的标准对象——威慑/防御/强制 |
| **IV** | **法西斯战争机器** | 国家被战争机器接管——总力战导致自毁 | 纳粹德国 | 安全研究的"极端案例"——国家生存权被悬置 |
| **V** | **冷战-全球资本战争机器** | 两极化+全球资本主义捕获战争逻辑 | 北约vs华约、反恐战争 | 传统安全分析仍然可以处理——但"叙事竞争"（胡波）开始溢出现有框架 |
| **VI** | **第七型恐怖战争机器** | 游牧形式+教条欲望的畸形混血——既反对国家又复制国家压抑 | ISIS、全球圣战网络 | 对安全研究形成根本挑战：无法用工具理性解释（Devji切卡0503） |
| **→** | **VII* ?** | **? 灰色地带战争机器** | **灰色地带操作——胡波的"三灰"标准** | **剩余安全的核心案例——尚未被理论化的战争机器新类型** |

### 关键观察：安全研究只认识 III-V 型

传统安全研究的"战争机器"概念在类型学上只覆盖 **III（国家）、IV（法西斯）、V（冷战/全球资本）** 三型。它将:
- **I 型**完全排除（因为不涉及暴力）
- **II 型**归入"叛乱"或"不对称威胁"——丢失了其作为"对抗国家捕获"的哲学本质
- **VI 型**误读为"非理性暴力"——丢失了其欲望经济的真正动力（Devji切卡0503: 圣战作为自足的行动）
- **灰色地带操作**——传统类型学无法定位：它既不是国家战争机器（III），也不是反国家的战争机器（II），也不是恐怖（VI）

**这就是灰色地带操作的悖论性地位**：它是国家在安全化名义下调用战争机器能量，却有意使其保持在"不战不和"的阈限状态。从 DG 占有的角度看，灰色地带不是国家占有战争的失败，而是占有的一种**策略性留白**——国家不完全占有战争机器（因为完全占有意味着升级为战争），但也不放弃占有（维持对创造性暴力的潜在调用权）。

## 五、桥接：战争机器概念如何接入剩余安全框架

### 接入线路一：占有-剩余的结构对称性

**安全化 ≈ 国家占有战争机器的局部表现。**

国家通过安全化操作占有"威胁"的合法定义权——将某一议题移入紧急政治领域，使其服从于主权暴力的逻辑。但正如国家占有战争机器永远不完全（Reid切卡1612），安全化操作也永远有剩余：
- 安全化在时间中留下的不可逆差异（新的法律框架、制度安排、主体位置）——这些构成剩余安全的时间维度（见"非安全化的时间条件"草稿）
- 安全化操作中被排除但无法消除的欲望能量（恐怖作为虚空的病理化——切卡0504）——这些构成剩余安全的本体论维度

**形式化表达**：剩余安全 ≈ 安全化占有的结构性溢出 = d(占有)/dt ≠ 0 条件下的 ∫(战争机器(t) - 被捕获部分(t)) dt

### 接入线路二：克劳塞维茨偶然性 = 安全威胁的非实际化剩余

Reid 切卡 1611 揭示了克劳塞维茨三一模型的第三个元素——**偶然性**（chance）——是德勒兹最为珍视的维度：
- 偶然性是"创造性精神可以在其中自由漫游"的领域
- 是国家编码无法触及的战争机器的剩余能量
- 在安全语境中，偶然性表现为：威胁的**不可预测性**、预警的**认知边界**（Dibb 切卡：政治领导人不愿相信预警不是情报失灵而是政治认知问题）

**安全中的偶然性**是安全研究的盲点：传统威慑理论试图通过克洛诺斯时间（可计算信号、可靠威慑）消除偶然性，但灰色地带操作恰恰是**对偶然性的战术化调用**——胡波切卡揭示的"切香肠战术"就是利用对方无法在时间上准确定位"威胁的时刻"。

**接入操作**：将克劳塞维茨的"偶然性"从战略理论术语翻译为剩余安全的时间剩余（艾昂的虚拟储备），从而将"偶然性"从战略规划的对立面转化为对安全操作不可占有性的正规定义。

### 接入线路三：非全与战争机器的结构平行

Zevnik 的"非全"（non-all）——存在在结构上不完整——与 DG 战争机器的不可占有性在结构上平行但不同源：
- **非全**是拉康式的：结构性的空位——符号秩序永远缺少一个元素
- **战争机器不可占有**是德勒兹式的：过剩——差异永远溢出编码

两种剩余不相同，但在安全语境中产生可观察的等价效果：
- 非全→安全化无法完全闭合系统（总有一个"不可安全化"的剩余）
- 战争机器不可占有→安全化无法完全捕获欲望（总有一个创造性逃逸的剩余）

**这解释了为什么第七章中"恐怖作为虚空的病理化"（切卡0504）和Colebrook的战争机器先于国家（切卡0499）看似对立却在反恐分析中得出相似的结论**：两者都指向同一个结构——安全操作面对的基本不是"外部威胁"，而是自身占有所产生的结构性溢出。

## 六、安全理论含义：如何重新定位"安全暴力"

1. **暴力不是战争机器本质，而是国家占有的产物**。DG 战争机器本身拒绝等同于暴力——其"战争"内涵是拓扑学挑战和社会-存在论的创新。将战争机器直接等同于暴力是安全研究中最大的类别混淆。

2. **安全化的暴力不是主权的例外，而是占有的失败**。Colebrook（切卡0499）对阿甘本的批判在此获得具体指向：反恐战争不是主权例外决定的"法外状态"，而是国家多次尝试占有战争机器后产生的毁灭性反馈回路——每一次安全化都在强化而非解决威胁。

3. **"去安全化"（desecuritization）需要一种新的战争机器类型学来支撑**。目前哥本哈根学派的去安全化概念过于简化（"回到正常政治"），但如果引入战争机器类型学，去安全化就变成了**将安全操作从 III/V 型（国家/全球资本逻辑）转译为 I/II 型（创造性的、非暴力的逃逸）的实践**。不是"减少安全"，而是"改变占有的模式"。

4. **灰色地带操作不是战争机器的放松而是反讽式强化**。胡波揭示的策略性模糊化——国家主动不占有战争（停留在战争门槛以下）——是一种"消极占有"（negative appropriation）：国家通过不占有战争来维持对战争暴力的潜在调用权。这与 Reid 切卡 1615 中"反战略思想传统"的克莱斯特/卡夫卡式的逃逸形成讽刺性对照：国家学会了战争机器的游牧策略，但为了控制而非解放。

## 七、未决张力与写作方向

| 张力 | 描述 | 需要进一步的工作 |
|------|------|----------------|
| **正占有 vs 负占有** | 传统国家积极占有战争机器（III型）vs 当代国家消极不占有（灰色地带）——两者都是占有，但逻辑不同 | 需要写一张"占有的两张面孔：正占有与负占有"的专门卡 |
| **逃逸线 vs 毁灭线的实践不可区分性** | Reid切卡1614揭示的伦理困境：逃逸线和毁灭线在事发时无法通过先验标准区分——只能在事后通过效果判定 | 需要接入"剩余安全的门槛"概念——政治创造性与毁灭性的区分依赖于对剩余快感的处理（切卡0504的三维框架） |
| **Devji的非工具性暴力如何接入** | 如果圣战是"自足的行动"（切卡0503），那么传统安全分析中的因果逻辑（威胁→反制→回撤）对恐怖完全无效 | 接入"剩余快感的非工具性"（切卡0482）——剩余不是可交换的货币，而是驱力的自持运动 |
| **战争机器的七型 vs 实际多样性** | 切卡0497警告第七型内部可能有不同子型——第七型可能掩盖VI种不同张力 | 类型学不应成为封闭的格子 |
