# 有限性海陆双向战略的哲学基础：对一种空间本体论的重构

## 摘要

传统地缘政治理论将海陆复合型战略视为一种注定失败的选择，理由是资源在海洋与陆地之间的分散必然导致战略效能低下。然而，这一判断的深层问题不在于资源分配的计算，而在于它对"海"与"陆"的本体论差异缺乏自觉——它假定了海洋和陆地是同质的战略空间，资源在其中按照同一逻辑运作。本文借助一个已成熟的"三层本体论"框架——虚空（Void）、内在性平面（Plane of Immanence）、实在界（the Real）——对海陆的空间本体论差异做出比现有的"平滑/纹理空间"二分法更根本的奠基。核心论点是：海陆的战略差异不是地理类型学的差异，而是**吸引子结构的差异**——陆地是强吸引子空间（秩序易于稳定形成和维持），海洋是弱吸引子空间（秩序难以稳定、永恒的平衡不存在）。由此，"有限性"海陆双向战略不是一种策略上的自我节制，而是一种本体论条件的自觉：在弱吸引子空间中，任何秩序的建立都必然是局部的、临时的和需要反复维护的；任何试图永久填满海洋"虚空"的战略努力，都会遭遇实在界的回归——即被排除的剩余以不可控的方式反击。在此基础上，本文重新分析了法国、德国和苏联三个历史案例，指出其战略成败的关键不在于"资源被分散"，而在于它们的战略装置与目标空间类型的吸引子结构之间是否发生了本体论错配。最后，本文提炼出"有限性"概念的四层哲学含义，为海陆复合型国家的战略设计提供更具原创性的理论基础。

关键词：有限性海陆双向战略；三层本体论；虚空；内在性平面；实在界；吸引子结构；海陆复合型国家


## 一、引言：资源分散论的破产

传统观点认为，将目光投向海洋和陆地的国家大多以失败收场。法国、德国以及苏联被反复援引为海陆复合型战略失败的典型案例。经典解释的链条是：海陆复合型国家面临来自海洋和陆地两个方向的安全威胁 → 必须将资源分散投入到陆军和海军 → 资源分散导致在每个方向上都弱于专注单一方向的对手 → 战略失败。

这个链条看似合理，但它在两个根本性假设上站不住脚。

**第一，它假设海洋和陆地是同质的**——资源在海洋上和陆地上的效用遵循同一逻辑。由此推论出"如果陆军需要100个单位，海军也需要100个单位，那就必须投入200个单位，而专注陆地的对手只用100个单位就能对抗你的陆军"——这只有在陆军和海军的战略效用可以被同一尺度衡量时才成立。但如果陆军和海军运作于不同的本体论空间，它们的效用不可以被简单相加或比较。

**第二，它假设战略失败的原因是"不够专注"**——因此隐含解决方案是"选一个方向，要么陆地要么海洋"。但历史证据不支持这个推论：法国的失败不是因为它同时有陆军和海军，而是因为它无法协调这两种力量；德国的失败不是因为海军分散了陆军资源，而是因为它建造了一支在自己无法有效运作的空间中的舰队；苏联的崩溃不是因为资源分散，而是因为其装置丧失了弹性。

这些历史异象提示我们：海陆复合型战略真正的问题不在于资源的"量"，而在于空间性质的"质"。这正是本文需要从本体论层面重新奠基的起点。

## 二、三层本体论框架及其空间含义

本文的理论基础是一个已经相对成熟的三层本体论框架（[[20260621-0505]]），由虚空（Void）、内在性平面（Plane of Immanence）和实在界（the Real）三个概念构成。这一框架脱胎于对Zevnik（2016）的"肯定性虚空"、Bonta和Protevi（2004）的"复杂性吸引子"以及Žižek（2012）的"剩余/少于无"三个理论工程的整合阅读。以下是框架概要及其在空间分析中的映射：

### 2.1 虚空（Void）：作为肯定性"无"的空间基底

虚空不是"什么都没有"的虚无，而是一种"肯定性的无"——它不预设任何本质、任何必然的秩序原则。在虚空中，没有任何东西"内在"地引导理解或实践走向特定的终点。世界并非缺乏本质，而是一一更根本地说——它**不需要**本质来运作。

这对海陆战略的核心含义在于：海洋和陆地之间**不存在本体论上的天然差异**。传统地缘政治理论（从马汉到麦金德）假设"海"和"陆"是两种具有固定战略属性的空间——海洋=机动性/开放性/通道，陆地=稳定性/主权/定居——但这种差异不是空间的本质属性，而是特定秩序逻辑（=国装置、资本主义、国际法）的产物。海洋并不是"本来就更难控制"——它只是**尚未被以同样的方式和密度纹理化**。

由此，"海"与"陆"的战略差异不是来自地理的物理性质（水vs土地），而是来自历史-制度的**纹理化程度差异**。陆地已经被数千年的人类定居、主权划分、财产登记充分编码；海洋的纹理化则始终是局部的、断续的、需要反复维护的。虚空概念的引入使战略思维摆脱"海陆性质固定"的本体论承诺，转而关注**秩序如何被建立**以及**不同空间对秩序建立的抵抗程度为何不同**。

### 2.2 内在性平面（Plane of Immanence）：吸引子结构解释强弱空间

如果虚空是"条件"，内在性平面就是"机制"。内在性平面不是空的——它是虚拟吸引子（virtual attractors）的拓扑空间，是系统自组织行为的倾向性结构（Bonta & Protevi, 2004）。吸引子不决定系统做什么，但它们建立了系统**可能做什么**的范围——即吸引子盆地（basins of attraction）。

将吸引子理论应用于空间分析：**不同空间具有不同强度的吸引子结构**。

- **陆地空间**是强吸引子空间。人类社会在陆地上建立的秩序（国界线、财产权、法律管辖）具有高度的稳定性和自我维持能力。一旦一条边界被划定，它的存在本身就会产生持续的效应——人们在边界的约束下建设道路、征税、部署警察，这些活动反过来强化边界的现实。陆地上的吸引子是"深的"——系统状态一旦进入某个吸引子盆地，就需要大得多的能量才能离开。

- **海洋空间**是弱吸引子空间。在海洋上建立的秩序（航线、巡航区域、专属经济区）缺乏陆地秩序那样的自我维持机制。一条海上边界不会因为被"划出"就产生约束——需要海军舰船的定期巡逻、国际法的反复援引、盟国的一致承认。一旦巡逻停止、话语衰退、联盟松动，海洋秩序就会迅速衰减到无序状态。海洋上的吸引子是"浅的"——系统状态容易被扰动推出盆地。

这个差异是对D&G"平滑/纹理"二分法的深化和精确化。平滑/纹理描述的是空间的状态（被编码vs抵抗编码）；吸引子理论解释的是**为什么**某些空间更容易被编码（强吸引子）而另一些空间持续抵抗编码（弱吸引子）。原因不在于物理性质本身，而在于系统的**反馈结构**：陆地上的秩序行为（划界、定居、控制）产生的是正反馈——秩序强化秩序；海洋上的秩序行为产生的是弱反馈——秩序需要持续输入能量才能维持。

这一本体论差异直接导向了"有限性"战略的哲学根基：**在弱吸引子空间中，任何秩序的建立不可能一劳永逸。** 这不是因为国家投入的资源不够多，而是因为海洋空间的结构决定了——用德勒兹的术语——再辖域化（reterritorialization）的效果不是永久的。航母离开后，它建立的界域就开始衰减。

### 2.3 实在界（the Real）：战略的剩余效应

虚空是本体论条件（允许不同秩序），内在性平面是机制论条件（约束秩序形成的可能性范围），而实在界是**每一种秩序运作时必然产生的剩余效应**——不能被那种秩序吸收、但持续影响其运作的残余。

实在界概念对战略分析的贡献是双重的。

**第一，实在界揭示了战略计算的不可完成性。** 任何战略都是在一个符号化的空间（认知地图、威胁评估、资源规划）中展开的，但这个空间永远不可能完整覆盖它所指向的实在状态。总有一部分"落在地图之外"——这就是战略的实在界。历史上的海陆复合型国家的失败，通常不是因为资源分配的比例错误（可以在计算中修正的错误），而是因为战略计算的框架本身排除了某些关键变量——这些被排除的变量以"实在界的回归"的方式最终颠覆了战略。

**第二，实在界在战略层面表现为欲望生产（desiring-production）的剩余。** 德勒兹和加塔利（1972）在《反俄狄浦斯》中将以一个关键洞察颠覆了经典马克思主义的因果链：不是生产方式使生产得以实施，而是欲望生产本身使某种特定方式成为它所是的那类方式。将此逻辑应用于战略分析：不是"战略必要性"（客观安全结构）决定了国家为什么建造海军，而是"国家欲望"——被承认为世界强国的情动、民族荣耀的投注、海军将领对预算和地位的制度性渴求——使"战略必要性"这种论述得以被需要和被创造。战略理性（"我们需要海军来保护日益增长的海外贸易"）是欲望流动凝固后的事后合法性论述。**被理性计算排除的欲望剩余，就是海陆战略中最危险的实在界。** 威廉二世的海军建设、拿破仑的征服、苏联的军事僵化——都不符合任何"理性"战略框架的最佳分析，但它们恰恰是战略中最具决定性的力量。

### 2.4 三层框架的空间映射

将三层框架映射到海陆战略分析上，可以构建以下分析模型：

```
┌──────────────────────────────────────────────────────────┐
│  层次              海 (sea)              陆 (land)        │
├──────────────────────────────────────────────────────────┤
│  虚空 (Void)       弱纹理化的空间          强纹理化的空间       │
│                   秩序需持续再建立          秩序已充分编码       │
│                                                         │
│  内在性平面        弱吸引子结构            强吸引子结构          │
│   (吸引子)         浅盆地，易扰动           深盆地，自维持        │
│                                                         │
│  存在/具象化       海军装置               陆军装置             │
│   (装置/制度)      航母/航线/港口          边界/基地/驻军      │
│                                                         │
│  实在界 (Real)     欲望的剩余              秩序的剩余           │
│                    (对无限海洋的权力幻想)   (官僚化惯性)         │
└──────────────────────────────────────────────────────────┘
```

这个模型的核心优势在于：它不预设海和陆是不同性质的"实体"，而是将差异定位在纹理化的完成度和吸引子的强度上——这是程度之差而非性质之差。由此，"有限性海陆双向战略"可以重新定义：**不是在海和陆之间做"少投入"的自我约束，而是认识到两种空间的吸引子结构差异后，为不同的空间配置不同的装置，并且意识到每种装置运作时都会产生未被完全计算的实在界剩余。**

## 三、历史案例的三层本体论重读

在传统的海陆复合型战略叙事中，法国、德国和苏联被反复引证为"分散资源于海陆两方向而最终失败"的案例。但如果我们以三层本体论的框架重新审视这三个国家的战略经验，会得出不同于传统诊断的结论。

### 3.1 法国：虚空的间距与实在界的回归

法国案例揭示的是两个层面的本体论错配。

**虚空层面的错配：解域化范围超出再辖域化能力**

法国的战略装置将国家力量从固定领土解域（deterritorialize）到全球海洋——地中海、大西洋、印度洋乃至加勒比海。这种解域化本身在弱吸引子空间中极为有效：法国海军可以在远离本土的水域建立临时界域，开展贸易、部署力量、传递信号。但问题在于：全球海洋的虚空——作为弱吸引子空间、作为肯定性的"无秩序"——不提供任何"自然"的界域支撑点。法国在北美、印度、加勒比海建立的每一个贸易站、海军基地、殖民据点，都需要持续的流动资源投入才能维持其"秩序"状态。当法国的战略注意力转移到欧洲大陆时，这些海洋界域就开始解体和被他者捕获。

这是虚空层的核心教训：**弱吸引子空间不保留秩序投资**。你在2023年建立的海军基地和巡航模式，如果2024年不去维护，2025年就不再属于你。法国在七年战争中的失败不是因为它"投入不够多"，而是因为它的装置没有理解海洋虚空的这一根本特性——在该空间中的投资不能像在陆地上那样"一次投资、永久受益"，它必须被持续更新。

**实在界层面的错配：革命战争机器被国家装置捕获**

拿破仑战争并非起因于周边安全环境的恶化，而是起因于法国大革命这一内部政治事件的对外溢出。用三层框架的语言：法国大革命释放出的创造性解域力量——公民军队、民族主义动员、社会平等理想——是**虚空的肯定性力量**（"一切可以被不同地组织"）在政治层面的爆发。这台革命战争机器最初是创造性的：打破旧制度的纹理化等级秩序，开辟新的社会可能性空间。

但拿破仑的帝国装置捕获了这台战争机器，将它从虚空的创造性力量变成了**实在界的回归**——征服的冲动不再服务于任何可识别的战略目标，而成为欲望驱动的无限扩张，最终退化为"毁灭之线"（德勒兹语）。当革命的逃逸线变成了帝国的毁灭之线，法国的战略装置丧失了在两种空间逻辑之间切换的能力——它只能在陆地上无限扩张，而英国从海洋的深处持续侵蚀着它的每一次再辖域化努力。

特拉法尔加海战的失败在这个框架下不是战术上的，而是本体论上的：法国的陆地装置和海上装置从未构成一个真正的"装置"（assemblage）——即各种异质性要素互为前提、共同运作的整体。它们是两个各自独立运作的系统，各自消耗资源，各自遵循不同的空间逻辑。海陆双向战略的真正失败不在于资源"被分散了"，而在于两个方向上的力量从未构成同一个装置的不同面向。

### 3.2 德国：欲望生产与实在界对理性计算的暴政

德国案例是最能体现"实在界"破坏力的案例。

**虚空层面的错配：德国海军的目标空间——北海——不是"开放的"而是已纹理的**

德国将大规模海军力量投射到北海时，它犯了虚空层面的判断错误：它把北海当作"开放的海洋"——一个有无限可能性、等待被秩序的弱吸引子空间。但事实上，北海已经被英国皇家海军经过数百年的持续投资转化为高密度纹理化的强吸引子空间。北海的每一处锚地、每一条航道、每一个海峡都已经被英国的基地网络、航线控制、盟友体系和作战经验所编码。德国公海舰队进入这个空间时，面对的不是"虚空"（开放的可能性），而是另一个装置的界域——一个已经被充分编码和占用的空间。

这就是为什么德国投入了如此庞大的资源建造公海舰队，却几乎从未改变英国在北海的战略主导地位：**在北海这个已经被纹理化的空间中，德国的装置不是在虚空中建立秩序，而是在另一个装置的界域内制造扰动。** 制造扰动不等于建立秩序——一只蚊子在房间里飞不能证明它拥有房间的所有权。日德兰海战的"战术胜利、战略失败"在此获得了本体论层面的含义：战术胜利发生在舰对舰的遭遇（弱吸引子空间中的点对点对抗），而战略失败是因为德国从未能改变这个空间的吸引子结构本身。

**实在界层面的错配：欲望替代了战略计算**

威廉二世德国的海军扩张在传统安全分析中难以得到令人满意的解释。德国并不是一个海上贸易帝国，它的主要安全威胁来自陆地（法国和俄国），它的地理条件——北海出海口受英国控制——意味着大规模海军的战略效用极为有限。但德国仍然倾举国之力建造了一支足以挑战英国海上优势的公海舰队。

三层框架对此的解释是：**威廉二世的"海军必要性"论述不是战略计算的产物，而是欲望生产（desiring-production）的产物。** 德勒兹和加塔利的"欲望生产"概念揭示了一个被战略理性话语掩盖的事实：不是战略必要性驱动国家行为，而是被承认为世界强国的情动（affect）、民族荣耀的身份投注、海军将领对预算和地位的制度性渴求——这些欲望流动凝固后，才"生产"出需要海军这一战略必要性作为其合法性论述。威廉二世要的不是安全的海洋，而是"**作为强国的海洋**"。

这个欲望剩余——即被理性战略计算排除但实际驱动战略决策的实在界——是海陆双向战略中最难以管理的变量。不同于可用兵棋推演量化的资源分配问题，欲望生产不响应"投入产出比"的逻辑。当德国的海军扩张在战略计算上是非理性但在欲望逻辑上是完全理性的，任何"更合理的资源分配方案"都不可能有效——因为它解决的是策略层面的问题，而问题发生在实在界层面。

希特勒时期的德国案例强化了这一诊断。第三帝国同时进行了两个方向上的装置投射——陆军向东欧推进，海军在大西洋开展狼群战术。这两个装置在各自的空间逻辑中都是有效的——德国陆军在陆地上无可匹敌（在一个已经被充分纹理化的欧洲大陆上，它的装置与目标空间匹配），德国潜艇在大西洋几乎成功掐断了英国的生命线（在弱吸引子空间中，狼群战术是高度适应性的装置配置）。但这两个装置从未形成同一个战略装置的不同面向——陆上装置和海上装置各自独立地消耗资源、各自独立地产生效果、各自独立地陷入僵局，缺乏任何协同或切换的机制。

当苏联战场和大西洋战场同时陷入僵局时，德国的战略装置在两个空间中同时丧失了弹性。这种弹性的丧失在装置理论中意味着：装置的"面向无器官身体"的那一面（拆卸、流动、重新配置的能力）被"面向层"的那一面（固定化、制度化、欲望惯性）完全压制。一个没有逃逸线的战略装置注定崩溃。

### 3.3 苏联：虚空封闭与再辖域化过度

苏联的案例揭示了海陆双向战略的另一种结构性风险：虚空被意识形态完全封闭后装置丧失弹性。

**虚空层面的封闭：意识形态填海**

苏联采取海陆双向战略的初始动机与法国和德国不同。它不是在"开放的可能性空间"中做选择，而是试图将一种已经确定的意识形态秩序强制投射到全球空间。苏联的虚空——即"一切秩序都是偶然的"这一认识——被马克思列宁主义的历史必然性话语所封闭：空间不是有待被不同方式组织的可能性场域，而是已经被"科学规律"决定了发展终点的领土。

由此产生的问题是：苏联的海军装置虽然在技术能力上达到了相当高的水平（戈尔什科夫时代建立了覆盖全球大洋的远洋海军），但它始终受困于意识形态对虚空的封闭。苏联不能像美国那样灵活地在不同空间逻辑之间切换——不能像美国既能支持民主国家又能与独裁政权合作；不能像美国既能进行主权军事干预又能通过国际法机制运作——因为苏联的"科学理论"已经规定了空间的本质，排除了其他可能性。

这就是虚空封闭的战略后果：**当虚空的肯定性"无"（一切秩序偶然）被一种意识形态的"有"（历史必然）所填充时，战略装置就失去了在两种空间逻辑之间切换所必需的自反性和灵活性。**

**内在性平面层面的僵化：强吸引子的病理化**

苏联的内向性平面——其系统的行为倾向空间——在几十年间经历了从"多样化的可能性"到"单一路径依赖"的转变。斯大林时代的苏联经历了近乎无限的解域化力量（工业化的野蛮推进、社会结构的彻底重组、地缘扩展），但赫鲁晓夫之后的苏联已经完成再辖域化，进入了稳定的吸引子盆地。

这个稳定的吸引子就是冷战的结构性对抗：军备竞赛、代理人战争、两个体系的意识形态对峙。苏联的所有战略决策都被这个巨型的吸引子所捕获——它不需要在每个具体问题上独立判断，因为"与西方对抗"这条路径的吸引子盆地已经事先规定了所有可能的行为选项。

由此导致了"强吸引子的病理化"：系统在某个吸引子盆地里太深了，以至于任何扰动都不足以让它跳出来。当戈尔巴乔夫的改革试图打开一个新的吸引力——与西方和解、政治开放、经济重构——他发现的不是平滑的相位空间，而是已经被几十年冷战吸引子所深刻塑造的地形。改革试图把系统从一个深吸引子拉到另一个方向，但需要的能量远远超出可用的政治资源。

**实在界层面的回归：装置僵化与弹性丧失**

德勒兹和加塔利在讨论装置时做出了一个对苏联案例有极强解释力的区分：装置虽然面向层（在那里被组织和固定），但也面向无器官身体（在那里被拆卸和流通）。一个健康的装置必须能够同时在两个方向上运作：它既能建立界域，也能解除界域；既能结构化为有效的组织，也能拆卸和重新配置。

苏联的装置在战略层面达到了高度的确定性——信息确定、命令清晰、执行坚决。但这个确定性是以牺牲装置的弹性为代价的。当1956年的匈牙利和1968年的捷克斯洛伐克出现逃逸运动时，苏联装置的回应不是重新配置界域，而是暴力镇压——用纹理化的国家力量去封锁弱吸引子空间中涌现的逃逸线。

更有趣的是苏联海军装置的空间困境：虽然戈尔什科夫时代的苏联海军获得了远洋能力，但它始终被限制在几个被高度纹理化的出海口（波罗的海、黑海、日本海、巴伦支海）。苏联从未能像英国或美国那样在全球的弱吸引子空间中自由地建立和解除临时界域。它的装置被锁死在"面向层"的那一极——过度组织化、过度制度化、过度再辖域化。

当内部的改革压力和外部的技术竞争同时到达临界点时，这个没有逃逸线的装置没有弹性来吸收任何冲击——它只能整体性崩溃。苏联的崩溃不是"资源分散于海陆"的后果，而是虚空被封闭后、装置的自我更新能力被完全掏空的必然结果。

### 3.4 三个案例的比较总结

| 维度 | 法国 | 德国 | 苏联 |
|------|------|------|------|
| **虚空层错配** | 解域化范围超出再辖域化能力 | 目标空间已被对手纹理化却视为虚空 | 意识形态封闭了虚空本身 |
| **内在性平面错配** | 陆/海装置各自独立运作，未构成统一装置 | 欲望替代战略计算——北海是已纹理化空间却被当虚空对待 | 深吸引子捕获——冷战对峙路径锁定 |
| **实在界回归** | 革命战争机器被国家装置捕获为征服 | 欲望生产——对强国的渴望替代理性分析 | 装置弹性丧失——没有逃逸线的崩溃 |

## 四、"有限性"的四层含义

基于三层框架和三个历史案例的三维分析，"有限性海陆双向战略"之"有限性"获得了四层相互嵌套的含义。

### 4.1 虚空的有限性：弱吸引子空间中秩序的局部性和临时性

最根本的"有限"来自虚空的肯定性力量在弱吸引子空间中的运作方式。海洋作为弱吸引子空间，其核心本体论特征是：**在它上面建立的任何秩序都不具有自我维持的能力。** 这与陆地完全不同。

陆地上的秩序——边界、财产、主权——具有正反馈的结构。一旦被建立，它们在日常生活与实践中的持续使用就会不断再生它们。你不需要每年重新宣布"这个县属于这个省"——边界就在那里，人们按照它行动，它自动复制自己。海洋上的秩序没有这个机制。一条海上边界仅在被巡逻、被援引、被承认时"存在"。当这些行为停止，边界就在本体论意义上消失了——不是因为有人擦掉了线，而是因为没有人再画它。

这就是虚空对有限性的第一层要求：**在弱吸引子空间中，战略资产的有效性时刻衰减，任何秩序投射都是局部的、临时的、需要反复维护的。** "有限"不是因为你资源不够所以只能做这么多，而是因为**在吸引子结构中不存在"一劳永逸"的选项**。认识到这一点不是战略上的谦卑，而是本体论上的诚实——对空间本身性质的准确判断。

### 4.2 内在性平面的有限性：装置与目标空间的吸引子匹配

第二层"有限"来自内在性平面的吸引子结构对不同装置的约束。一个为强吸引子空间配置的装置（如陆军：固定、可见、通过控制领土来实现安全）和一个为弱吸引子空间配置的装置（如海军：机动、隐蔽、通过持续在场来实现影响）遵循不同的运作逻辑。它们在各自的空间中有效，但不能被简单地互相替代。

装置与目标空间之间的这种匹配约束，构成了一种结构性的有限性：**你不能用一个为强吸引子设计的装置去运作于弱吸引子空间，反之亦然。** 德国的公海舰队在北海（已被英国纹理化的强吸引子空间）中无效，不是因为舰队不够大，而是因为舰队的设计逻辑（在弱吸引子空间中建立临时界域）与目标空间的实际状态（强吸引子化的英国海上势力范围）不兼容。

在这个意义上，"有限性"不是关于"少投入"的自我约束，而是关于**"给正确的东西用正确的空间逻辑"**的本体论自觉。有限性海陆双向战略不是两个方向上有一样多的资源，而是知道如何在两种空间类型之间切换和分配注意力，并根据不同空间类型的吸引子结构配置不同的装置。

### 4.3 实在界的有限性：欲望与剩余的可管理性

第三层"有限"来自实在界的压力——即每一套战略装置运作时产生的未被完全计算的剩余。

实在界在战略层面的核心含义是：**战略的理性计算永远不完整。** 在任何战略框架中，总有一些因素被"排除"——不是因为它们的决策者不了解，而是因为框架本身的建构性排他操作。对于海陆复合型国家而言，这种排他操作有以下几种主要形式：

- **欲望被排除于计算之外**：威廉二世的海军建设在理性的威胁平衡框架中不成立，但它不是由"理性"驱动的。德勒兹的欲望生产揭示了一个核心机制：战略决策往往不是从安全威胁出发推导对策，而是从一种前理性的"想要成为某种国家"的欲望出发，反向建构安全威胁论述。被排除的欲望以实在界的形式回归——它既不响应成本收益分析，也不被战略咨询所说服。

- **制度化惯性的排除**：苏联的装置决策往往不是基于对当前战略形势的重新评估，而是基于以往决策的模式复制。制度化的"路径依赖"产生了一种结构性的剩余——每个后继决策都被前一个决策所约束，最终走向一个没有人"选择"但所有人都无法逃脱的终点。

- **对手的自反性被排除**：每一套战略都假设对手按照同一框架回应。但对手的自反性——对手有自己的战略文化、欲望逻辑和制度惯性——恰是战略框架中最常被排除的变量。法国未曾预料七年战争会在北美爆发（对手的自反性被排除）；德国未曾预料英国会为了比利时的中立而参战（对手的价值观和承诺逻辑被排除）。

"有限性"的第三层含义是：**战略决策者必须承认，任何战略框架都包含一个"不透明的内核"——这个内核不能被完全知识和操控，但它决定了战略的长期命运。** 对实在界的有限性的认识，不是让决策者放弃战略分析，而是让决策者保持对战略分析本身的反思：什么被排除了？什么被假设了？欲望在哪？

### 4.4 正无与负无的辩证法：有限性的操作化

卡[[20260622-0019]]对Žižek的"正无/负无"区分进行了系统的处理。Zevnik的虚空是"正无"——肯定性的可能性海洋，一切秩序从中涌现又消失。Žižek的"少于无"是"负无"——驱力的残余、剩余快感、无法被吸收的症状。

将这一区分引入战略分析会发现一个深层悖论：**海陆双向战略的"有限性"操作，本质上是在"正无"和"负无"之间维持动态平衡的实践。**

弱吸引子空间（海洋）作为"正无"呈现——它是开放的可能性场域，允许不同方式的界域创造。但海陆复合型国家在弱吸引子空间中展开战略实践时，总会产生"负无"——被排除的欲望剩余、制度化惯性、无法被框架吸收的自反性对手。

有限性战略的第四层含义是：**不试图消灭"负无"（剩余），而是与它共处——在战略框架中为不可计算之物留下空间。** 德国如果能意识到其海军扩张中的欲望成分（"作为强国的海洋"）并对其设限，而非让欲望完全殖民战略理性，它就不需要选择"不做海陆双向战略"——它只需要选择"不完全相信自己的战略论述"。

这听起来像是战略研究中的"反思性"或"自反性"的一般建议，但三层本体论的框架赋予它更具体的操作含义：**在虚空中建立秩序（正无的运作），承认总会产生未被计算的剩余（负无的涌现），保持装置的弹性以便在剩余失控时重新配置（正负之间的动态平衡）。** 法国在拿破仑时代的失败，本质上是装置被负无完全捕获（革命能量退化为征服欲望）的后果；苏联的失败，本质上是正无被意识形态封闭的后果（虚空的"一切可被不同组织"被历史唯物主义话语所填充）；德国的失败，本质上是正无与负无之间的平衡被欲望打破的后果。

## 五、有限性战略的一般原则

基于以上分析，可以提炼有限性海陆双向战略的四项一般原则。这些原则不依赖于特定时期的政策倡议（如"一带一路"），而是从海陆空间的本体论差异——三层框架的描述——中推导出的结构性要求。

### 5.1 本体论自觉原则

海陆双向战略的首要能力不是资源分配的能力，而是判断目标空间实际本体论状态的能力。一个在1970年代还是弱吸引子的水域（如南海）可能在2020年代已经被部分纹理化（通过岛礁建设、专属经济区主张、军事存在）；一个在冷战后被充分纹理化的空间（如东欧）也可能在某些维度上被重新平滑化（通过信息战、灰色地带行动、非国家行为体的渗透）。

本体论自觉要求战略思维从"这个空间属于谁"的地缘政治问题转换到"这个空间目前处于什么样的吸引子状态"的空间本体论问题。前者已经预设了纹理化的完成（有"谁"意味着有所有权）；后者承认空间始终在两种逻辑之间波动，允许战略在不同阶段对不同状态做出回应。

### 5.2 再辖域化有限性原则

在弱吸引子空间中创造界域是必要的——持续的巡航、联合演习、海外保障设施——但再辖域化必须是有范围、有节制、可逆的。一个"全宇宙的界域"不仅不可能实现，而且其追求本身就会触发他者的抵抗和再平衡行为。

再辖域化有限性的操作性含义是：**不在弱吸引子空间中追求强吸引子空间才能达到的秩序完成度。** 在强吸引子空间中，你可以建立永久基地、划定主权边界、派驻驻军——秩序可以"一劳永逸"（至少在一段时间内）。在弱吸引子空间中，有效秩序不需要"永久基地"——它需要的是可持续的在场能力、可反复激活的界域标记、可快速切换的存在模式。法国的失败部分在于它试图在弱吸引子空间中建立强吸引子般的殖民秩序；德国则试图在一个已被纹理化的空间中当它还是一片虚空。

### 5.3 虚拟/实际平衡原则

从三层框架的虚空/实在界的张力中可以推导出一个关键的操作性区分：战略力量的实际投入（物理资源、人员、武器）和虚拟效果（威慑力、信号传递、认知影响）之间不存在线性关系。

有限性海陆双向战略在这两个维度之间建立战略性分离：用实际投入的有限性换取虚拟力量的持续性。一支舰队不需要实际参战，它就已经在改变区域的力量计算；一次航行自由行动不需要占领任何领土，它的效果就在于持续改变空间的界域感知。在弱吸引子空间中，虚拟力量往往比实际力量更有效——因为吸引子浅，不需要推动系统很大距离就能改变其行为。

但虚拟/实际平衡的关键边界条件是：**如果力量从来不任何一次被实际使用——永远停留在纯虚拟状态——其虚拟效果最终会衰减。** "存而不用"只有在"用"的可能性被感知到时才有效。"有限"资源投入的核心不是"少花钱"，而是维持虚拟力量的可信度。

### 5.4 编码意识原则

"有限性"的最高级操作含义涉及战后国际体系中的编码系统的结构性变迁。在传统的威斯特伐利亚体系中，和平与战争之间有一道相对清晰的编码分界线：打了就是战争，不打就是和平。但核武器的出现、经济相互依赖的深化、全球化的推进，使得全面战争的编码极被封锁——不是通过政治选择，而是因为一旦触发这个编码，后果不可接受。当"全面战争"这个极点被封锁之后，国家间的冲突行为只能流向编码之间的灰色地带——网络攻击、贸易战、灰色地带行动、信息战、认知战——这些新型冲突形式在"和平/战争"的二元编码内无法被归类。

有限性海陆双向战略的核心能力是**在编码之间操作的能力**：在不触发"战争"编码的前提下改变空间的界域格局。有限进攻是在一个空间中推进但不让对方能将此行为编码为"战争行动"——航行自由行动、"民兵"船队、紧急避险渔船编队——这些行动在中文/国际法的编码系统中被各自归类为完全不同的行为类型。积极防御是在不退到消极被动的前提下保持稳健——不是放弃进取，而是保证每一步进取都不触发对手的编码系统将整个局势锁定为"战争"状态。

在弱吸引子空间中，编码权本身就是战略资源：同一支编队的同一项行动，在不同国家、不同时间、不同话语框架下可以被编码为从"和平通行"到"武装挑衅"之间的任何位置。有限性战略的关键能力就在此：在他人对你的行为进行最不利的编码之前，调节行为的密度、节奏和符号意义，以维持编码间操作的空间——不让任何一个单一行为被锁定为可以定义整体局势的"编码事件"。

## 六、结论

本文用三层本体论框架（虚空—内在性平面—实在界）重新奠基了海陆双向战略的理论基础，证明了"有限性"不是一个策略选择而是一个本体论条件。

海陆双向战略的核心挑战不是资源在两个地理方向上的分配争议，而是战略实施者必须同时在两种不同的吸引子结构空间中维持装置的有效运作。陆地是强吸引子空间：秩序可以稳定建立，边界具有自我维持能力。海洋是弱吸引子空间：秩序始终是局部的、临时的和需要反复维护的。两个空间的差异不是地理类型学的差异，而是吸引子结构强度的差异——这直接决定了战略装置的配置逻辑和历史案例中"失败"的原因。

在虚空层面，海陆差异不是性质差异而是纹理化程度的差异——这要求战略摆脱"海/陆性质固定"的预设。在内在性平面层面，两种吸引子结构的差异要求装置配置与目标空间类型匹配——法国、德国和苏联的失败分别源于解域化过度、欲望替代计算和虚空被封闭这三种不同类型的本体论错配。在实在界层面，战略的理性计算永远会产生不能被框架吸收的欲望剩余——威廉二世德国的案例揭示了欲望生产如何殖民战略理性。

"有限性"海陆双向战略不是"少做"或"不做"消极建议，而是对上述三层本体论条件的综合回应：对虚空的肯定（秩序总是可重构的），对吸引子结构的敏感（不同空间需要不同装置），对实在界的承认（剩余不能被消除只能被管理），以及在正无和负无之间维持动态平衡的装置配置艺术。

三层框架提供了比现有的"平滑/纹理空间"框架更深厚的哲学基础。平滑/纹理描述的是空间状态；三层框架解释了**为什么**空间处于这种状态——不是因为德勒兹和加塔利说它是，而是因为海洋的弱吸引子结构决定了任何在它上面建立的秩序都需要持续的能量输入才能维持，而陆地的强吸引子结构允许秩序一旦建立就享有自我维持的正反馈。"有限性"的概念由此获得了它的哲学尊严：它不是一个无奈的选择，而是对空间本体论条件的准确认知。
