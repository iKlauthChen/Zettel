# 安全与战争机器——虚拟的管辖权之争

## 一、一个对称的结构

基根说战争是一种文化，德勒兹说战争机器是国家捕获前的逃逸线，Reid用战争机器重新读了国际政治。但安全在这个图景中没有位置——或者说，安全被隐含地等同于不战争的日常状态。这个默认设定掩盖了一个结构性的对称。

战争机器是虚拟的。在德勒兹的本体论中，"虚拟"不是"还未实现"的同义词——它是最严格意义上的实在，只是不与经验事实重合。战争机器作为虚拟结构，是平滑空间提供的逃逸可能、是欲望不被编码时的运动轨迹，是"让新事物出现"的那种潜能。战争是这一虚拟被国家装置捕获后的实际化——从逃逸变为编码，从创造变为毁灭，从运动变为制度。

安全是这一过程的逆操作。安全不是战争的反面，安全是阻止战争机器实际化的装置操作。如果战争机器释放潜能，安全系统则负责关闭潜能——把无限的可能威胁裁剪为有限的安全议程，把不可预测的未来锁定为可管理的风险，把虚拟的逃逸线网格化为可识别的威胁类型。

战争机器和安全性争夺的不是暴力的垄断——它们争夺的是对**虚拟的管辖权**。

## 二、安全作为编码操作

安全从什么时候开始？不是从威胁出现的那一刻。对德勒兹来说，装置（apparatus/agencement）的工作是编码、梳理、界域化。安全系统就是装置的安全子系统：它负责将尚未被命名的、前个体的、流动的不确定性转化为一个"安全问题"。

哥本哈根学派的"安全化"理论在这个框架中获得了一个更深的本体论定位。安全化不只是言语行为——政治精英说"这是安全问题"的时刻不只是改变议题的性质。它是一个本体论操作：**将虚拟中的威胁潜能实际化为一个可处理的安全议程**。

说"这不是安全问题"则是逆操作——去安全化。将实际上已经存在的不安全排除在可辨识的威胁坐标系之外。

每一次安全化都是对虚拟的一次管辖权声明。

## 三、威慑的悖论性位置

核威慑在这个框架中暴露了一个结构性的悖论。

威慑理论的核心逻辑是：国家将战争机器捕获为毁灭性潜能（核武库），然后安全系统要求这一潜能停留在"虚拟"状态——永远不实际化，但永远可见。核武器被存放在发射井中，作为纯粹潜能的象征存在。

这个位置是悖论性的。战争机器被国家捕获后，它的实际化（核战争）是不可接受的，因此国家必须阻止它实际化。但战争机器的虚拟性不能因此被取消——如果它完全退回到虚拟，威慑就失效了。因此安全系统必须把战争机器冻结在一个半实际化的临界点上：足以可见，但不允许使用。用卡尔·施密特的术语说，威慑就是"被搁置的决定"的绝对形式。

安全不消灭战争机器——它把战争机器钉在虚拟和实际之间，让它在存在论意义上无法选择自己的方向。这是安全的最高成就，也是安全自身的最大风险：一旦临界点被误判，战争机器就从"被冻结的实际化"直接跳入不可逆的实际化。

## 四、安全的不可能性与剩余安全

如果安全是装置对虚拟的关闭操作，那么一个本体论推论不可避免地出现：**每一次关闭都会留下未被编码的剩余**。

因为虚拟在德勒兹的意义上是不可穷尽的。装置可以裁剪经验事实、建立分类体系、制度化风险评估程序——但它无法清除虚拟本身。总有一些恐惧是未被命名的，总有一些脆弱性是制度不承认的，总有一些安全诉求无法被纳入现有的威胁坐标系。

这就是"剩余安全"（surplus security）概念获得的哲学根据。剩余安全不是装置的操作失误——不是"安全系统还不够完善"的暂时遗憾。它是本体论必然：因为虚拟是不可穷尽的，每一次安全操作都必定会漏掉某物。剩余不是安全的失败，而是安全的构成性边界。

剩余安全可以表现为两种形式。在装置内部，它是下一次操作要修补的漏洞——情报机构的"未知未知"永远是下一个预算申请的理由。在装置外部，它是被主流安全话语排斥的不安全的表达形式——边缘社群的生存恐惧、非传统安全议题在制度化过程中的扭曲、以及那些根本不被承认为安全问题的持续存在性威胁。

战争机器为剩余安全提供了另一层意义：剩余不只是装置的失败，它也是战争机器的在场证明。那些不被安全系统编码的、残留的、无法被制服的不确定性，恰恰是战争机器未被完全捕获的信号。装置永远无法彻底关闭虚拟。

## 五、对国际政治本体论的含义

如果将安全重新定位为装置对虚拟的管辖权操作，国际政治的基本问题就发生了变化。

主流安全研究问的是：国家如何保障安全？批判安全研究问的是：谁在安全化什么？D&G—Reid框架问的是第三个问题：**每一次安全操作关闭了什么可能性？**

"什么被关闭了"与"什么被保卫了"是同一个装置操作的两个面。每一次威胁识别都是一次本体论筛选——从虚拟的无限可能中挑选出一个子集作为"真实威胁"，而其余的可能性被排除在安全议程之外，退回虚拟之中。这个被排除的可能性集合，正是安全操作的成本——不体现在预算中，但体现在虚拟的管辖权从"开放的生成"转移到"被编码的可管理性"这个运动本身。

剩余安全的政治学在此展开：一种不以恐怖和控制的循环来定义安全，而是以维持虚拟的开放性来定义——不是防止坏事情发生，而是防止安全装置本身对虚拟的过度编码。在最根本的意义上，剩余安全追求的不是安全的完善化（永远不可能），而是安全的去装置化：释放那些被安全操作关闭但不应被关闭的可能性。

战争机器最终提供的不是说战争是好事，也不是说安全是坏事。它提供一个本体论的度量维度：每一次安全操作都可以被问一次——这次操作是让事物变得更加定型、更加可预测、更加服从于装置的逻辑，还是在安全的同时保留了逃逸的可能？

答案会告诉我们，这一段安全化是在编码还是在解放。
