# 剩余安全：本体论涌现、拉康式对象小a与德勒兹式编码重塑

## 研究提纲草稿

---

## 一、研究问题

安全实践（securitization）是否系统性地生产出一种无法被消除的结构性"剩余"（surplus），而这一剩余具有本体论地位，而非仅仅是认识论意义上的建构产物或话语效果？如果是，这一剩余在具体的地区安全实践中，是如何被组织、被反复再生产，又是否存在被引向不同组织方式的可能性？

本研究试图论证：安全生产中的结构性剩余，可以被恰当地理解为拉康意义上的"对象小a"（objet petit a）；而对这一剩余的分析，不应止步于诊断其存在，还应进一步追问——在承认剩余不可消除的前提下，安全实践的具体编码方式是否存在从偏执型的封闭循环，转向更具生产性、更少压迫性的配置的可能。

---

## 二、理论框架

### 2.1 从跨领域"裂隙"到本体论涌现的初步论证

- 出发点：政治经济学中的"剩余价值"概念，与语言学中能指/所指的裂隙、安全建构中的安全困境，呈现出结构上的相似性。
- 关键区分：形式同构（formal isomorphism）、结构同源（structural homology）、隐喻性挪用（metaphorical borrowing）——三者对本体论主张的支撑力度依次递减，需要明确本研究的论证落在哪一层级。
- 支撑性论证资源：
  - 柄谷行人《跨越性批判》中"买-卖"与"教-学"关系的不对称性——缺乏第三方裁决位置、必须依赖非必然的"跳跃"弥合关系。
  - 自指系统的形式定理谱系：哥德尔不完备定理、塔斯基真理不可定义定理、Lawvere不动点定理——凡满足"系统试图从内部确证自身与外部对应关系，却缺乏第三方验证位置"这一形式条件的系统，裂隙的出现具有数学必然性，不依赖观察者认知状态。（详见 [[20260713-0005-formal-theorems-security-fissure]]）
  - 奎因《本体论的相对性》——翻译不确定性被奎因本人明确定性为本体论后果，而非认识论局限，为"分析哲学传统内部独立走向本体论化"提供关键支点。（详见 [[20260713-0006-quine-ontological-relativity]]）

### 2.2 拉康式诊断：剩余安全作为对象小a

- 核心论点：安全生产系统性地围绕一个无法被真正占有、却持续驱动行为体行动的"对象因"（object-cause of desire）运作。
- 关键概念挪用：
  - 本体论维度（空的容器，纯粹匮乏本身）与经验维度（具体能指，如"科索沃"、"南海诸岛主权"）的区分。
  - 幸福性幻想（beatific fantasy）与恐怖性幻想（horrific fantasy）的双重结构。
  - "享乐的窃取"（theft of enjoyment）——结构性失败被转嫁给"他者"的机制。（详见 [[20260713-0008-theft-of-enjoyment-security]]）
- 方法论警示：需要与主流"本体论安全研究"（Mitzen 2006; Steele 2008，Laing-Giddens范式）明确区分——后者讨论的是身份连贯性的维持，与本研究"赋予裂隙/剩余以本体论地位"并非同一概念操作。

### 2.3 德勒兹式"临床"：拒绝目的论式解决的编码重塑

- 关键理论张力：德勒兹与加塔利在《反俄狄浦斯》中明确以"欲望是生产，而非匮乏"反对拉康"匮乏是存在之基"的立场，两者在欲望的基础理论上正面对立，不可无缝衔接。
- 本研究采取的解决路径（路径二）：不调和两者的形而上学，而是将其分置于不同分析层面——
  - 拉康处理**共时性的结构必然性**（为何剩余必然产生）；
  - 德勒兹处理**历时性的生成过程**（这一必然性在具体历史配置中，是被封闭性地重复领域化，还是可能被引向解域化）。
  - 可借助批判实在论（Bhaskar）的分层本体论（实在的/现实的/经验的领域）为这一分层提供更节制的元理论表述。
- "临床"概念的精确界定：借鉴德勒兹"批判与临床"传统（Coldness and Cruelty; Critique et Clinique）与精神分裂分析——将安全实践作为症状而非表征来读，追踪其背后的编码/领域化机制，定位可能的解域化路径。
- 核心限定：本研究明确**不**承诺剩余可被消除或"治愈"。"治疗"仅指：在承认剩余不可消除的前提下，追问安全实践的编码方式能否从偏执型重复，转向承认剩余、却仍能维持行动生产性与灵活性的配置。

## 三、方法论：从理论概念到可编码经验指标

本节将第二节的理论概念——对象小a、幻想、享乐窃取、偏执型重复、编码重塑——操作化为可用于经验案例分析的编码指标。方法论基础是Lene Hansen（2006）的后结构主义话语分析框架，经Vulović与Ejdus（2024）在拉康式本体论安全研究中的具体改编。

### 3.1 分析范畴操作化

#### 3.1.1 对象小a（objet petit a）的识别步骤

借鉴Vulović与Ejdus（2024）的三步识别法，适配于安全话语分析：

| 步骤 | 操作 | 安全话语中的具体实现 |
|------|------|---------------------|
| **步骤一：追踪重复模式** | 识别话语中围绕特定能指的反复回返——它被反复提及、反复争论、反复作为行动的理由出现 | 在安全政策文本/官方声明/外交辞令中，哪些议题/地名/身份标记被"不可绕过地"反复提及？频率不等于重要性，但高频+高情感强度构成候选 |
| **步骤二：结构性位置检验** | 检验该能指在叙事结构中的功能——如果去除它，整个安全叙事是否崩塌？ | "南海主权"在北京的南海叙事中是否构成不可移除的结构性节点？去除后"中国为什么要维护海洋权益"的叙事是否失去因果支撑？ |
| **步骤三：不可占有性检验** | 检验该能指是否始终处于"触手可及但不可占有"的状态——每次安全化操作都宣称距离"拥有"更近一步，但从未真正到达 | 六方会谈的"无核化"目标：每一轮会谈都宣布"取得重要进展"，但"完全无核化"这一目标始终未实现——且每次"进展"后新的"障碍"随即出现 |

**编码规则**：同时满足三步检验的能指，被编码为"对象a候选"。在案例分析中，记录（a）候选能指的名称和文本来源，（b）它在叙事中的结构位置（哪个因果链依赖它），（c）支持"不可占有性"判定的证据（历史轨迹中的"接近但从未到达"模式）。

#### 3.1.2 幻想的双重结构：幸福性幻想与恐怖性幻想

在安全话语中，幻想不是"虚假信念"而是"组织欲望的叙事框架"（Žižek 1989）。本研究区分两种幻想结构：

| 幻想类型 | 叙事结构 | 安全话语中的典型形式 | 编码指标 |
|---------|---------|---------------------|---------|
| **幸福性幻想** (beatific fantasy) | 描绘欲望实现后的和谐状态——"如果X被解决，一切都会好起来" | "南海行为准则（COC）一旦达成，地区和平将得到制度保障"；"半岛无核化实现之日，东北亚安全架构将稳定下来" | 文本中出现条件句结构："一旦……就……"、"当……时候，将……"；且被描绘的"实现后状态"具有静止、和谐、永久解决的特征 |
| **恐怖性幻想** (horrific fantasy) | 描绘他者享受了本应属于我们的享乐——"他们在窃取我们的东西/破坏我们的秩序" | "域外国家在南海享受航行自由的同时损害中国主权"；"朝鲜以核武库为筹码在国际制裁下享受特权" | 文本中出现"他者享受/获得不应得之物"的叙事；且这种享受被呈现为"零和"——他者之得=我方之失；常用动词：窃取、破坏、享受（在谴责语境中）、获得不正当优势 |

**编码规则**：每份文本同时编码两种幻想的存在与否和强度（高/中/低/无）。由于两种幻想往往在同一叙事中配对出现（恐怖性幻想标记了"被窃取的享乐"，幸福性幻想标记了"被恢复的完满"），记录二者的配对模式。

#### 3.1.3 "享乐的窃取"（theft of enjoyment）的操作化

"享乐的窃取"是拉康式分析中连接幻想与他者化的核心机制。在安全话语中，它指结构性失败（安全永远不够）被转嫁为他者的道德-政治责任的操作。

**编码指标（四维）**：

1. **失败归因的外部化**：安全实践的结果未达预期时，原因被归于特定他者（"因为X在破坏/阻挠，所以……"）而非结构性条件本身
2. **他者的道德化**：该他者被赋予道德缺陷——不仅"阻挠了我们"，而且"不怀好意"、"居心不良"
3. **享乐的零和化**：故事被构造为"他者从中获得了利益/快乐/优势，而我们被剥夺了应有的东西"
4. **自我叙事的受害化**：自己一方在这一叙事中被定位为"受害方"——不是简单的利益受损，而是"本应属于我们的东西被夺走了"

**编码模板**：

| 维度 | 指标 | 数据来源 | 编码值 |
|------|------|---------|--------|
| 失败归因外部化 | 安全挫折的原因陈述中，他者作为主因的频次 | 外交声明、国防白皮书 | 有/无；频次计数 |
| 他者道德化 | 他者被赋予道德化谓词（"恶意的""不负责任的""挑衅的"）的频次 | 官方声明、发言人答记者问 | 有/无；频次计数 |
| 享乐零和化 | 文本中出现"他者获得=我方失去"的零和构造 | 政策文件、战略报告 | 有/无 |
| 自我受害化 | "我方是受害方"的叙事定位 | 外交声明、国内动员文本 | 有/无；程度评估 |

### 3.2 核心区分：偏执型重新领域化 vs. 解域化式编码重塑

这是本研究的核心分析工具，用于判定安全实践的编码方向。二者不是二元对立，而是在一个连续统上的两个极点。具体操作中，一段安全实践可能同时包含两种倾向。

#### 3.2.1 偏执型重新领域化（paranoid reterritorialization）

**定义**：结构性空位保持不变，仅更换具体能指内容；行为体反复诉诸同一套"他者窃取享乐"的叙事结构，安全化操作的编码框架未被触碰。

**编码指标（五条，满足三条及以上则判定为偏执型）**：

| 指标 | 操作化描述 | 数据表现 |
|------|-----------|---------|
| **P1. 能指替换而非结构变更** | 安全叙事的核心空位（对象a的位置）本身从不被触动，只是在空位中放入不同的具体内容 | 从"科索沃是不可放弃的"到"科索沃已是我们的但被非法占领"，空位不变，填充物微调 |
| **P2. 享乐窃取的叙事再生产** | 每一轮安全化操作都重新生产"他者窃取=我方受害"的叙事，叙事结构本身不被反思 | COC谈判每轮破裂后的官方声明都重复"某些国家缺乏诚意/另有图谋"的归因模式 |
| **P3. 截止日期的反复重置** | 被设定的"解决方案时间点"反复被错过，错过之后不是追问框架本身的问题，而是设立新的截止日期 | COC谈判2002→2018→2023→2026年截止日期的连续重置；六方会谈2003→2005→2007→2009的共同文件→失效→新文件的循环 |
| **P4. 欲望的自我循环** | "更多安全化操作"被呈现为解决安全化操作失败的方案——每一次失败不是引发"为什么这样还不够"的反思，而是引发"我们需要更多"的投入 | 朝核问题中"制裁不够→加强制裁→还不够→再加强→朝方反而加速核试验"的循环 |
| **P5. 诉诸根本性"他者"** | 系统性僵局被解释为一个不可改变的"根本性他者"的恶意——该他者的存在本身使得问题的结构性解决不可能 | "某国从来就不希望半岛稳定"、"域外国家的根本目的是遏制中国"——这些陈述将结构性僵局转化为特定行为体的意志，从而取消了追问制度框架本身的必要性 |

#### 3.2.2 解域化式编码重塑（deterritorializing coding reshaping）

**定义**：行动的组织坐标从"裁决不可裁决的问题"切换为"在承认不可裁决的前提下管理具体功能性事务"，且这一切换具有可观察的制度化痕迹。

**编码指标（五条，满足三条及以上则判定为编码重塑）**：

| 指标 | 操作化描述 | 数据表现 |
|------|-----------|---------|
| **D1. 从裁决到管理** | 行动目标从"解决根本争议"（裁决主权归属/历史定性/合法性）切换为"管理具体事务"（功能合作/危机管控/技术协调） | 南海：从"谁拥有这些岛礁"转向海警热线、渔业共同管理、海洋环境保护合作 |
| **D2. 议题降敏感化** | 高政治争议被主动搁置或绕过，行动聚焦于低政治功能性领域，且这种绕过是有意的制度设计——而非无意的忽略 | 中日韩三边合作刻意限定经济/环境/文化议题，避开历史/领土问题 |
| **D3. 制度化痕迹** | 功能合作的制度设计具有可观察的组织形式——机制设立、定期会晤、热线通道、联合行动程序 | 中韩海警热线、中日防务部门海空联络机制、COC框架下的热线建立 |
| **D4. 对僵局的承认而非否认** | 公开话语中承认"某些问题在可预见的未来无法被裁決"，并将此承认纳入制度设计之中——而非将僵局归因为他者恶意 | "搁置争议、共同开发"——承认主权争议不可即时解決，但不因此放弃一切合作 |
| **D5. 编码框架的弹性** | 安全化框架本身显示出可调整性——它不将某一议题永久锁定在"安全威胁"的编码中，而是允许议题在不同编码（安全/经济/技术/环境）之间移动 | 渔业纠纷在某些时期被安全化，在另一些时期被去安全化为技术性资源管理问题 |

#### 3.2.3 连续统与混合型

实践中，同一案例可能在不同维度或不同时段表现出混合特征。本研究不强制每个案例进入非此即彼的分类，而是使用以下评分矩阵：

| 案例/时段 | P1-P5 满足数 | D1-D5 满足数 | 混合模式描述 |
|-----------|-------------|-------------|-------------|
| 南海COC谈判(2002-2018) | | | |
| 南海COC谈判(2018-2023) | | | |
| 南海功能合作（海警热线等） | | | |
| 朝核六方会谈(2003-2009) | | | |
| 中日韩三边合作(2008-至今) | | | |

关键检验不是"这个案例属于哪一种"，而是**在安全的不可裁判性被承认的条件下，偏执型操作和编码重塑操作分别以何种方式共存、在何种条件下此消彼长**。

### 3.3 方法论风险与边界

**风险一：套用德勒兹术语重新命名既有IR概念。** 必须警惕将"议题联系"（issue linkage）重命名为"解域化"、将"低政治/高政治分层"重命名为"编码重塑"的倾向。本研究与既有功能主义理论的核心分界线在于——功能主义预设问题可通过外溢走向实质性解决（目的论预设），本研究明确拒绝这一预设。编码重塑不"解决"任何根本争议，它只是在承认不可解决的前提下切换操作模式。如果一段实践中出现了"功能合作→信任积累→争议实质性解决"的叙事，那恰恰是功能主义而非编码重塑。**编码重塑的"成功"标准不是问题被解决，而是行动从"被不可裁决问题所困的偏执型循环"中被释放到"可操作的功能领域"**。

**风险二：编码指标的可争议性。** 上述指标（尤其是P1-P5和D1-D5）本质上是阐释性的——同一段官方声明，不同的编码者可能做出不同的判断。本研究采取两项措施管理这一风险：（1）为每个指标提供至少两个具体文本示例作为编码锚点（coding anchor）；（2）在案例分析中，不仅呈现编码结果，还呈现支持该编码的证据段落，使读者可以独立评估编码的合理性。

**风险三：规范性目标的隐含性。** "编码重塑优于偏执型重复"这一价值判断已经内置在概念命名中（"重塑"暗示改善）。这一规范性预设不等于"解域化在政治上是好的"——正如草案2.3节所限定的，本研究不承诺剩余可被消除。（详见"临床"概念的方法论限定）

## 四、案例研究设计

### 4.1 南海：COC谈判作为偏执型重复的范例

- 经验材料：COC谈判自2002年《宣言》、2018年正式文本谈判、2023年三年内完成的承诺、至2026年菲律宾担任东盟轮值主席国"年底前完成"的最新表态——二十余年间反复设定截止日期，核心争议（法律约束力、地理范围、第三方角色）从未被实质裁决。
- 对照性的编码重塑迹象："主权归我、搁置争议、共同开发"的实践逻辑；海警热线机制；渔业与海洋环境共同管理安排——将行动坐标从主权裁决切换至功能性共管。

### 4.2 东北亚：两组对比案例

- 偏执型重复：朝核问题六方会谈/无核化框架的历史循环——核心僵局从未被触及，仅反复更换具体谈判议程与承诺内容。
- 编码重塑及其脆弱性：中日韩三边合作机制——刻意将合作限定在低敏感度功能性领域，主动绕开历史与领土问题；但该机制在历史/领土争端被具体事件重新激活时极易中断，呈现编码重塑作为**脆弱、需持续维护的动态平衡**而非单向进步的特征，这本身构成对德勒兹式解域化/再领域化动态博弈的具体印证。

## 五、预期理论贡献与文献定位

- 相较于既有拉康派IR文献（Edkins; Kapoor; Zevnik; Vulović & Ejdus等）多聚焦"匮乏"（lack）本身，本研究将分析焦点系统性地转向"剩余"（surplus）及其跨领域生成机制，并尝试建立处理这一剩余的、明确拒绝目的论的德勒兹式分析工具，这一双重转向目前未见于现有文献。
- 相较于Zevnik（2016）已完成的拉康-德勒兹综合研究，本研究的差异点需在文献综述中明确阐述（如：Zevnik聚焦主体本体论的一般性重构，本研究聚焦安全生产这一具体领域，并以显性的地区案例进行经验检验）。
