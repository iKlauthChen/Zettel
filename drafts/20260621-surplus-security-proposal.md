# 剩余安全（Surplus Security）：将"剩余"理论模型迁移到国际关系安全概念

## 一、"剩余"模型的结构同构性

"剩余"作为跨理论迁移模型的核心形式结构：

| 理论家 | "剩余"概念 | 系统运作 | 剩余的功能 |
|---|---|---|---|
| 马克思 | 剩余价值（surplus value） | 资本主义再生产 | 剥削的驱力 + 再生产的动力 |
| 马尔库塞 | 剩余压抑（surplus-repression） | 文明/统治 | 超出基本本能的额外压抑 |
| 拉康 | 剩余快感（surplus jouissance） | 欲望/象征秩序 | 对象a—欲望的对象-成因 |
| 齐泽克 | 剩余服从（surplus-obedience） | 意识形态/权力 | 超出生存所需的服从 |
| 齐泽克(主体论) | 主体作为剩余 | 符号秩序 | "丢失的链条"—系统一致性的条件 |

**形式定义**：在所有案例中，"剩余"指系统运作过程中产生的**超出系统自我维持所需的那部分产物**——它既是系统的驱动力（没有它系统无法运行），也是系统的症状（它揭示了系统的内在不一致性）。

## 二、剩余安全（Surplus Security）的理论构想

### 2.1 基本安全 vs 剩余安全

借用一个形式对位：

| 基本安全（basic security） | 剩余安全（surplus security） |
|---|---|
| 维持生存所需的安全保障 | 超越理性生存需要之外的安全追求 |
| 防御性现实主义的安全逻辑 | 进攻性现实主义的核心驱力，但不止于此 |
| 安全困境的"物质面" | 安全困境的"符号面" |
| 可满足的（sufficient security） | 不可满足的（insatiable security drive） |

**剩余安全的定义**：国家（或其他安全主体）在已经满足基本生存安全需求后，仍然持续追求超出理性限度的安全的行为与驱力。但这个定义不是"过度反应"的心理学术语——剩余安全不是个体心理的病态，而是 **国际体系的结构性产物**。

### 2.2 剩余安全与四种迁移

**① 剩余价值 → 剩余安全（马克思）**

安全困境中的"和平-军事开支的悖论"：国家将资源投入军事，超出"合理防御"的需要。这部分"多余"的军事开支=剩余安全的生产。剩余安全不是"多余的安全"（太多的导弹），而是**安全的生产关系本身所产生的超出理性计算的驱力**。正如剩余价值是资本主义再生产的核心驱力但被掩盖在"公平交易"的幻象下，剩余安全是国家体系再生产的核心驱力但被掩盖在"防御必要性"的幻象下。

> "在资本方面表现为剩余价值的东西，正好在工人方面表现为超过他作为工人的需要，即超过他维持生命力的直接需要而形成的剩余劳动。"（马克思）
>
> → 在霸权国方面表现为剩余安全的东西，正好在边缘国方面表现为超过其维持主权所需的剩余不安全感。

**② 剩余快感 → 剩余安全（拉康）**

剩余安全不只是物质性的（武器、预算），它也是**符号性和欲望性的**。主权国家的身份本身建立在一种"剩余快感"之上——通过对他者的排斥（主权边界）和对内在例外的管理（紧急状态）来获得主权主权身份的快感。安全话语的功能不仅是保护国民，也是为主体（国家/公民）提供一种"痛苦中的欢乐"——认同于国家安全叙事所带来的归属感和崇高感，即使这种认同以牺牲自由和资源为代价。

> "剩余快感并非快感的剩余或多余，而是通过认同和接受象征秩序而获得的快乐，其基础是痛苦。"（赵淳）
>
> → 剩余安全不是安全的剩余或多余，而是通过认同和接受国际安全秩序而获得的"安全"，其基础是不安全感。

**③ 剩余压抑 → 剩余安全（马尔库塞）**

国家安全机器在"保护安全"的名义下施加的**超出基本需要的额外压抑**——监控社会、例外状态、边境暴力。这些不是在"不安全"环境中的必要手段，而是在安全叙事下被合法化的剩余压抑。正如马尔库塞指出在物质极大丰富的现代社会中压抑以"剩余压抑"的方式体现——在安全技术极大发达的今天，不安全感也以"剩余压抑"的方式体现。

> "剩余压抑由处于特权地位的团体、机构和个人来实行，受统治者的特殊利益所支配，旨在维持和扩大某种统治制度。这是对本能的基本压抑之外附加于人的压抑，是不必要、不合理的。"（赵淳转述马尔库塞）

**④ 主体作为剩余 → 安全的主体位置（齐泽克-拉康）**

安全秩序的"加一"（plus-One）：国际体系为了构成自身，必须有一个既在其内又在其外的元素——"主权"本身可能就是这一个"加一"。它使国际社会成为可能（每个国家是一个自足的单位），也使国际社会永远无法闭合（主权排除了真正的全球共同体）。**无实体主体性**在安全领域的对应：那些被排除在安全秩序之外的存在（难民、无国籍者、"失败国家"）不是安全秩序的边缘，而是安全秩序构成自身必须排除的剩余——它们的存在暴露了安全秩序的不一致性。

## 三、剩余安全的内在性平面：概念创造检验

用孙利天的概念创造三原则（[[20260621-0463]]），检验"剩余安全"是否是一个真正的概念创造还是一个造词运动：

### 原则1：客观性——锚定在可观测的物质过程中

剩余安全可以锚定在以下物质过程中：
- **军事开支超出理性防御需求的部分**：从数据上看，全球军费开支（2024年超2.4万亿美元）远远超出任何"防御所需"的合理估计
- **监控/国家安全的扩张**：后9/11时代监控国家的指数级膨胀
- **安全化操作的溢出效应**：从气候变化到公共卫生，越来越多的领域被纳入"国家安全"的框架
- **安全-工业复合体**：军工产业对安全话语的塑造能力

**结论**：✅ 通过。有可观测、可度量的物质过程锚定。

### 原则2：对立统一——从真实矛盾中生长

剩余安全根植于国际关系的核心矛盾中：
- **安全困境的内在矛盾**：国家对安全的追求越多，整体体系越不安全——剩余安全正是从这个矛盾中生长出来的概念
- **主权原则与人权的矛盾**：主权安全vs个人安全的张力
- **安全与发展矛盾**：资源在安全和发展之间的分配冲突
- **剩余安全的内部矛盾**：它既是安全秩序运行的动力（安全话语使现代国家成为可能），也是安全秩序无法闭合的症状（不安全感永远无法被彻底消除）

**结论**：✅ 通过。从国际体系的真实矛盾中生成。

### 原则3：理想性——蕴含规范性方向

剩余安全提供了一条从"是"到"应当"的路径：
- **诊断功能**：揭示了国家安全追求中"超出理性防御"的那部分的结构性根源——不是简单归因于个体国家的"非理性"或"贪婪"，而是国际体系的内在矛盾
- **批判功能**：暴露了剩余压抑（监控、边境暴力、例外状态）的合法化机制
- **重构功能**：指向一种"安全观的转换"——从安全作为稀缺品（零和博弈）到安全作为"被共有的东西"
- **与人类安全的关联**：剩余安全概念打开了从"国家安全"向"人类安全/全球安全"的内在性平面转换的理论空间

**结论**：✅ 通过。提供了规范性指引。

## 四、剩余安全与已有安全理论的差异化定位

| 安全理论 | 核心论点 | 剩余安全的补充/差异 |
|---|---|---|
| **现实主义** | 安全是国家生存的优先目标，由无政府状态决定 | 解释了为什么国家在获得充足生存安全后仍不满足——剩余安全揭示了安全困境的符号面（欲望/快感/身份） |
| **自由主义** | 安全通过制度、经济相互依存和国际合作实现 | 解释了为什么制度深入发展后安全竞争仍然存在——剩余安全是系统无法闭合的"症状" |
| **建构主义** | 安全是社会建构的，通过规范内化与身份认同 | 超越了"观念/利益"二分法——剩余安全不仅是观念建构，更是系统结构的不一致性本身 |
| **哥本哈根学派** | 安全化是言语行为，将议题提升为"存在威胁" | 解释了为什么安全化操作永远不能"完成"——总有一个剩余无法被安全化捕获 |
| **批判安全研究** | 安全作为解放/从结构暴力中解放 | 揭示了"解放"本身可能的剩余——任何解放方案都产生新的剩余 |

## 五、概念的操作化可能性

### 5.1 剩余安全的四维分析框架

```
剩余安全维度的四象限：
               物质性
                 ↑
                 │
    剩余价值维度  │  剩余压抑维度
    （军费/军事化）│  （监控/例外状态）
                 │
     ←──────────┼──────────→ 符号性
                 │
   无实体主体性  │  剩余快感维度
   （难民/排除者）│  （安全叙事/爱国主义）
                 │
                 ↓
               欲望性
```

### 5.2 可研究的经验领域

1. **全球军费支出中的"剩余"部分**——哪个部分超出了合理防御需求？
2. **监控国家的符号学分析**——"安全"话语如何合法化剩余压抑？
3. **难民/无国籍者的安全秩序位置**——他们作为安全秩序的"剩余"如何暴露主权体系的不一致性？
4. **国家安全叙事的欲望结构**——为什么公民愿意为"安全"牺牲自由？（剩余快感维度）
5. **"安全-工业复合体"的知识政治**——谁生产了关于"威胁"的知识？谁从剩余安全中获益？

## 六、反思与边界

### 6.1 剩余安全理论的乌托邦陷阱

借鉴魏浥尘对齐泽克的批判（[[20260621-0485]]）：剩余安全概念如果只是揭示了"安全困境的结构性根源"而没有给出"如何超越"的具体路径，就沦为了另一种"美丽的理论"。"剩余"的神奇之处在于它同时是驱力和症状——但指出这一点不等于改变了它。

### 6.2 剩余安全与剩余快感的危险亲缘

剩余安全中最危险的可能性是它可能被用于**正当化剩余压抑**——如果"剩余"是系统所必需的，那么"多余的安全"（监控、例外状态）也是不可避免的。这恰恰是拉康派批判要警惕的"享乐的政治"的陷阱。

### 6.3 概念创造的四步检验（从[[20260621-0461]]）

| 检验 | 结果 |
|---|---|
| ① 是否锚定在可观测的物质过程中？ | ✅ 军事支出、监控扩张 |
| ② 是否从真实矛盾中长出来？ | ✅ 安全困境的内在矛盾 |
| ③ 是否改变了"什么算相关事实"的标准？ | ✅ 从"什么行为算过度"到"什么结构产生了过度" |
| ④ 是否提供了从"是"到"应当"的规范性路径？ | ⚠️ 部分通过——诊断力强于建设力 |
