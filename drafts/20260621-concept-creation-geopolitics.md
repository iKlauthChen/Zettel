# 概念创造如何应用于地缘政治：一个理论框架

> 本提案来自对孙利天、刘绍艺《哲学如何创造概念》的 ingest 切卡（[[20260621-0460]]~[[20260621-0465]]）与 Bonta & Protevi《Deleuze and Geophilosophy》卡片群（[[20260621-0437]]~[[20260621-0453]]）的交叉阅读。

---

## 核心论点

地缘政治的底层竞争不是"谁控制了什么空间"，而是**"空间以什么坐标被呈现"**。概念创造就是创立新的内在性平面，重新定义什么算作相关事实的标准。

---

## 一、地缘政治概念就是"内在性平面"

当一个概念——如"印太"、"人类命运共同体"、"修昔底德陷阱"、"文明的冲突"——被创造出来时，它不是在描述一个预先存在的现实，而是在**创立一个新的内在性平面**。

### 案例："印太"

"印太"不是印度洋和太平洋的简单地理并集——它创造了一个新坐标：
- 使印度海军成为东亚安全的"内部变量"
- 使美日印澳四国机制"自然而然"地成为该区域的合法安全架构
- 把中国置于一个被审视的"中心位置"——被海洋民主联盟从外围包围

这正符合孙利天所说："它可能没有发现新对象，却从根本上改变了对象性世界的理解方式。"

### 案例："修昔底德陷阱"（Allison, 2017）

> 这是一个从古希腊历史中提取的"概念人物"（德勒兹语）。

它将中美关系置于一个二分法的内在性平面上（"崛起国"vs"守成国"），使对抗成为唯一可预见的轨迹。其说服力不在于它是否符合历史事实（雅典和斯巴达的案例本身就存在大量争议），而在于它**提供了一个简洁的坐标尺度来组织所有关于中美关系的讨论**。

> 概念创造不是信息增量（发现新事实），而是认知架构的切换（改变"什么算事实"的标准）。

---

## 二、谁有权设定坐标尺度

用德勒兹的**多元内在性平面**来看，地缘政治概念竞争是不同坐标系之间的竞争：

| 美国系概念平面 | 中国系概念平面 |
|---|---|
| 印太战略 → 海洋民主联盟 vs 威权 | 人类命运共同体 → 共生 vs 零和 |
| 自由国际秩序 → 规则 vs 破坏规则 | 新型国际关系 → 相互尊重 vs 霸权 |
| 民主 vs 威权 → 制度优劣的二分 | 多极化 → 文明多样性的承认 |
| 基于规则的国际秩序 | 以联合国为核心的国际体系 |

**这些不是同一坐标系下的对立选项，而是不同的坐标系本身。** 在一方看来"正当的自卫行动"，在另一方看来是"扩张主义"——不是因为事实判断不同，而是因为双方内在性平面的坐标原点不同。

> 美国系平面以"个体自由"为原点，中国系平面以"主权平等"为原点。原点不同，坐标轴不同，同一个事件被分配到了象限的不同位置。

### 竞而不交的困境

多元内在性平面的不可还原性（德勒兹的核心立场）在地缘政治中的后果是：**不同概念平面之间的"翻译"几乎不可能**。这不是语言问题，而是不同平面之间的概念组分（conceptual components）没有稳定的对应关系。美国"自由国际秩序"平面中的"规则"不能简单地对应中国"人类命运共同体"平面中的"规范"——它们在各自平面中的意义由整个概念网络赋予。

这解释了为什么中美战略对话经常出现"鸡同鸭讲"的现象：不是双方不诚实或不聪明，而是他们从不同的内在性平面出发，使用的概念虽然词语相近但坐标位置完全不同。

## 三、马克思三原则作为概念创造的质量标准

孙利天提炼的三条原则，可以用来区分"地缘政治概念创造"和"地缘政治造词运动"：

### ① 客观性原则：概念必须锚定在物质过程中

| 标准 | 有效案例 | 无效案例 |
|---|---|---|
| 概念能否锚定可观测的物质过程？ | **"全球南方"** — 抓住了去殖民化后 G77+China 的经济结构位置变迁，有贸易流、投资流、国际制度投票行为等物质过程支撑 | **"锐实力"（Sharp Power）** — 造出来后未能锚定可测量的物质过程，从学术话语中迅速消退 |

### ② 对立统一原则：概念必须从真实矛盾中生长出来

| 标准 | 有效案例 | 无效案例 |
|---|---|---|
| 概念是否捕捉了结构性的矛盾关系，而非外部套用公式？ | **"修昔底德陷阱"** — 抓住了崛起国/守成国之间的结构性安全困境，矛盾是内在的 | **"冷战2.0"**（2018年前后的简单版本）— 简单类比替代了矛盾分析，没有解释2022年后中俄关系的实质演变 |

马克思对蒲鲁东的批评在地缘政治中同样有效：把概念当作现成的公式外在地套用到现实上时（"这就是新的冷战"），会丧失对现象的穿透力。真正有效的概念创造来自对内在矛盾的揭示。

### ③ 理想性原则：概念必须蕴含规范性方向

| 标准 | 有效案例 | 无效案例 |
|---|---|---|
| 概念是否提供了从"是"到"应当"的路径？ | **"人类命运共同体"** — 不仅是描述性概念（全球化和相互依存），更是规范性愿景（共生而非零和），有超越性维度 | **"美国优先"** — 虽然是成功的政治动员概念，但缺乏理想性维度（没有回答"优先之后去哪"），这解释了它在盟友中持久的说服力缺失 |

孙利天的深刻之处在于强调：**"在批判旧世界中发现新世界"**——概念的理想性不是空想的乌托邦，而是从对现实矛盾的批判中自然生成的方向。最有效的地缘政治概念恰恰是那些既揭示了旧秩序的失败、又给出了新秩序蓝图的概念。

## 四、"事件"与概念创造：危机时刻的窗口

> "事件作为本体论的断裂或间隙，是同一性逻辑的中断，是异质性、多样性的突围……意味着新概念、新原则和新的必然性的显现。"
> — 孙利天、刘绍艺（援引德勒兹/巴迪欧/齐泽克）

地缘政治中最重大的概念创造几乎都发生在危机事件之后：

| 事件 | 催生的概念 | 后续影响 |
|---|---|---|
| 9/11 | **"反恐战争"（War on Terror）** | 重塑了整个 2000 年代的国际秩序，改变了主权、干涉、先发制人等基本规范 |
| 2008 金融危机 | **G20 领导人峰会** | 从财长会到首脑会的制度跃迁，改变了全球经济治理格局 |
| COVID-19 | **"生物安全""供应链韧性""去风险"** | 从"效率优先"到"安全优先"的坐标系转换 |
| 2022 俄乌战争 | **"基于规则的国际秩序"vs"多极化"** | 两种概念平面的对立公开化 |

### 战略启示

**如果你在危机事件发生后才开始想"需要创造什么新概念"，已经太晚了。**

概念创造"客观演绎"（黑格尔）的逻辑要求概念在事件之前已经在思想史的潜在性中存在。孙利天强调的概念创造"面向未来的理论想象"维度在此直接转化为战略行动原则：

> 真正有效的地缘政治概念创造要求**在事件发生之前就已经准备好潜在的"概念平面"**，事件来临时只是让它在公共意识中"豁然贯通"。

案例："人类命运共同体"
- 2013 年提出 → 当时看起来空洞
- 2020 年疫情中 → 在全球治理失效的背景下被重新发现
- 2023 年全球南方话语兴起 → 获得实质性内容
- **概念平面在前，事件在后，事件提供了现实化的契机**

对照：美国"民主 vs 威权"叙事
- 2020 年"民主峰会"推出 → 缺少前置的哲学概念创造（只是既有冷战话语的翻新）
- 2023 年加沙危机 → 叙事崩溃（因为以色列/巴勒斯坦的复杂矛盾无法在这个平面上被处理）
- **没有概念平面，就无法应对意外事件**

## 五、一个实践框架：地缘政治概念创造的四步检验

```
┌─────────────────────────────────────────────────────────────┐
│         地缘政治概念创造的四步检验                          │
│                                                             │
│ ① 客观性：你的新概念是否锚定在可观测的物质过程中？        │
│    ↓ 否 → 这是造词运动（word-making），不会持久              │
│    ↓ 是 → 继续                                              │
│                                                             │
│ ② 对立统一：它是否从真实的结构性矛盾中生长出来？           │
│    ↓ 否 → 它是外部套用的公式，可以被任何人随意挪用          │
│    ↓ 是 → 继续                                              │
│                                                             │
│ ③ 内在性平面：它是否改变"什么算相关事实"的标准？           │
│    ↓ 否 → 它只是已有坐标系中的一个新词                      │
│    ↓ 是 → 继续                                              │
│                                                             │
│ ④ 理想性：它是否提供了一条从"是"到"应当"的规范性路径？      │
│    ↓ 否 → 它是描述性知识，不是政治概念                      │
│    ↓ 是 → ✅ 这是一个真正的概念创造                          │
└─────────────────────────────────────────────────────────────┘
```

### 检验案例："一带一路"

| 标准 | 结果 |
|---|---|
| ① 客观性 | ✅ 基础设施投资的物质过程可锚定 |
| ② 对立统一 | ✅ 南北发展矛盾、全球基础设施赤字 |
| ③ 内在性平面 | ✅ 重构了从欧亚大陆中心看世界的坐标（非海洋中心） |
| ④ 理想性 | ✅ 提供了一条"互联互通 → 共同发展"的规范性路径（尽管有争议） |

### 检验案例："民主 vs 威权"二元叙事

| 标准 | 结果 |
|---|---|
| ① 客观性 | ❌ 物质过程锚定薄弱大量灰色区域国家无法归类 |
| ② 对立统一 | ❌ 把复杂矛盾还原为二分法 |
| ③ 内在性平面 | ⚠️ 提供了简洁坐标但无法容纳异质性 |
| ④ 理想性 | ❌ 规范性方向模糊（"民主"的内涵不断被掏空） |

结论：这是一个有动员力但脆弱的叙事组合，而非真正的概念创造。
